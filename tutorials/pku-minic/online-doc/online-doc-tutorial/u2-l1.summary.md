本讲拆解 `docs/index.html` 的配置与主题系统，共四个最小模块。配置上，`window.$docsify` 是声明式全局对象，分核心库消费项（`loadSidebar`、`subMaxLevel`、`auto2top` 等）与插件专属项（`search`、`pagination`、`copyCode`）；配置脚本须排在核心库之前加载，各插件从同一对象取走属于自己的配置段。主题由三层叠加构成：`docsify-themeable` 提供 CSS 变量底座，亮暗两套配色表 `theme-simple`/`-dark` 为其填值，`docsify-darklight-theme` 充当切换器；明暗切换底层依赖带 `title` 的备选样式表（alternate stylesheet）机制，选择经 localStorage 持久化。KaTeX 公式需 `docsify-katex.js`（识别 `$...$`）与 `katex.min.css`（数学字体）协同。本地 `main.css` 因加载在所有 CDN 样式表之后、覆盖 `:root` 上同名 CSS 变量而生效；主色由 HSB 三分量 `--theme-hue`、`--theme-saturation`、`--theme-lightness` 驱动，改一行即可换肤，`--code-font-size`、`--sidebar-nav-indent` 等为观感微调的变量开关。
