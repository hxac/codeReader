本讲解析 nano-vllm 的张量并行（TP）：把同一层权重按某维度切给各 rank（tp_size 为卡数、tp_rank 为卡号）。列并行（ColumnParallelLinear，tp_dim=0）切输出特征，各 rank 算互不重叠的输出段，无需通信；行并行（RowParallelLinear，tp_dim=1）切输入特征算部分和，须 all_reduce 求和，偏置仅在 rank 0 加一次。所有并行层继承 LinearBase，通过挂在 Parameter 上的 weight_loader 钩子把「分片加载策略」与「前向计算」彻底解耦。QKVParallelLinear 用字符串 shard_id 处理 GQA 下不等宽的 q/k/v 三段合并；MergedColumnParallelLinear 用整数 shard_id 合并等宽的 gate/up。词表并行中 VocabParallelEmbedding 借 mask 加 all_reduce 复制 embedding，ParallelLMHead 用 gather 把 logits 汇总到 rank 0（因后续每层都需完整 embedding，而采样只需 rank 0）。经典的「列→行」配对使每个 attention 块和 MLP 各仅发生一次 all_reduce，是 Megatron-TP 省通信的核心。
