本讲义以统一 FIFO 封装 `Fifo`（`toplevel="surf.fifo"`）为被测对象，教读者从零写出结构完整的 cocotb 回归测试。核心是五段式骨架：许可证头 → 四段式方法学头（Sweep/Stimulus/Checks/Timing，说清「证明什么、不证明什么」，禁用通用模板）→ `TB` 类 → `@cocotb.test()` 协程 → `PARAMETER_SWEEP` 与 pytest 包装函数。`TB` 类用 `env_flag`/`env_sl` 从环境变量读回泛型、自适应设置，并把所有开放式等待用 `with_timeout` 加上界；同步 FIFO 必须用 `start_lockstep_clocks` 单协程驱动两路同源时钟。每个协程只证明一个行为，可用 `if ...: return` 限定部分用例，断言落在外部可见信号。参数扫描优先精心矩阵而非笛卡尔积，`parameter_case` 的 ID 同时充当 pytest 用例名与 sim_build 目录名；同一份参数字典走两条通道——`hdl_parameters_from` 过滤 `_G` 键喂 GHDL 泛型，原字典经 `extra_env` 进环境变量供 `TB` 读取。关键辅助来自 `regression_utils.py`，入口是同步函数调 `run_surf_vhdl_test`。
