本讲讲解 SURF 的完整 AXI4 总线，覆盖三块。其一，AxiPkg.vhd 将 AXI4 的五个独立握手通道（读地址 AR、读数据 R、写地址 AW、写数据 W、写响应 B；读走 AR→R、写走 AW→W→B）折叠成读/写主机与读/写从机四条 VHDL 记录，沿用「VALID 与数据归生产方、READY 归消费方」的归属口诀。突发（burst）由 arlen/awlen（存「次数−1」）与 arsize/awsize（存 log2 字节数）刻画，getAxiLen 用向上取整除法把字节数换算成突发长度，写选通 wstrb 控制逐字节写入。其二，AxiRam 是最小完整的 AXI4 从机，以 SimpleDualPortRam 为后端，写三态机（ADDR→DATA→BLOWOFF）与读三态机（ADDR→PIPELINE→DATA）逐拍处理突发，突发长度错位时上报 bresp="10"（SLVERR），读路径永远回 OKAY；典型技巧为写地址预减 1、按现态寻址、wrEn=uOr(wstrb)。其三，AxiToAxiLite 是纯组合降级桥，收窄 wdata 至 32 位、展宽 rdata、rlast 恒置 1，仅 ID 跟踪需寄存，硬性前提为 32 位对齐的单拍事务。初值常量 _INIT_C 为全 0 静止态，_FORCE_C 把消费侧 READY 拉高以占位空接口。
