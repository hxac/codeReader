本讲承接 u10-l1（GPT→Llama），把视野扩展到两个工业级现代 LLM：Qwen3（稠密版）与 Gemma3，核心结论是「现代 LLM = 在解码器骨架（嵌入→N×TransformerBlock→norm→out_head）上对归一化、位置编码、注意力、前馈、精度五类零件做替换与叠加」，并用一张对比表固定了 GPT→Llama→Qwen3→Gemma3 的逐项差异。最小模块三块：Qwen3 架构、Gemma3 架构、四模型差异对比。Qwen3 ≈ Llama + qk_norm（对 Q/K 各做一次 RMSNorm）+ 解耦 head_dim（写死为独立超参，使 W_query 输出维可超 emb_dim）+ 长上下文 RoPE（base=1e6）+ SwiGLU + GQA（8 组），并有靠 tokenizer 注入空 `<think>` 标签在 reasoning/instruct 间切换的设计；MoE 变体只替换 FFN 一个零件。Gemma3 四张名片：5:1 滑动窗口/全局注意力混合调度、双套 RoPE（local 1e4、global 1e6）、零中心 RMSNorm（scale init=0、乘 (1+scale)）、MQA（n_kv_groups=1）+ 嵌入缩放（×√emb_dim）+ 每块 4 个 Norm + GeGLU（GELU 门控）+ query_pre_attn_scalar 预缩放。共性趋势：两者都用 RMSNorm、RoPE、门控 FFN、KV 压缩、bfloat16、weight tying、解耦 head_dim，即现代 LLM 的标配零件箱；差异本质是 Qwen3 保守改良、Gemma3 激进省内存（为超大词表与长上下文腾空间）。关键术语 qk_norm、解耦 head_dim、SwiGLU/GeGLU、GQA/MQA、SWA 混合调度、双套 RoPE、零中心 RMSNorm、嵌入缩放、weight tying 供后续讲义承接。
