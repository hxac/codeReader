本讲实现完整预训练循环 `train_model_simple`，将 u5-l1 的交叉熵损失用于更新权重。核心是 PyTorch 训练四件套：`zero_grad()`（梯度默认累加故必清零）→ 前向算 loss → `backward()` 反传梯度 → `step()` 更新参数；优化器用 AdamW，靠自适应学习率与解耦权重衰减（weight_decay=0.1）稳定大模型训练。`train_model_simple` 为外层 epoch、内层 batch 的双层循环，`global_step` 从 -1 起按 `eval_freq` 调度评估，`tokens_seen` 累计处理 token 数。循环内埋两条互补监控线：数值线 `evaluate_model` 给定量损失趋势，样本线 `generate_and_print_sample` 给定性文本质量。`plot_losses` 以双 x 轴绘制曲线；核心结论是训练损失持续降而验证损失停滞即过拟合，根因为训练数据极小（《The Verdict》约 5145 token）却反复迭代 10 轮，最终解法是加载 OpenAI 预训练权重。
