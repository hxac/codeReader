本讲补齐 Transformer 块的三个非注意力「零件」，它们都逐位置作用在特征维 `emb_dim` 上，不改变 `batch × num_tokens × emb_dim` 的整体形态。一是 LayerNorm（层归一化）：针对内部协变量偏移，沿特征维把每个 token 拉成均值 0、方差 1，再用可学习参数 `scale`（初始 1）与 `shift`（初始 0）做仿射变换；方差统计显式用 `unbiased=False`（有偏估计），以对齐 GPT-2 原始训练、便于加载 OpenAI 预训练权重。二是 GELU：GPT-2 的激活函数，本项目采用 tanh 近似实现，处处可导、平滑，且在负区间保留非零梯度，优于硬截断的 ReLU；它无可学习参数。三是 FeedForward（前馈网络 FFN）：`emb_dim → 4×emb_dim → emb_dim` 的两层瓶颈结构，中间夹 GELU，对每个 token 独立、共享地变换，是模型参数的重要来源。三个类均汇总于自包含的 `gpt.py`，与多头注意力并列，为下一讲组装 `TransformerBlock`（含 pre-LayerNorm 与残差连接）提供了全部前置模块。
