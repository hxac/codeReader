本讲解决自回归语言模型训练时不能看到未来 token 的问题：u3-l1 的双向注意力会偷看未来，造成信息泄露。为此引入因果注意力（masked self-attention），屏蔽注意力矩阵对角线以上位置，使每个位置只关注自身及之前的 token。核心实现采用 -inf 技巧：softmax 前用 torch.triu(..., diagonal=1) 生成严格上三角布尔掩码，masked_fill_ 把对应分数置 -inf，softmax 后这些位置自然变 0 且自动归一化。掩码是不可学习常量，须用 register_buffer 注册：不被优化器更新，却能随 .to(device) 迁移并默认进入 state_dict（persistent=False 可改为非持久化）；若当普通张量挂 self 会在 GPU 训练时报设备不一致错误。最终整合出支持 batch 维（用 transpose(1,2) 转置、按 num_tokens 切片掩码）的 CausalAttention 类，其掩码写法在第 4 章 MultiHeadAttention 中被原样继承，因果机制在多头下完全不变。
