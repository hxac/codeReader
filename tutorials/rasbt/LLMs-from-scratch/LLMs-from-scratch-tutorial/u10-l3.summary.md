本讲从「能跑通」转向「跑得好、跑得省」，含三项独立工程技能。其一，内存高效加载：朴素加载会让权重在加载瞬间两份并存、峰值内存翻倍；省 GPU 显存让 state_dict 先进 CPU 再逐参数写入，省 CPU 主存用 meta 空壳、mmap=True 或逐张量加载，推荐组合为 meta 空壳 + load_state_dict(mmap=True, assign=True)。其二，扩展 tokenizer：新 token 只能注册为 special token、不可改动 BPE 合并表，ID 从 n_vocab 起编号并入 special_tokens，encode 须用 allowed_special 放行；新 ID 会越界，须同步扩容 tok_emb 与 out_head，weight tying 模型只需共享 tok_emb.weight，新增参数须微调。其三，训练加速：以 tok/sec 衡量吞吐，10 项优化中 bfloat16、FlashAttention（SDPA）、torch.compile 收益最大，且均不改模型数学。
