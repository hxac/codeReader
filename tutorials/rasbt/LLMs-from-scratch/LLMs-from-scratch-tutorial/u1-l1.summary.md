本讲义是整个学习手册的开篇，只读 README.md、pyproject.toml、requirements.txt 三个顶层文件，为后续讲义建立「全局地图」。它包含三个最小模块：项目定位、顶层目录与章节结构、技术栈与依赖清单。

核心结论：本仓库是《Build a Large Language Model (From Scratch)》的官方代码库，首要目标是「教学」而非复现 SOTA——用 PyTorch「从零」手写一个类 ChatGPT 的 GPT 模型，全程不依赖 transformers 等高层 LLM 库，唯一例外是分词器借助 tiktoken。这条「文本数据处理 → 注意力机制 → GPT 模型组装 → 预训练 → 分类微调 → 指令微调」的流水线，镜像了真实大规模基础模型的训练方法，但可在普通笔记本上跑通、不强依赖 GPU。

引入的关键术语：LLM、GPT（解码器型 Transformer）、PyTorch（张量计算 + 自动求导）、Jupyter Notebook、从零（from scratch）、BPE 字节对编码、checkpoint、bonus 附加材料。结构约定上，顶层按 ch02–ch07 与 appendix-A/D/E 组织，每章 01_main-chapter-code 含正文 `<章>.ipynb`、汇总脚本与 previous_chapters.py（承担章节间自底向上的代码复用，如 ch04 继承 ch03 的 MultiHeadAttention，ch05 继承 ch04 的 GPTModel）。技术栈精简为 PyTorch（实现与训练）、tiktoken（分词）、TensorFlow（仅用于加载 OpenAI GPT-2 权重做结构验证，不参与训练），辅助库含 JupyterLab、matplotlib、pandas、tqdm。普通读者用 pip install -r requirements.txt 即可跑通正文，该文件还逐行标注了「库 ↔ 章节」对应关系。
