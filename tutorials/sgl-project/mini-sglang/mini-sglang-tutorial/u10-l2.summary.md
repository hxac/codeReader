本讲讲解 Mini-SGLang 的自定义 kernel 包：它通过 **tvm-ffi** 把 C++/CUDA 源码编译为 Python 可调用模块，提供 `load_jit`（按 dtype/尺寸的模板参数特化并缓存多份）与 `load_aot`（无参数、构建一次复用）两条懒加载路径。三个 kernel 各司其职：`indexing` 用 warp 级整行搬运实现词表并行 embedding，masked 模式把越界 token 置零以支持 TP，被 `VocabParallelEmbedding` 调用；`store_cache` 把新算的 K/V 按 `out_loc` 分散写入分页 KV 池，是每层 attention「先落池再算」的第一步；`fast_compare_key` 是唯一的 CPU kernel，用 `std::mismatch` 返回公共前缀长度，服务 radix 树前缀匹配。此外 `PyNCCLCommunicator` 是自研 NCCL 通信器，初始化时用 gloo 组广播 UID、预分配「对称内存窗口」（大小取最大激活张量与 1 GiB 上限的较小者），使 all_reduce 命中固定地址缓冲、从而可被 CUDA Graph 录制。
