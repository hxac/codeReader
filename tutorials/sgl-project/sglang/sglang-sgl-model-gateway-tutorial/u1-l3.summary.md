本讲以 README、main.rs、version.rs 三文件为主，讲清网关如何用一条命令跑起来。核心结论：一次构建产出 sgl-model-gateway/smg/amg 三个二进制，共用同一份 main.rs，`smg launch` 与直接参数两种写法等价。引入的关键术语：`Backend` 枚举（sglang/vllm/trtllm/openai/anthropic，默认 sglang，其中 vllm/trtllm/anthropic 尚未实现会降级为普通路由）；版本信息由 build.rs 在编译期注入、version.rs 用 `env!` 固化，`--version` 与 `--version-verbose` 在 main() 中先于 clap 拦截。最重要的概念是：README 的「五种模式」在源码里由三个正交维度组合——`RoutingMode`（Regular/PrefillDecode/OpenAI）、`ConnectionMode`（Http/Grpc，由 worker URL 协议决定）、IGW 开关；启动横幅的 Mode 文字按 IGW>OpenAI>PD>Regular 优先级显示，但仅供展示，真正决定路由器构造的是 `to_router_config()` 里的 RoutingMode。常规 HTTP 路由最常用的入口参数是 `--worker-urls` 与 `--policy`（默认 cache_aware）。
