本讲是手册首篇，仅读 README、Cargo.toml、src/lib.rs 三个文件，为 sgl-model-gateway（简称网关）建立整体地图。网关是面向大规模 LLM 部署的高性能模型路由控制面与数据面，深度优化 SGLang 运行时，亦兼容 vLLM/TRT-LLM/OpenAI 后端，位于客户端与一群 Worker（推理进程）之间，做负载均衡与故障转移。核心结论是「四区域架构」：控制面（WorkerManager/JobQueue/Registry，经 JobQueue 异步串行注册、不阻塞转发）、数据面（HTTP/gRPC/OpenAI/IGW 路由器与 RouterManager，转发每次推理请求）、可靠性层（重试/熔断/限流/排队，以包装方式嵌入数据面）、可观测性层（指标/追踪/日志，横切全程）。引入的关键术语：Worker、控制面/数据面、JobQueue、core 跨区域底座。Cargo.toml 产出名为 smg 的库及三个二进制别名（sgl-model-gateway/smg/amg），依赖 smg-* 协同发布的姊妹 crate；lib.rs 导出 11 个自有模块加 5 个外部 crate 重导出。此四区域骨架供后续各讲放大细化。
