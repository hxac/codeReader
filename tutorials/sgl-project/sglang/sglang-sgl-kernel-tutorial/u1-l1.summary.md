本讲确立 sgl-kernel 的全局定位：它是一个独立于 SGLang 引擎、可单独打包与发布的 GPU 算子库，为 LLM/VLM 推理引擎提供定制化高性能计算原语。须分清三个名字——源码目录 `sgl-kernel/`、PyPI 分发名 `sglang-kernel`、Python 导入名 `sgl_kernel`（包历史上由 `sgl-kernel` 改名而来）。构建后端为 `scikit-build-core`（非 setuptools），驱动 CMake 与 NVCC 把 CUDA 扩展编译进 wheel；运行时要求 `torch==2.11.0`（ABI 钉版），构建期下限为 `torch>=2.8.0`，且 `dependencies=[]` 故 pip 不会自动安装 torch。版本号源自单行文件 `python/sgl_kernel/version.py`，经 `__init__.py` 第 4 行 re-export 为 `sgl_kernel.__version__`，需与 `pyproject.toml` 手动同步（可用 `make update` 批量改，CUDA/ROCm/CPU 三后端各维护一份 pyproject）。wheel 仅打包 `python/sgl_kernel/` 一个目录，开源协议为 Apache-2.0。
