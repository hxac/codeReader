本讲承接 u2-l4/u2-l5，聚焦 SGLang 在 AMD MI300X 上的硬件适配与量化，分三大模块。一是 AMD 适配 roadmap：从 2024-10 的量化分享，经框架级适配到 2025-08 的专用内核，资料横跨 README 多区段构成资料簇。二是 fp8/mxfp 量化：fp8 为每元素 8 位浮点(推理主用 E4M3)；mxfp 采用每 32 元素共享缩放因子的块缩放，精度与压缩更平衡；DeepSeek 因原生 fp8 训练而推理损失小。粗算量化把 671B 权重从约 1342GB 压到 fp8 约 671GB、mxfp4 约 336GB，直接决定大规模 EP 多卡部署能否成立。三是 AITER 与 MoRI：AITER 是基于 Triton、编译到 ROCm 的 AMD 推理内核库，地位类似 NVIDIA 侧 FlashInfer，补齐 attention 与 MoE grouped GEMM 等底层算子。核心结论：上层调度跨硬件共通，差异在最底层算子；量化、专用内核与 MI300X 大显存三者协同，造就 AMD nightly 上的 SOTA。
