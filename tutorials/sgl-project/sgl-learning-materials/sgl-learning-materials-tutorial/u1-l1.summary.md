本讲是整本手册的入门第一篇，定位 sgl-learning-materials 仓库：它是 SGLang 官方维护的学习资料聚合库，而非运行时代码库，不含任何 .py/.ts/.rs 等可执行代码，核心资产仅 README.md 导航索引与 slides/、blogs/ 少量仓库内文件，采用 MIT 许可证，版权方为 sgl-project 组织。

引入的关键术语：SGLang 是开源 LLM 推理服务引擎（serving engine），主打高吞吐、低延迟、低成本；README 把资料分为 Slides/Blog/Videos/Paper/Documentation 五大类，其中 Slides 为「仓库内 + 外部」混合，其余四类主要指向外部。判断资料归属看链接是否以 https:// 开头。核心边界：本仓库是「路标」，真正知识在链接终点；引擎源码在主仓库 sgl-project/sglang，用法在文档站 sgl-project.github.io，原理在 arXiv 论文，三者同属 sgl-project 组织，构成「资料→用法→原理→代码」的递进链路。
