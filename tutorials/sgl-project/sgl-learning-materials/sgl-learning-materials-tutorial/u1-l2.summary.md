本讲义讲清 sgl-learning-materials 仓库的物理结构。根目录仅三块基石：README.md（导航枢纽）、LICENSE（MIT 许可，版权 sgl-project）、.gitignore（仅忽略 .DS_Store），且不含任何可执行代码，印证「资料库而非引擎」的定位。内容目录中，blogs/ 仅一篇 .md 博客「Efficient LLM Deployment and Serving.md」，配 docs/figs/ 下 9 张 PNG；slides/ 体量最大，共 27 文件（25 PDF + 1 PNG adoption.png + 1 PPTX sglang_adoption_logo.pptx，即仓库唯一 .pptx），命名遵循「事件前缀」惯例。引入的关键术语包括相对路径（从引用文件算起，正文与配图须绑定）、文件扩展名、永久链接。核心方法：判断资料归属看 README 链接是否以 https:// 开头（外链）或 slides/ 相对路径（本地资产）。最后，sgl-learning-materials-tutorial/ 是本学习手册目录，由 manifest.json 驱动、讲义以 uN-lM-*.md 编号命名，属教学讲义层，与根目录 README.md（SGLang 资料导航）层级不同、不可混淆。
