本讲把 FlashAttention 反向从 Ampere（SM80）推进到 Hopper（SM90）与 Blackwell（SM100/SM110），主题是「同一套反向数学（dQ/dK/dV 五式，重算 S 并用 LSE 复原 P），三代硬件用不同手段落地」。核心难题始终是 dQ 的跨 thread block 累加：主循环按 n_block 切工作，同一 m_block 被多个 thread block 写入。Sm90 在 Sm80 数学之上引入 WGMMA（异步、操作数取自 smem）与 TMA（mbarrier 按字节计完成），采用 producer/consumer 二分，dQ 由专用 store warp 经「rmem→smem→批量原子加」落盘。Sm100 质变为累加器住 tmem（tcgen05/UMMA），靠 16 个 warp 的深度专门化（reduce/compute/mma/load/relay/empty）协作，mma 与 compute 解耦；dQ 归约比 Sm90 多一跳「tmem→rmem」（tcgen05.copy）。2CTA dQ reduce 让 cluster (2,1) 内两 CTA 经 relay warp 与 cluster mbarrier 交换 dS、按 stage_offset 切分 M 行，所有按 CTA 计的量（MMA M 维、tx_count）翻倍；块稀疏反向将连续 m_block 迭代换成稀疏迭代，但与 2CTA 互斥。三代 dQ 策略可压缩为「rmem 原子加→smem 批量原子加→tmem 归约+批量原子加」，数学等价。
