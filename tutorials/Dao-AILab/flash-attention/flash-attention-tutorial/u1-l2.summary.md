本讲义解决仓库「导航」问题，覆盖三个最小模块：三代实现定位、辅助目录（测试/基准/训练/调试笔记）、构建脚本与子包共存机制。核心结论：仓库三代同堂——FA2（顶层 flash_attn/ 与 csrc/，C++/CUDA，包名 flash-attn，主打 Ampere）、FA3（hopper/，包名 flash-attn-3，专攻 Hopper）、FA4（flash_attn/cute/，Python+CuTeDSL，包名 flash-attn-4，面向 Hopper/Blackwell），各产出独立 wheel。引入的关键术语：命名空间包、pkgutil.extend_path、运行时 JIT 编译（FA4，安装时不编译）对比安装期 nvcc 编译（FA2/FA3）。三代共存靠三件套配合：FA2 的 setup.py 用 find_packages 排除 flash_attn.cute、FA4 的 pyproject.toml 声明 packages=["flash_attn.cute"]、flash_attn/__init__.py 用 extend_path 将 flash_attn 变为命名空间包。注意同名 flash_attn_func 在三代中同源不同实；FA4 公开 API 仅 flash_attn_func 与 flash_attn_varlen_func。
