本讲深入 `crossbeam-channel` 有界通道（`bounded(cap>0)`）的底层实现 `flavors/array.rs`，并配合 `waker.rs` 完成无锁 MPMC 环形缓冲的讲解。核心是基于 **Vyukov 有界队列**的设计：每个槽位持有一个原子戳记 stamp，把 `{lap（圈数）, mark（断开位）, index（槽下标）}` 打包进一个 `usize`，其中 `mark_bit=(cap+1).next_power_of_two()`、`one_lap=mark_bit*2`。状态判定只需两个等式：可写 ⟺ `stamp==tail`，可读 ⟺ `stamp==head+1`；写后戳记变 `tail+1`，读后变 `head+one_lap`。`lap` 用于防止跨圈 ABA，`mark` 表达通道断开。

关键概念包括：**两阶段协议**——start 阶段用 CAS 占位并填入 `ArrayToken{slot, stamp}`，commit 阶段（`write`/`read`）写读消息并改戳记，从而把 select 的「选中」与「提交」解耦；以及 **Acquire/Release 配对**保证消息内容先于戳记变更可见。满/空时阻塞版 send/recv 走「自旋退避→登记 `SyncWaker`→复查就绪→`wait_until` 阻塞」，对端在 commit 时 `notify` 唤醒，靠「先登记再复查」杜绝丢失唤醒；断开由 `tail.fetch_or(mark_bit)` 点亮并经 `Waker::disconnect` 唤醒全部等待者，最后一个 receiver 离开时清理残留消息。
