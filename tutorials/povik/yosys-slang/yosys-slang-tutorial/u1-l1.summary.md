sv-elab（原名 yosys-slang）是一个 SystemVerilog 精化器（elaborator），其核心定位是把 SystemVerilog 设计精化为「字级（word-level）网表」，作为综合工具链的「前端」。精化指解析参数、展开 generate、把模块层次实例化为确定的电路结构；字级是相对门级而言，保留设计者的高层结构（如一个 32 位加法单元），便于下游做高层优化与技术映射。

sv-elab 不负责 SV 的词法、语法、语义分析，这些交给开源库 slang 完成。关键职责边界：slang 把 SV 源码解析、建 AST 并做语义分析产出「设计表示」；sv-elab 遍历该表示，翻译成 Yosys 的字级 RTLIL 网表。slang 与 {fmt} 库以源码子模块内嵌在 third_party/，故 clone 需 --recursive。

sv-elab 是 OpenROAD 综合工具与 Yosys（v0.67+ 内置）的组件；对老版本 Yosys 以 slang.so 插件形式加载（yosys -m slang 或 plugin -i slang），加载后覆盖内置版本。无论哪种集成方式，对用户暴露的命令都叫 read_slang。它只支持 SV 的「可综合子集」（IEEE 1800-2017/2023），并非全部合法 SV。
