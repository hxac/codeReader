本讲讲解 hir2ir 阶段：把上一讲得到的「万物皆 Call」的 HIR 翻译成具体的 Tile IR Operation。核心是把 hir2ir 理解为一个「HIR 解释器」——遍历 Block/Call，按被调用对象类型（FunctionTy、BoundMethodTy、ClosureTy、DTypeConstructor 等）分派：stub/内置走 `ImplRegistry` 查表发射单条 Operation，用户函数与闭包则内联，并用 `async`/`await` 协程实现软件栈以绕开 Python 递归上限。

四个最小模块：`hir2ir`（主分派器）、`loop_impl`（循环降级）、`if_else_impl`（分支降级）、`PhiState`（汇合点 phi 状态）。关键结论：循环被编译成函数式 fold——循环体内被改写的局部变量（`body.stored_indices`）成为携带值，以 `initial_values` 流入，每轮由 `continue` 回传；`if/else` 降级成 `IfElse`，编译期常量条件直接展平。`continue_`/`break_`/`end_branch` 不直接跳转，而是登记进 `ControlFlowInfo.jumps` 供外层收尾。引入的关键术语：携带值、归纳变量、LoopVarState（body_phi/result_phi 双 phi）、ControlFlowInfo/JumpInfo、Builder、Scope。
