本讲拆解编译流水线第一段 ast2hir：把 Python 内核翻译成 HIR——AST 与 Tile IR 之间的归一层。其核心设计「万物皆 Call」：不定义具体 Operation，而用函数调用统一建模，算术、属性、下标、字面量乃至 if/for/while 控制流都被归一为 Call（控制流即对 stub 的 Call 加嵌套 Block）。核心结构为 Value、Call、Block、Function，Block 由线性 Call 序列与终止 Jump、stored_indices 组成。入口 get_function_hir 带缓存，经取源码、if True 缩进技巧、构造含闭包的全局表、作用域分析、建 _Context、lowering 与收尾产出 hir.Function，且只调一次。ast_get_all_local_names 是地基：静态算清本函数局部名并分配稳定下标，把读写翻译成 load_var/store_var，用 stored_indices 标记块内改写的局部，供下游做循环 carried value 合并。控制流编法：if→if_else 加两块 END_BRANCH；for→loop，体块以归纳变量为参数、结尾 CONTINUE；while 体首插条件 break 再套 loop；链式比较与 and/or 展开为短路嵌套 if_else。下讲 hir2ir 将把这些 Call 与 Block 翻成具体 Tile IR Operation。
