本讲覆盖 cuTile 的数据类型与类型提升。DType 是不可变单例对象（而非 Python type），数值类型分 Boolean、Integral、Float（含 bfloat16）与 RestrictedFloat（tfloat32、float8_* 等）四族，族间有序、默认 int32/float32；其中 RestrictedFloat 不算术，禁止隐式提升，须用 astype 显式 cast。类型提升的核心是区分 loosely typed 常量（字面量、Constant 值，运算时迁就对方）与 strictly typed 操作数（有确定 dtype）：两者皆 loose 时结果仍为 loose 常量；否则先将 loose 整数按值归为 int32/int64/uint64、浮点归 float32，再按「跨族强族胜、同族查提升表 _common_dtype_table」求公共类型；涉及受限浮点、有无符号混合或 f16 与 bfloat16 混合均报错要求显式 cast，loose 整数吸收进更窄类型时做范围检查，越界抛 TileValueError。此外，RoundingMode（默认 RN）控制浮点运算与类型转换的舍入，PaddingMode（默认 UNDETERMINED）控制 load 越界元素的填充，越界时应显式指定（如 ZERO）。
