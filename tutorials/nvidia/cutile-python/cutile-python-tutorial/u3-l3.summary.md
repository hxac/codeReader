本讲覆盖 cuTile tile code 中可用的 Python 控制流子集及其底层 IR 映射，核心是三个最小模块：**if/for/while 语法子集**、**range 的 step 限制**、**控制流的 IR 结构（IfElse / Loop）**。

tile code 是「被翻译的 Python」，无 Python 运行时：编译器把 `if/else`、`for ... in range(...)`、`while` 解析成结构化 Tile IR 操作，三者可任意嵌套。`if` 条件须为标量布尔。`for` 的迭代对象目前仅 `range`，其返回区间值 **RangeValue**（start/stop/step 三元组，类型 **RangeIterType**）。`range` 的 **step 必须严格 > 0**：常量负步长在 `range_` 直接报错，经变量间接传负步长属未定义行为。`break` 仅 `while` 支持（`for` 编成定数计数循环 ForOp，无跳出语义），`continue` 两者都支持；不支持 for/while-else、match-case、异常。

`while` 的 iterable 为 None，靠循环体首部插入「if cond: pass; else: break」实现条件，编成通用 LoopOp。因 tile 不可变，循环体内被重新赋值的变量成为**携带值（carried values）**：经 initial_values 入循环体，每轮由 continue 回传，结束时产出循环结果——本质是 SSA / 函数式 fold，也是 GEMM 累加器沿 K 维流动的形态。
