本讲在 u3-l1 的 load–store 骨架中填补 compute 环节，覆盖三个最小模块：逐元素算术、astype 类型转换、形状变换。算术运算中运算符（+ - * @ 比较等）与 ct.add/ct.mul 等函数等价，二者均做形状广播（NumPy 规则）与类型提升（u2-l4），返回 TileOrScalar；需注意 Tile.__eq__ 被重载为返回布尔 tile 的逐元素比较，而非对象相等。astype 做受 RoundingMode 影响、不改 shape 的数值转换，须区别于保持比特不变的位重解释 bitcast。reshape（支持 -1 推断）、permute（全轴置换）、transpose（交换两轴）、broadcast_to、expand_dims（含 x[:,None] 语法）、cat 均返回新 tile，受每维须为 2 的幂约束，cat 还要求两输入同形。贯穿全讲的根本性质是 tile 不可变：所有算术与形状操作都产生新 tile，对应底层 SSA 风格的 Tile IR。综合实践为把 (M,N) 矩阵转置为 (N,M) 的内核，可用 transpose 或 load 的 order 参数两条路径实现。
