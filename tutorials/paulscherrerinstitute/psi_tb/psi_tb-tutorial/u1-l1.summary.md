本讲是 psi_tb（PSI 开源的 VHDL testbench 工具库）项目总览，不涉及代码。它先厘清核心概念：VHDL、仿真、RTL（可综合、进芯片）与 testbench（仅仿真、不进芯片）之分；psi_tb 专服务于 testbench，故不要求可综合，可自由使用 VHDL 全部语言能力。其归属边界由 README 的 What belongs / does not belong 划定，只收 testbench 专用代码，可综合代码归 psi_common。它解决三类问题：总线功能模型 BFM（如 AXI、I2C）、数值检查函数（自动比对期望与实际值并打印可读错误）、自动激励生成（时钟同步等待、选通脉冲等），分别对应后续单元 u5/u7、u3、u4。维护者为 Benoît Stef，作者为 Oliver Bründler 与 Benoît Stef；许可证为 PSI HDL Library License（LGPL 加固件/FPGA 例外，允许把库综合进 bitstream 后闭源发布）。版本遵循 major.minor.bugfix 语义化规则，2.0.0 与 3.0.0 为不向后兼容的里程碑，3.0.0 是为对齐 psi_common v3 而做的重构。关键术语 RTL、testbench、BFM、psi_common 将由后续讲义承接。
