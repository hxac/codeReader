本讲解读 psi_common 库中被引用最多的 psi_common_math_pkg：它不描述硬件，而是提供一批编译期（elaboration time）求值的工具函数，让端口位宽、计数上限等常量随 generic 自动推导、不产生逻辑门，是全库"全 generic 化"风格的底层支撑。核心分四组：log2ceil/log2/isLog2 用于位宽推导，其中 log2ceil 向上取整，是 RAM 地址、指针、电平端口的标准工具；max/min/choose 取极值与做条件选择，choose 凭多个类型重载充当端口声明区无法写 if 的三元运算符；to_uslv/to_sslv 与 from_uslv/from_sslv 完成整数与 std_logic_vector 互转，from_str 把含小数与科学计数法的字符串解析为 real（并有逗号分隔数组重载）；ratio 给出大小频率比的向上取整，is_int_ratio 判定两频率是否整数倍（同步整数比 CDC 的前提），nearest_pow2 返回≥a 的最小 2 的幂（用于 AXI 寄存器计数常量，不可综合）。这些函数将被后续 RAM、FIFO、CDC、strobe、AXI 讲义反复引用，使用时需留意 max_a/min_a 初值为 0、nearest_pow2 超 2^31 返 0、ratio 两值相等时打 warning 等边界局限。
