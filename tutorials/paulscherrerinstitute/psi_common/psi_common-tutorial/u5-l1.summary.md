本讲是 CDC 单元首篇，精读 `psi_common_pulse_cc`：跨异步时钟域传单周期脉冲与复位的基石组件。脉冲跨越采用经典「翻转 → 3-FF 同步 → 异或」三步法，将稀疏事件先转为长期稳定电平、经 B 域多级同步器采样后再还原为脉冲；约束是同通道脉冲间隔须满足 ≥ n_sync/f_b，否则两次翻转相互抵消致脉冲丢失。复位走「异步复位、同步释放」的 4 级同步链，内部复位信号带初值以保证上电即处于确定的有效复位态；`a_rst_pol_g`/`b_rst_pol_g` 两个 generic 经 `if generate` 与 or/and 选择，使一份代码支持高/低有效复位的四种搭配。引入的关键术语：CDC、亚稳态、同步器、脉冲、`num_pulses_g`（并行脉冲通道数，代码中「fifo width」注释有误导），以及 ASYNC_REG/shreg_extract/syn_srlstyle 三种综合属性——它们把同步器锁定为真实触发器并自动施加异步路径约束，故组件无需手写时序约束。本组件被 `simple_cc` 直接例化以传 valid 脉冲，是后续 status_cc、async_fifo 复位跨越的思想基础。
