本讲是 psi_common 手册开篇，仅读文档型文件，不涉及 .vhd 源码。`psi_common` 是瑞士保罗谢尔研究所（PSI）维护的通用、可复用、可综合 VHDL 库，只收录「项目无关且能完全 generic 化」的基础模块（时钟域跨越 CDC、FIFO、厂商无关 RAM、扩展语言的 package），与专注定点信号处理的 `psi_fix` 互补。许可证为 PSI HDL License = LGPL + 硬件例外，把 bitstream/flash 镜像明确算作 binary，允许闭源商用。库内 60+ 组件被 `doc/README.md` 分为 Memory、FIFO、CDC、Conversions、TDM、Arbiters、Interfaces、杂项、Packages 九大类，是全手册索引。版本号遵循 major.minor.bugfix 规则（不兼容升 major、加功能升 minor、仅修 bug 升 bugfix），当前 3.0.1；v3.0.0 为统一代码风格、不向下兼容的大重构，附 JSON 迁移脚本。引入的关键术语包括 VHDL、FPGA/ASIC、Generic（类属参数）、综合、CDC、TDM，以及 snake_case、`_i/_o/_io` 命名规范，供后续讲义承接。
