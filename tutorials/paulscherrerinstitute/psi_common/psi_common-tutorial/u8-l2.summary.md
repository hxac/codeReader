本讲覆盖 psi_common 中并-TDM 互转的最小模块：`psi_common_par_tdm`（并行→TDM）、`psi_common_tdm_par`（TDM→并行），以及作为外部节拍源的 `psi_common_strobe_generator`。核心思想是 TDM 与并行只是「N×W 位宽总线」与「N 个 W 位节拍」的等价表示重排，总数据量守恒，因此存在「1 拍↔N 拍」的节拍失配，握手须处理反压。

关键术语与概念（供后续承接）：**TDM（时分复用）**、**通道**（从 0 编号）、**节拍**、**隐式通道循环**——psi_common 约定通道 0 放最低位且最先收发，靠固定循环无需通道号旁路信号（仅适用同速率场景，可变通道数见 `_cfg` 变体与 u8-l3）。`par_tdm` 用 `ShiftReg` 移位寄存器每拍右移 `ch_width_g` 位把最低通道移出，配 `VldSr`/`LastSr` 通道位图，`last_o` 仅最末通道为高。`tdm_par` 用 `Idx` 按 0…N-1 填充 `DataReg`，满 N 或遇 `tdm_last_i` 即回绕输出，产 `par_keep_o`（每通道有效位，提前结束的短包只置已收位）与 `par_last_o`；通过 `v.VldReg(0):=vld_i` 保证背靠背不丢样本。两个转换器均不带速率源，节拍频率须满足 f_strobe ≤ f_clk/N。
