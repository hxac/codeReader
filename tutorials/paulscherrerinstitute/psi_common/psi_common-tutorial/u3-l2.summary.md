本讲承接 u3-l1 的简单双口 RAM，讲解 psi_common 的真双口 RAM：`psi_common_tdp_ram` 与带字节使能的 `psi_common_tdp_ram_be`。核心区别在于「真双口」的 A/B 两端口完全对称，各自具备 `clk/addr/wr/dat_i/dat_o` 五件套，可独立读也可独立写，两端口时钟可完全异步；而 `sdp_ram` 是一端只写、一端只读的有方向结构。关键术语包括：字节使能 `be_i`（宽度 `width_g/8`，`be_i(0)` 控最低字节，位宽须为 8 的倍数，默认全 1 退化成整字写入）、`behavior_g`（RBW 读前写/WBR 写前读，由 shared variable 立即生效实现）。核心结论：①是否选 `tdp_ram` 取决于「是否需要双向读写」而非「是否跨时钟」，两者都支持异步时钟；②`tdp_ram` / `tdp_ram_be` 不暴露 `ram_style_g`（与 `sdp_ram` 的显著差异），唯一综合旋钮是 `behavior_g`；③跨时钟须手动加 `set_max_delay` 约束，取较快时钟周期为双向上限；④库内真实使用者是 `ping_pong`（乒乓缓冲），异步 FIFO 实际用 `sdp_ram` 而非 `tdp_ram`——需纠正此常见误解。
