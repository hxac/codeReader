本讲是项目首篇，只读 README.md、Changelog.md、License.txt 三个说明文件，建立整体认知。核心定位：vivadoIP_mem_test 是一个 AXI 内存测试器 IP 核，自动向被测存储器写入 pattern、读回并逐拍比对，统计错误数与首个错误地址，用于验证存储子系统（如 DDR）可靠性。黑盒接口：控制面是 AXI-Lite 从机 S00_AXI（8 位地址，CPU 配置寄存器、启动测试、读状态与错误统计）；数据面是 AXI4 主机 M00_AXI（接被测存储器，发起突发读写）。关键术语：FPGA、IP 核、AXI-Lite（轻量配置）、AXI4 Full（突发传输）、Pattern（测试图形）、S00_AXI/M00_AXI、generic 参数（如 C_M00_AXI_DATA_WIDTH）。来源为 PSI FPGA HDL 公开库，作者 Oliver Bründler，维护者 Jonas Purtschert；许可证为 PSI HDL Library License（LGPL 加硬件特例，FPGA 比特流可按自己条款分发）。当前版本 1.2.1（C 驱动兼容 C++）。详细文档在 doc/mem_test.pdf。本讲不涉及 VHDL 语法，后续讲义进入寄存器地图与 RTL 细节。
