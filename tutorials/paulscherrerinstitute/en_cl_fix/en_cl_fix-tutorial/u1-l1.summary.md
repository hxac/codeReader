本讲是 en_cl_fix 学习手册的第一篇，不涉及公式，只读文档与目录结构。核心结论是：en_cl_fix 是一套跨 VHDL、MATLAB、Python 三语言的**位真（bit-true）定点运算库**，三套实现共享同一套定点语义，可对同一输入逐位比对输出。由此引出关键设计理念——**位真协同仿真（co-simulation）**：在高级语言建参考模型评估算法与量化误差，在 VHDL 综合成电路，再以位真一致性验证正确性。

本讲覆盖四个最小模块：项目定位与位真理念、三种语言的测试入口、语义化版本号策略、Changelog 演进历史。引入的关键术语包括：定点数、位真、协同仿真、`[S,I,F]` 格式（下一讲详讲）、Testbench、PSI HDL Library License（LGPL 加 FPGA 比特流例外条款）、`major.minor.bugfix` 版本号。读者还需掌握三语言的源码组织差异（VHDL 单文件包、Python 多模块、MATLAB 一函数一文件）、测试覆盖现状（Python 与 VHDL 有自动化测试，MATLAB 暂无）、Python 依赖 numpy、以及历史里程碑：1.0.0 首发 VHDL+MATLAB，1.1.0 补入 Python 与 VHDL testbench，1.2.0 加入 >53 位 wide fixed-point 支持。
