本讲是「位真双模型与测试方法论」单元的收口讲义，落到 psi_fix 库统一的 VHDL 编码风格，以 `psi_fix_mov_avg`（差分-累加-增益校正三级流水）为主样板，`psi_fix_resize_pipe`（带反压的两级流水）为辅。

核心引入三个概念：第一，**两段式方法 (two-process method)**——把时序电路拆成组合进程 `p_comb`（描述下一拍状态、写 `r_next`，承担全部定点算术）与时序进程 `p_seq`（仅 `r <= r_next` 打拍加同步复位）。其灵魂是 `v := r`，使未赋值字段默认保持，省去 `else r<=r` 样板。第二，**record 流水封装**——固定类型名 `two_process_r`，配 `signal r, r_next`，字段名后缀 `_0/_1/_2` 标流水级；valid 以数组 `Vld : std_logic_vector(0 to N)` 住进 record，用切片赋值 `v.Vld(1..N):=r.Vld(0..N-1)` 与数据同步逐级平移，既驱动输出有效，又守卫反馈环路（如累加器仅在 `Vld(0)='1'` 时更新）。复位是**选择性同步复位**，只复位 valid 与累加器，数据通路寄存器靠首个有效数据覆盖。第三，**命名约定**：全 `snake_case`；端口后缀 `_i/_o`、generic 后缀 `_g`；AXI-S 握手用 `dat/vld/rdy` 同义词（mov_avg 无 rdy、resize_pipe 有双向 rdy）；内部常量另用 `_c`。结论：学会这一套同构风格，即可快速读懂全库任意可综合实体。
