本讲承接 u2-l1，盘点 `psi_fix_pkg` 中可直接调用的可综合定点运算函数，分三模块。基本运算 `resize / add / sub / mult / abs / neg` 统一走「`psi_fix2_cl_fix` 翻译格式 → 委托 `cl_fix_*` 内核」三步走，结果格式 `r_fmt` 完全由调用者指定、函数不自动位增长；库级默认 `trunc / wrap`（最省资源、最易溢出），与组件层偏安全的默认值相反。移位函数 `shift_left / shift_right` 是唯一不直接复用 `en_cl_fix` 的函数——因其动态移位在 Vivado 下不可综合，故 psi_fix 自行重写：先放大到无损中间格式 `FullFmt_c`、移位、再 resize 回去，并用 `for i in 0 to maxShift` 循环把运行时移位量常数化以实现 `dynamic=True`。比较与范围函数 `compare / in_range / upper_bound / lower_bound` 按数学值对齐两数、不产生数值结果故无需 `r_fmt`。引入的关键概念：定点格式三元组 `[s,i,f]`、位增长三规则、`FullFmt_c` 无损中间格式、`dynamic/maxShift` 动态移位参数、Manual Splitting 三级流水范式、`-- Bittrue available in Python` 位真镜像标记（后续协同仿真前提）。
