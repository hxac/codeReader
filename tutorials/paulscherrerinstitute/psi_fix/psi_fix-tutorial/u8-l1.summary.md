本讲聚焦 psi_fix 中「函数无关」的非线性函数计算内核 `psi_fix_lin_approx_calc`，它用「分段线性近似 + 查表」逼近 sin/sqrt/inv/gaussify 等连续函数。原理是把输入区间等分成 P 段，每段存一对定点参数：offset（段中心函数值）与 gradient（段中心导数），运行时只算 \(\hat f(x)=o_k+g_k\cdot\text{reminder}\)，一次乘一次加；近似误差随段宽 Δ 平方下降。运行时先把输入拆成「索引」（取高 B 位作表地址）与「无符号余数」（其余低位），零除法；再通过「MSB 取反 + 有符号重解释」把余数从相对段左端 \([0,\Delta)\) 转成相对段中心 \([-\Delta/2,+\Delta/2)\)，VHDL 用位翻转、Python 用减法、位等价。内核不含表，靠 `addr_table_o` 出地址、`data_table_i` 回「gradient 高位 | offset 低位」拼接总线与外部表解耦。数据走 7 级流水：拆索引/余数→（外部 ROM 注册读占一拍）→切表项→乘→加（trunc/wrap，刻意不量化以落进 DSP）→末端 resize（round/sat）。各级中间格式（RemFmt/IdxFmt/IntFmt/AddFmt）在 VHDL 与 Python 两侧用同一组位宽算式推导，是位真一致的印章；由此「一个内核 + N 张表 = N 个函数组件」，表的自动生成留待 u8-l2。
