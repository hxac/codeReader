本讲是 psi_fix 学习手册开篇，仅读 README、introduction、Changelog、License 四份文档，不涉及代码。psi_fix 是 PSI 提供的定点 DSP VHDL 库，将 FIR、CIC、CORDIC、DDS 等通用算法封装为可复用组件，用于 FPGA 实时信号处理。核心哲学是「位真双模型」(bittrue dual model)：每个可综合 VHDL 组件必须配套一个逐位一致的 Python 模型；它并非理想浮点算法，而是主动约束到硬件的定点格式（含舍入、饱和、位增长），充当黄金参考 (golden reference)，由自检测试台逐位比对 VHDL 输出，不一致即以 `###ERROR###` 报错；位真模型与自检测试台是不可妥协的入库门槛。运行需与 en_cl_fix(≥1.2.0)、psi_common(≥2.15.0)、psi_tb(≥2.7.0)、PsiSim(≥2.1.0) 按固定目录并排摆放，或用 psi_fpga_all 一键拉取，外部依赖 Python 3.5+ 及 SciPy、NumPy。许可证为 PSI HDL Library License（LGPL2.1 加 FPGA 比特流例外，商业比特流可闭源分发）；版本号采用 major.minor.bugfix 三段式，4.0.0 因 camelCase→snake_case 接口重构而升 major。
