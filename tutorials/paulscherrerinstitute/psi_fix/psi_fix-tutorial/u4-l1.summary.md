本讲为「简单处理组件实战」首讲，首次走通「VHDL 实现 + Python 位真模型 + 自检测试台」完整闭环，覆盖 `psi_fix_mov_avg` 与 `psi_fix_resize_pipe` 两个模块。核心结论：mov_avg 用差分-累加结构（延时线 + 减法器 + 条件累加器）求滑动和，算术资源与抽头数 N 无关；各级格式由位增长规则严格推导——DiffFmt 减法 +1 整数位、SumFmt 累加 ⌈log₂N⌉ 整数位，保证内部不溢出。引入的关键术语：增益校正三模式 NONE（不校正，增益 N）/ROUGH（右移近似除法，增益 ∈[0.5,1]、无需乘法）/EXACT（右移后再乘修正系数，增益 1.0、耗 1 个 DSP）；AdditionalBits=⌈log₂N⌉；GcCoefFmt=[0,1,16] 与封顶 25 位的 GcInFmt 使 EXACT 乘法恰好落入单个 DSP48 slice，VHDL 与 Python 用同一组算式推导保证位真。另引入 out_regs_g 可配输出寄存器链（配合综合工具重定时）、resize_pipe 的「舍入级+饱和级」手动拆分流水与 AXI-S 反压（rdy_o/rdy_i）。该差分-累加结构为 u6 CIC 滤波器的雏形。
