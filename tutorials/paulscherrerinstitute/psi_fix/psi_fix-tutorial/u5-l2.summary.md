本讲详解 psi_fix_cordic_rot——CORDIC 旋转模式组件，把极坐标 (r,θ) 变换为直角坐标 (I,Q)，常用于 DDC/NCO。核心是「移位+加减」迭代微旋转：按 arctan(2^-i) 表逐级逼近目标角、方向由剩余角符号决定，无需乘法器算三角函数。整圈角度经「半圈折叠」纳入收敛锥（约±0.277圈），用角度最高2位 quad 判断是否对 (x,y) 同时取负以撤销180°折叠，故 angle_int_fmt_g 须为 (1,-2,x)。mode_g 分 PIPELINED（吞吐1样本/周期、资源∝迭代数）与 SERIAL（资源常数、吞吐≈1/N）两档，延迟相近、区别仅在吞吐-资源。CORDIC 引入幅度增益 G(N)≈1.65，gain_comp_g 控制是否乘 1/G 补偿（耗2乘法器），系数 GcCoef 用 (0,0,17)。关键：pl_stg_per_iter 不属本组件，而是姊妹件 cordic_vect 的旋钮；cordic_rot 的 ready 仍叫 rdy_i（方向实为输出），vect 已修正为 rdy_o。VHDL 与 Python 位真镜像，config.tcl 以 4 组参数覆盖。
