本讲把位真双模型与两段式风格应用到三组工程组件上：`psi_fix_dds_18b`（数控振荡器 DDS/NCO）、`psi_fix_demod_real2cplx`（实→复解调/下变频）与 `psi_fix_mod_cplx2real`（复→实调制/上变频）。DDS 的核心是无符号相位累加器每拍加相位步进 `phi_step`、溢出自然绕圈，再用一张 sin 表查 sin，并将相位加 0.25 圈复用同一张表得到 cos；`FirstSplCnt` 强制首样本相位为 0，TDM 用延时线让各通道各自维护累加值。关键结论一：非整数比用「同表不同步长」实现——`ratio_num` 点的 sin/cos 表不变，指针每拍步进 `ratio_den`、模 `ratio_num` 回绕，本振频率为 `f_LO=(ratio_den/ratio_num)·fS`，是类 Bresenham 的模运算技巧（`ratio_num/ratio_den` 为 5/3、100/3 即覆盖非整数比）。关键结论二：demod 与 mod 互为镜像，共用同表同指针；差别在 demod 混频后接长度恰为 `ratio_num` 的滑动平均（梳状低通，零点压掉 2·f_LO 镜像），故只适合窄带信号，而 mod 直接输出 `RF=I·sin+Q·cos` 无需滤波；系数缩放上 demod 除以 `ratio_num` 预补偿平均增益，mod 不除。三者 Python 黄金模型与 VHDL 在指针公式、格式常量、round/sat 上完全对齐，`config.tcl` 参数矩阵含非整数比覆盖，`###ERROR###` 为唯一失败判据。本讲确立「查表本振」这一 sin/cos 生成路线，与 u5-l2 CORDIC、u8-l3 线性近似形成三路线对照。
