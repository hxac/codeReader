本讲为 CIC 滤波器族收尾，围绕三个多通道抽取器展开。先立术语：TDM（时分复用，N 个通道轮流共用一根线，一组完整样本称一个 TDM 帧）与 par（每通道独立线）；CIC 命名规则中 `<input>_<output>` 字段取 par/tdm/-，`<fix|cfg>` 标示比率是否运行时可改。

三类结构：par_tdm（并行进、TDM 出）为每通道配独立累加器（2D 数组），抽取后用 psi_common_par_tdm 串行化为 TDM，积分器反馈无需延时线；tdm_tdm 用单组算子被全部通道时分复用，核心公式是积分器反馈延时 channels_g−1 拍、梳状器延时 channels_g×diff_delay_g 拍，配 Chcnt（帧内通道）与 Rcnt（帧间比率）双计数，每 ratio_g 个输入帧释放一个含 channels_g 个样本的输出帧，每通道抽取比恰为 R。cfg 变体把综合期常量 ratio_g 拆为 max_ratio_g（按最坏情况定宽累加器）加三个运行时端口 cfg_ratio_i（Ratio−1）、cfg_shift_i、cfg_gain_corr_i，常量移位被替换为流水化动态移位器 psi_common_dyn_sft，配置只能在复位时改；其代价是累加器永远按最大比率占位宽，比率确定的应用应优先用 fix。三者共用同一份位真 Python 模型，测试台逐位比对、###ERROR### 为唯一失败判据。
