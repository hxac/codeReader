PsiPyUtils 是 PSI 出品的通用 Python 工具库，只收录与具体问题无关的通用功能，不收业务专用代码，由此解释了它为何仅含 8 个机制性模块（TempWorkDir、FileWriter、ExtAppCall 等）。本讲只读三份「元信息」文件：README 给出库定位与使用契约，Changelog 记录版本演进，setup.py 决定打包。核心结论有四：许可证为 PSI HDL Library License，即 LGPL2.1 加固件/二进制使用例外；版本号遵循 major.minor.bugfix 语义——破坏性变更升 major、新功能升 minor、仅修 bug 升 bugfix；当前为 3.0.1，其中 3.0.0 是破坏性升级（`__init__.py` 重导出使 import 写法由 `from PsiPyUtils.<模块> import <类>` 改为 `from PsiPyUtils import <类>`，并开始以 pip 包分发），3.0.1 为纯 bugfix，修复 TextReplace 的行尾敏感与贪婪匹配问题；setup.py 借 `package_dir={"PsiPyUtils":"."}` 使扁平布局成立，并以 CustomSdist 在打包前清理旧产物。本讲留下的悬念是旧 import 写法在 3.x 是否仍兼容，留待 u1-l2 读 `__init__.py` 验证。
