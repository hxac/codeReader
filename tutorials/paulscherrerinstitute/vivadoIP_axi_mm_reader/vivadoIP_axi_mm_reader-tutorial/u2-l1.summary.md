本讲为进阶首讲，建立 IP 核自顶向下的架构地图。黑盒对外三类接口：s00_axi（从机，软件配置）、m00_axi（主机，只读不写，主动读别人）、m_axis 或 RdData（送出结果），外加 Trig 单拍启动脉冲与 DoneIrq 完成脉冲，共用单一时钟域。一次读周期：软件填 RegTable → 核心 FSM 在 Trig 或超时驱动下逐个取地址 → AXI 主机经 m00_axi 发单拍读事务 → 读回值连同 Last 位压入 FIFO → 按输出模式交出。wrapper 内五角色：AXI 从机解码器、RegTable 双端口 RAM、核心 FSM、AXI 主机、读数据 FIFO。输出由 g_axis/g_naxis 两个 generate 块切换：AXIS 直连 m_axis，AXIMM（默认 Output_g="AXIMM"，故默认无 m_axis 端口）把 FIFO 映射到 RdData/RdLast，软件读 RdData 才弹 FIFO，须先读 RdLast。超时换算为 ⌊ClkFrequencyHz×TimeoutUs_g/10⁶⌋，默认 100 MHz/100 µs 即 10000 拍。
