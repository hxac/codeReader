本讲是 fpga_base 学习手册的总览篇，只读 README、Changelog、License 等门面文件，不涉及 HDL 逻辑。核心结论：fpga_base 是 PSI 维护的 Vivado IP 核，定位为 FPGA 基础设施 IP，通过 AXI 寄存器向软件暴露版本读出、固件/软件编译日期时间、LED 与 DIP 开关四类基础能力，旨在避免各工程重复造轮子。它采用 PSI HDL Library License（LGPL 2.1 加固件例外），允许综合出的 FPGA bitstream 闭源商业分发。依赖分三类：TCL（PsiSim/PsiIpPackage/PsiUtil，多为开发期）、VHDL（psi_common，综合必需）、VivadoIp（自指）；依赖清单写在 README 中并由 scripts/dependencies.py 解析，格式不可乱改。版本从 1.0.0 演进到 1.4.0，主线为「硬件→软件驱动→EPICS→工程化→版本追溯」，对应后续单元。本讲引入的关键术语：FPGA、VHDL、IP 核、Vivado、AXI、IP-XACT（component.xml）、PSI；并埋下伏笔：AxiAddrWidth 固定 8 位、1.3.0 起改用 psi_common 的 AXI 从机（故 psi_common 不可或缺）。
