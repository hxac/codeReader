本讲是学习手册开篇，在不涉及源码的前提下建立项目整体认识。psi_multi_stream_daq 是 Paul Scherrer Institute 开发的多流数据采集 FPGA IP 核，把多路并行数据流经 DMA 写入系统内存（如 DDR）。本讲只读 README、Changelog、License 三个顶层文件，关键结论：维护者 Daniele Felici 与作者 Oliver Bründler 非同一人；许可证为 PSI HDL Library License（LGPL 加硬件例外），允许把本库编译进 FPGA 比特流后闭源分发；版本号采用 major.minor.bugfix 三段式语义，不兼容改动动 major、新功能动 minor、仅修 bug 动 bugfix。引入的关键术语：FPGA、VHDL、IP 核、SoC/PS、DMA、AXI（Slave 与 Master）、DAQ，这些将在后续讲义沿用。依赖经 README 中一段可被 dependencies.py 脚本解析的区块声明，目录结构本身即契约：运行时需 psi_common（≥3.0.0），仿真开发还需 psi_tb（≥3.0.0）与 PsiSim（≥2.1.0）。当前最新稳定版为 1.2.3（仅文档类变更），首次开源发布为 1.2.0。
