`vivadoIP_data_rec` 是 PSI 开发的通用多通道数据记录 IP 核，行为类似示波器抓波形：软件 Arm 后，环形存储先积累前触发（Pre-Trigger）样本，触发事件到来后再补记后触发（Post-Trigger）样本，完成一段窗口录制，软件经 AXI 总线读走。五大特性为：前/后触发记录、基于信号电平的自触发（self-trigger）、四种触发模式、最多 8 通道、可配置采样深度。四种触发模式分别由谁触发需牢记：normal 与 external-trigger 靠外部信号（后者可多路 OR，v1.1.0 扩展到最多 8 路，由 `TrigInputs_g` 决定，区别于数据通道数 `NumOfInputs_g`）；free-running 靠软件触发循环；self-trigger 靠数据落入设定范围。当前版本 v2.4 新增可选端口 `Trig_Out`，转发 IP 内部经裁决后实际使用的触发信号（受 `TrigForwarding_g` 控制），便于级联记录器或与其他逻辑同步；v2.3.2 修复了非二次幂 `MemoryDepth_g` 回绕 bug，该路径后续专讲。项目依赖 psi_common、psi_tb（≥3.0.0），PsiSim/PsiIpPackage 仅开发需要。权威细节查 `doc/data_rec.pdf`，口诀：README 是入口，PDF 是权威，源码是事实。
