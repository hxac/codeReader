本讲以 `data_rec_register_pkg` 为「唯一真相源」，给出记录器对外 13 个寄存器（`0x0000`–`0x0030`）与存储区起点 `Mem_Addr_c=0x0080` 的完整地址地图，并区分只写（`Cfg`、`SwTrig`）、只读（`Stat`、`TrigCnt`、`DoneTime`）与读写寄存器。引入三组概念：字节地址（4 字节对齐）与字地址之别；封装层用的 IPIC 信号组（`reg_rd/wr/wdata/rdata`、`mem_addr`）；以及字段位常量 `_Idx_c`（单 bit）与 `_Sft_c`（字段移位）——其中 `Cfg` 的 `Arm`(bit0)/`TrgCntClr`(bit16)、`TrigEna` 的 Ext/Sw/Self(bit0/1/2) 是后续讲义反复用到的位定义。两个工具函数 `ToWordAddr=ByteAddr/4` 与 `MemAddr=0x80+(ch·2^⌈log2(depth)⌉+spl)·4` 分别用于索引寄存器数组与计算任意通道/样本字节地址，通道间距向上取整到二次幂以换取高位直接当通道选择。另有两项关键机制：复位默认 `RegRstVal_c` 令外部触发使能上电为全 1；读状态寄存器且处于 Done 态时自动产生 `Ack` 脉冲拉回 Idle。
