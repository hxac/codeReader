本讲只读 README、Changelog、License 三份文档，不涉及 VHDL 源码。核心结论：vivadoIP_spi_simple 是 Paul Scherrer Institute（PSI）开源的 PSI HDL Library 家族成员，一个基于 AXI4-Lite 风格寄存器接口的简单、高可配置的 SPI Master IP-core，让 FPGA 内的 AXI 主机（如 Zynq ARM 核）读写一组寄存器即可完成 SPI 收发，免去手写时序；相比 Xilinx 官方 SPI IP 的差异化优势是可配置性（典型如任意传输宽度）。引入并供后续承接的关键术语：SPI 主从式四线通信（SCK/MOSI/MISO/CS_n，片选低电平有效，Master 生成时钟）、AXI4 寄存器接口、IP-core、PSI HDL Library（含 psi_common、psi_tb、PsiSim、PsiIpPackage 等成员）。人员与合规：作者为 Oliver Bründler，维护者为 Waldemar Koprek（PSI）；授权为 PSI HDL Library License，即 LGPL 叠加一条硬件例外，明确把 FPGA bitstream 视为可自由分发的 binary，可纳入商业闭源工程。Changelog 演进：1.0.0 首次实现 →1.1.0 新增 dependencies.py 依赖解析脚本、AXI slave 改用 psi_common 实现 →1.2.0 首个开源发布并加 License 头 →1.2.1 C 驱动兼容 C++ →1.2.2 支持 R/W 寄存器的 AXI 读回 →1.3.0 新增 LE（Latch Enable 锁存使能）输出。回归测试入口为 sim 目录下 Modelsim 的 source ./run.tcl。当前 HEAD（提交 fda4db7）含尚未发布到 Changelog 的 3-Wire SPI 开发中改动，属后续讲义范畴。
