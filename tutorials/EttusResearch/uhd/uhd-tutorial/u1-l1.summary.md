本讲建立 UHD 的「项目定位」认知，仅看仓库顶层门牌文件，不涉及源码逻辑。核心区分：USRP 是 Ettus Research 制造的 SDR 硬件平台（主板、子板、固件、FPGA 镜像），UHD 是运行于主机的自由开源驱动与 API，用统一接口屏蔽不同型号硬件差异。UHD 并非单一二进制，而是横跨主机与设备端的软件集合，仓库分六大目录：host/（主机驱动，本手册主角）、mpm/（设备端外设管理进程）、firmware/（单片机固件）、fpga/（FPGA 镜像，含 RFNoC/CHDR 通路）、images/（镜像打包）、tools/（调试）。它是底层库而非端到端应用，供 GNU Radio、srsRAN 等上层框架调用。版本号为四段式 MAJOR.API.ABI.PATCH，当前为 4.10.0.0 且 HEAD 处开发分支，并压成整数宏 UHD_VERSION 供版本兼容判断。UHD 与 mpm 默认 GPLv3，可申请替代许可证，fpga/ 另有规定；升级前须查 CHANGELOG 的 API Changes 一节。
