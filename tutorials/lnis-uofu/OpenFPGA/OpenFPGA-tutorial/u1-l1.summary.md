本讲是手册开篇，旨在建立全局地图。核心结论：OpenFPGA 是首个有流片验证（silicon proofs）的开源 FPGA IP 生成器——它帮用户「造 FPGA 并给出配套 CAD 工具」，而非「在已有 FPGA 上跑设计」。它用一份 XML 架构描述同时自动生成完整 FPGA fabric 的 Verilog 网表与比特流工具，把传统一年以上的手工流程压到约一天。端到端流程分四阶段：综合（Yosys）→ 布局布线打包（VPR）→ fabric 构建（OpenFPGA）→ 网表/比特流/约束生成（OpenFPGA）。输入为用户 Verilog + VPR 架构 XML + openfpga_arch.xml；输出为 fabric Verilog 网表、配置比特流、SDC 时序约束、可选 SPICE 网表与自检 testbench。项目由 FPGA-Verilog、FPGA-SDC、FPGA-Bitstream、FPGA-SPICE 四部分组成；版本号集中于 VERSION.md（当前 1.2.4307）作单一数据源。本讲引入的关键术语：FPGA、可编程 fabric、比特流、EDA、综合、布局布线、VPR、Yosys、fabric 构建，供后续讲义承接。
