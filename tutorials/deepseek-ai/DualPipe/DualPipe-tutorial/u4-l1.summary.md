本讲讲解 DualPipeV 的 V 型调度（DualPipe 的单向变体），覆盖拓扑简化、last rank 的 V 型转折、八步调度三块。DualPipeV 只在 first rank 喂数据，前向跑至 last rank 折返、反向跑回 first rank 算 loss，呈 V 字形；DualPipe 则从首末两端各喂一半。由此拓扑更简：删除 is_in_second_half 与 is_middle_rank，phase 含义对所有 rank 全局统一（0 恒前向、1 恒反向），各原语开头的 phase 翻转全部消失。

关键术语：phase（全局阶段）、detach 折返、梯度桥、折叠点（V 型顶点即 last rank）。V 型顶点靠两对本地产出零通信衔接——前向把 phase 0 输出经 detach().requires_grad_() 折成 phase 1 输入，反向再把 phase 1 输入梯度桥成 phase 0 输出梯度，使整体梯度与一张连通的自动微分图等价；loss 仅在 first rank 且 phase==1 计算。

八步调度骨架与零气泡机制（WeightGradStore）照搬 DualPipe，循环公式把「半量」替换为「全量」，step_4 折叠点守卫由 is_middle_rank 换成 is_last_rank。核心结论：相同模型深度下 DualPipeV 只需 PP/2 个设备（DualPipe 需 PP 个），气泡与激活显存不变。
