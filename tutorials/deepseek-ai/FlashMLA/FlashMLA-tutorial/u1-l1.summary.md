FlashMLA 是 DeepSeek 为 V3/V3.2 设计的高性能注意力 kernel 库，按「阶段（prefill/decode）× 稀疏性（dense/sparse）」划分为四类 kernel，其中 Sparse 服务 DSA 稀疏注意力、Sparse Decoding 采用 FP8 KV cache。关键术语：MLA（多头潜在注意力）只缓存压缩后的潜在向量、K 与 V 同源；README 以 head_dim 区分两种 MLA 模式——MQA（head_dim_k=576，含 64 维 RoPE，head_dim_v=512，用于 decode 与 sparse prefill）与 MHA（head_dim 128/128，用于 dense prefill）；FP8 KV cache 每 token 占 656 字节 = 512(fp8 NoPE)+16(scale)+128(bf16 RoPE，RoPE 不量化)。支持矩阵仅覆盖 SM90（H800）与 SM100（B200），且非对称分布（Dense Decoding 仅 SM90、Dense Prefill 仅 SM100）。性能同时报 GB/s（memory-bound）与 TFlops（compute-bound）；DeepSeek 解码因 h_q=128、不做 Tensor Parallel 而落 compute-bound，分界点约 h_q·s_q≈128。
