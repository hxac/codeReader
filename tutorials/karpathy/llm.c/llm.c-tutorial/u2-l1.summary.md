本讲是「前向各层」单元首篇，聚焦 Transformer 入口——编码层 encoder：把 (B,T) 整数 token id 变成 (B,T,C) 浮点张量。引入关键术语：token embedding（词嵌入表 wte，形状 (V,C)）、position embedding（位置嵌入表 wpe，形状 (maxT,C)）、gather（查表）。前向公式 out[b,t,:]=wte[inp[b,t],:]+wpe[t,:]：词向量给词义、位置向量给顺序，二者逐元素相加而非拼接，以保持残差流维度 C 恒定，产出 acts.encoded 即残差流的第 0 层。反向为 gather 的逆运算 scatter-add（散播累加）：同一 id 或位置会被多处读取，故 dwte/dwpe 须用 += 累加而不能 = 覆盖；inp 是整数不可导，因此没有 dinp。wte 还兼任最终输出投影（权重绑定 weight tying），其梯度由 matmul_backward 与 encoder_backward 两处共同累加，依赖每步开头的 gpt2_zero_grad 清零保证正确。所有多维张量都是「一维数组 + 行主序指针算术」，寻址公式统一为「基地址 + 行号 × 每行元素数」。
