本讲只讲「组装」：把 u2 实现的各层 `*_backward`（外加融合的 `crossentropy_softmax_backward`）按 `gpt2_forward` 的严格逆序调一遍，这是链式法则的强制要求。引入两个关键术语：反向链路的「点燃」指用 `1/(B*T)` 填充 `dlosses` 作起点梯度，经融合反向产出第一个真实梯度 `dlogits`；「残差梯度的合流」指残差加法节点的梯度等于「主流直通」（来自更深层）与「子层回流」（经 LayerNorm 倒回）两路之和。核心结论有三：一是反向必须从 `crossentropy_softmax_backward` 开始、以 `encoder_backward` 收尾，中间严格逆序；二是残差合流靠 `residual_backward` 与 `layernorm_backward` 都用 `+=` 累加实现，相邻层共享 `residual3` 缓冲接力；三是权重绑定的 `wte` 在 logits 投影与输入查表两处累加梯度。`gpt2_zero_grad` 在每步反向前清零权重与激活梯度，是 `+=` 累加约定的配套基础设施，省略则梯度跨步累积、训练发散。
