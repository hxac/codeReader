本讲聚焦 GPT-2 MLP 块里两个逐元素算子。**GELU** 是网络唯一的非线性来源，llm.c 采用 tanh 近似式 `0.5x(1+tanh(√(2/π)(x+0.044715x³)))`，与 PyTorch `NewGELU` 一致、误差约 10⁻⁵；反向由乘积法则手推出局部梯度，仅需缓存前向输入 `l_fch`、不需前向输出，且 `gelu_backward` 外包 `#pragma float_control` 以规避 `-Ofast` 的数值破坏。**残差连接**前向是逐元素相加 `out=inp1+inp2`；其核心直觉是反向「**梯度分流**」而非减半——一份 `dout` 原样复制给两分支，故残差主流梯度可无衰减地贯通深层网络。残差反向用 `+=`，因主流梯度 `dresidual` 会被多个算子反复累加，依赖 `gpt2_zero_grad` 清零。组装上，GELU 在 MLP「升维后、降维前」出现一次（作用于 B·T·4C），残差在「注意力后」「MLP 后」各出现一次（作用于 B·T·C）。
