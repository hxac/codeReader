llm.c 旨在用简单纯粹的 C/CUDA 复现 GPT-2/GPT-3 训练，兼顾教育性与工程性：主线代码追求简单可读，速度与高性能实验交给 dev/ 目录与 cuBLAS/cuDNN/NCCL 等库。根目录有三套分工明确的实现——train_gpt2.py（PyTorch 参考）负责产出 .bin 权重并充当正确性标尺，train_gpt2.c（纯 C/CPU 参考）把每一层算法写得最清楚，train_gpt2.cu（CUDA 主线）支持混合精度与多卡多节点、跑得最快。三者通过 .bin 权重、debug_state.bin 标准答案与单元测试形成正确性闭环。引入的关键术语包括：token、前向/反向/训练循环、Transformer、尺寸缩写 B/T/C/V/L、精度别名 floatX，以及 llmc/（CUDA 头文件库）、dev/cuda（多版本内核教学库）、dev/data、doc、scripts 等目录角色。另有 train_gpt2_fp32.cu 作为更简单、冻结的 CUDA 学习入口。
