本讲完成 Transformer 前向的最后一步：把 `lnf` 之后的 `logits(B,T,Vp)` 变成概率并计算损失，只涉及 `train_gpt2.c` 中三个算子。**`softmax_forward`** 利用 softmax 的平移不变性，先减去真实词表 `V` 上的最大值 `maxval` 防止指数溢出，求和归一化后把填充区 `[V,Vp)` 显式清零（`V=50257` 真实词表、`Vp=50304` 对齐填充，二者区别贯穿全讲）。**`crossentropy_forward`** 取目标 token 处的概率，单点损失为 `-log(probs[ix])`，在 `gpt2_forward` 里对 `B*T` 个位置求平均得到标量 `mean_loss`；`targets==NULL` 时以 `-1.0f` 作哨兵。核心是**融合反向** `crossentropy_softmax_backward`：把对 crossentropy 与 softmax 的两步求导合并，得出招牌结论 `∂L/∂x_k = p_k - 1_{k=target}`，该式既约掉了危险的 `1/p_target`（数值稳定），又省去物化 `dprobs`。反向由 `gpt2_backward` 用 `1/(BT)` 填充 `dlosses` 启动，故 `dlogits = (probs - onehot)/N`；目标 token 处因 `indicator=1` 比非目标位置少 `1/(BT)`，即「梯度减一」。全讲沿用 `+=` 累加加 `gpt2_zero_grad` 清零的约定，可用中心有限差分验证梯度。
