本讲是前向单元的收束篇，把前六讲的单算子按 pre-norm Transformer 块顺序串成 12 层完整前向，主角是 gpt2_forward。每个 block 内部用 residual2（注意力子层后）与 residual3（MLP 子层后）两个残差缓冲，各存 L 份，以「基地址 + l*B*T*C」偏移寻址；block 的输入残差第 0 层为 encoded，其余层取上一层的 residual3。所有层相关激活按 (L,...) 连续布局并用层偏移寻址，是「一次性 malloc + 指针排布」技巧在层维度的应用，须完整保存以供反向复用。循环外完成最终三步：不分层的 lnf、logits 投影（复用 wte、bias 传 NULL，即权重绑定）与 softmax，再由 crossentropy 对 B*T 位置求平均得到标量 mean_loss。函数依 targets 是否为 NULL 兼顾训练（算 loss）与生成（置 -1.0f 哨兵）两条路径，为下一单元的反向组装奠基。
