本讲聚焦 GPT-2 中出现频率最高的 LayerNorm（层归一化）算子的前向与反向实现。数学定义为对每个位置 (b,t) 的 C 维向量做「去均值、除标准差、再缩放平移」，即 $w\odot\frac{x-\mu}{\sqrt{\sigma^2+\epsilon}}+b$，其中引入关键术语：均值 $\mu$、方差 $\sigma^2$（除以 C 的总体方差）、防除零常数 $\epsilon$（取 1e-5），以及本讲最核心的中间量 rstd（倒数标准差，reciprocal standard deviation）。GPT-2 采用 pre-norm 位置，将 LayerNorm 置于每个 Transformer 块最前面，保持残差主路干净以提升训练稳定性。

前向用三重循环对每个位置依次算 $\mu$、$\sigma^2$、rstd 并写出输出，同时把 mean、rstd 缓存进形状 (B,T) 的激活缓冲区供反向复用，体现 checkpointing 的显存与算力取舍。反向中，dweight、dbias 直接由 dout 与 norm 给出；dinp 经解析化简后只需两次对通道维的归约（dnorm_mean 与 dnorm_norm_mean，对应 term2 修正减均值耦合、term3 修正除标准差耦合，加上直通 term1）即可写完全部梯度。该算子在 train_gpt2.c 中被调用三次（ln1/ln2/lnf），前向与反向顺序严格镜像，所有梯度使用 += 累加并依赖 gpt2_zero_grad 清零。
