本讲精读 GPT-2 中唯一跨时间位置混合信息的算子——因果自注意力，含前向与反向。`ln1` 输出经一次 `OC=3C` 的 matmul 得 `qkv(B,T,3C)`，按 `+0/+C/+2C` 偏移切出 Q/K/V；多头靠 `hs=C/NH` 与指针偏移 `h*hs` 虚拟切分，不物理分块。前向四趟扫描：`scale·Q·K` 打分并记 `maxval` 以稳定 softmax、`exp(减max)` 求和、归一化且对 `t2>t` 显式置零（即因果 mask，体现为 `t2<=t`）、用权重 `att` 加权 value 得 `atty`。关键术语：`preatt` 存原始打分、`att` 存归一化权重；反向真正只读 `att`，前向 `preatt` 在 CPU 反向中未被消费，`dpreatt/datt` 仅为反向局部 scratch。反向三步（value→softmax 雅可比→q@k）是前向严格逆序，梯度一律 `+=` 累加并依赖每步 `gpt2_zero_grad` 清零；其 `(B,NH,T,T)` 矩阵的显存压力为后续 Flash Attention 埋下伏笔。
