本讲义解析 hdl-modules 的 `sine_generator`（DDS/NCO）：核心是相位累加器，每拍加 `phase_increment`、溢出回绕对应正弦周期，输出频率公式为 `f_sine/f_clk = phase_increment / 2^phase_width`，由 `get_phase_increment` 计算，受 Nyquist 约束（最高位恒 0，省 1 个 LUT）。`phase_width = memory_address_width + 2 + phase_fractional_width`，「+2」为象限位。`sine_lookup` 用四分之一象限对称只存 [0, π/2) 样本，靠反序读取与象限取负还原全程正余弦，整数相位下 SNDR = SFDR = memory_data_width + 1 ENOB。分数相位截断小数位会引入杂散，钉在 address+1 ENOB；两条互斥补救路径——相位抖动（LFSR 伪随机加到小数位，把杂散摊成底噪，SFDR↑、SNDR↓）与一阶泰勒展开（`查表值 + 导数×误差`，全塞 DSP48，SFDR/SNDR 翻倍，末尾 `saturate_signed` 钳位）。关键术语：SFDR、SNDR、ENOB、相位抖动、泰勒展开。验证用 VUnit + Python FFT 后检查与 netlist 资源回归。
