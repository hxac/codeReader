本讲是手册首篇，建立对 hdl-modules 的全局认知。核心结论：它是一组可复用、高质量、同行评审的 VHDL-2008 构建块，面向 FPGA 工程师，以宽松的 BSD 3-Clause 开源，可放心用于商业项目。仓库实际含 14 个模块（axi、axi_lite、axi_stream、bfm、common、dma_axi_write_simple、fifo、hard_fifo、lfsr、math、register_file、resync、ring_buffer、sine_generator），但 README 速览只精选 12 个，完整清单以 modules/ 目录为准。三条设计哲学贯穿全项目：可复用与可移植、面积/资源优先优化（FIFO 是典型）、用 generic 开关功能以省资源——综合时 generic 为假的功能对应的 generate 块被删除，未启用特性零资源占用。质量被置于首位：同行评审加单元测试加真实设计验证。本讲引入的关键术语供后续承接：FPGA、VHDL-2008、构建块/IP 核、generic（类属参数）、资源占用、CDC（跨时钟域）、AXI/AXI-Stream/AXI-Lite、BFM（总线功能模型）。工程约定上，项目以 hdl_modules/about.py 为标语的单一信息源，README 是其生成产物；每个模块按 src（可综合）/test（测试台）/sim（BFM）严格分目录。
