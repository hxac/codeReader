本讲聚焦 AERIS-10 雷达的发射链入口，由三个模块构成：`radar_transmitter`（顶层接线盒，做 CDC、边沿检测与 SPI 3.3V/1.8V 电平转换穿通）、`plfm_chirp_controller_enhanced`（chirp 状态机核心，文件名为 `plfm_chirp_controller.v`）和 `dac_interface_enhanced`（DAC 输出级，文件名为 `dac_interface_single.v`）。核心结论是状态机用 7 个状态产生「16 个长 chirp + GUARD + 16 个短 chirp = 32 chirp/帧」的 staggered PRI 节拍，这正是 u4-l4 双 16 点 Doppler 子帧的来源。关键术语与参数：LFM/chirp 与脉冲压缩、PRI/PRF、TR switch、ODDR；长 chirp 时长 `T1_SAMPLES/FS = 3600/120MHz = 30µs`，长段 PRI≈167µs 对应最大非模糊距离≈25km，带宽 20MHz 给出距离分辨率 7.5m。实现要点：长 chirp 波形来自 `long_chirp_lut.mem`（3600×8 位 BRAM 同步读），短 chirp 为内联 8 位 LUT；`short_chirp_i/q.mem` 是接收端匹配滤波的高精度参考，不在发射通路上。DAC 用 ODDR 转发时钟并打包数据到 IOB，`chirp_valid` 低时回中点 `0x80`。`new_chirp` 脉冲经「2 级电平同步 + 边沿检测 + toggle-CDC」从异步 GPIO 安全搬入 120M FSM，而 `mixers_enable` 电平信号只用 3 级电平同步。重要事实：当前 HEAD 中 ADAR1000 的 `adar_tx/rx_load_*` 全部硬拉低，发射期真正驱动的是 `adar_tr_*` TR 位、RF 开关与混频器使能，相位装载由 STM32 经 SPI 完成。
