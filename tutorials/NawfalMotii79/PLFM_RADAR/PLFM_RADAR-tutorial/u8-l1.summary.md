本讲建立 V7 GUI 架构地图，覆盖包结构、六标签页、优雅降级三个最小模块。V7 为「薄启动器 GUI_V7_PyQt.py + v7 包」结构：v7/__init__.py 作门面集中转出口，用 try/except ImportError 分层加载，使无 PyQt6 环境也能导入纯算法模块。v7 九大模块按数据→硬件→处理→线程→地图→回放→软件镜像→AGC仿真→表现层单向依赖，agc_sim 为独立离线工具不在 __init__ 导出。models.py 是共享数据层，含 RadarTarget、WaveformConfig 等数据类与 5 个可选依赖标志。主窗口 RadarDashboard 用 QTabWidget 组六标签页（Main View、Map View、FPGA Control、AGC Monitor、Diagnostics、Settings），靠 QTimer 统一刷新。关键区别：FPGA Control 经 4 字节 opcode（如 CFAR alpha=0x23）操作 FPGA 寄存器，Settings 操作主机端 DSP。优雅降级三步法：models 探针置标志 → processing 条件导入并短路返回空 → dashboard 禁用控件且 Diagnostics 显示状态，缺 scipy/sklearn/filterpy 时不崩溃。
