本讲解析 `numpy.matrixlib.defmatrix.py` 中的 `bmat` 与其私有依赖 `_from_string`。`bmat` 按输入类型分发三条路径：字符串交 `_from_string`、嵌套序列走两次拼接、`ndarray` 直接包装，返回值统一为 `matrix`。`_from_string` 用三层分隔符 `;`（分行）> `,`/空格（分块，行内等价）解析布局，把每个 token 当**变量名**在作用域字典里查找（先 `ldict` 后 `gdict`，均未命中抛 `NameError`），与 `matrix.__new__` 所用、把 token 当字面量的 `_convert_from_string`（`ast.literal_eval`）职责不同。关键概念：块矩阵、`sys._getframe().f_back` 栈帧回溯捕获调用者作用域、`axis=-1`（行内横拼，要求行数相同）与 `axis=0`（行间纵拼，要求列数相同）。易错点：`ldict`/`gdict` 仅在字符串路径且 `gdict is not None` 时生效；只给 `gdict` 会让 `ldict=None` 触发 `TypeError`，只给 `ldict` 则被忽略而走栈帧。
