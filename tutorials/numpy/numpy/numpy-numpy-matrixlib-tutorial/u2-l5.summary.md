本讲剖析 `np.matrix` 在 defmatrix.py 中对 `*` 与 `**` 的运算符重载，核心结论是同一符号在 matrix 上语义被改写：`*` 表示矩阵乘（非逐元素乘），`**` 表示矩阵幂。`__mul__` 用三条分支实现：对 ndarray/list/tuple 先 `asmatrix`（一维会被升成行向量）再 `np.dot`，标量直接缩放，其余返回 `NotImplemented`。`__rmul__` 实现 `other * self`（参数顺序反转），支持 `3 * mA`。`__imul__`/`__ipow__` 用 `self[:] = self * other` 的切片写回技巧，就地修改并保住 matrix 类型与对象身份。`__pow__` 一行委托给 `numpy.linalg.matrix_power`，要求方阵与整数指数、支持负指数求逆、保持输入子类型。`__rpow__` 显式返回 `NotImplemented` 拒绝「标量的矩阵次幂」，两端皆失败后 Python 抛 `TypeError`。本讲引入的关键概念：NotImplemented 协议（双向尝试）、显式重写特殊方法从而绕过 ufunc 与 `__array_priority__`、切片写回保身份。
