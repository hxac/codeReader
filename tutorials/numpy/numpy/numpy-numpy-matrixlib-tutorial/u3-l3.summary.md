本讲精读 `numpy/matrixlib/defmatrix.py` 中 `matrix` 类的五个只读属性 T/H/I/A/A1，建立在「matrix 恒二维、继承 ndarray 的 transpose/conjugate、视图经 `__array_finalize__` 收尾仍是 matrix」这一前提下。**T** 委托 `transpose()`，永不共轭；**H** 是共轭转置，其关键判定是 dtype 级别的 `issubclass(self.dtype.type, N.complexfloating)`——仅复数 dtype 才多走一步 `.conjugate()`，实矩阵退化为转置，与元素虚部是否为零无关。**I** 按 `M == N` 分发：方阵走 `numpy.linalg.inv`（奇异抛 `LinAlgError`），非方阵走 `numpy.linalg.pinv` 返回 Moore-Penrose 伪逆，两者均为函数体内延迟导入，结果经 `asmatrix` 包回 matrix。**A / A1** 反向脱壳为 `ndarray`：`A` 经 `__array__()` 返回同形状视图，`A1` 再 `.ravel()` 降至真一维，区别于 matrix 自身 `ravel()` 的 `(1,N)` 二维保形。此外源码以 `getT = T.fget` 等把 property 的 getter 复用成方法，作向后兼容别名。
