本讲是手册收官篇，演示为自定义数组函数交付「实现+注解+测试」三件套，含三个模块。模块一：把 ArrayLike（入参）、DTypeLike（配置 dtype）、NDArray[np.floating]（出参）三大公共别名拼进签名，强调用公共别名、不展开私有实现。模块二：用 @overload 修复精度丢失，按「窄分支前置、宽分支兜底」声明存根，使 dtype=type[np.float32] 时返回被精确推断为 NDArray[np.float32]；消歧全靠声明顺序，并区分 @overload（一一映射/精度提升）与上界标量类 TypeVar（同型进出）。模块三：把仅产 note 的 reveal_type 换成不匹配即报 error 的 assert_type，仿 reveal 目录写 .pyi 桩夹具，经 mypy 把期望返回类型固化为类型回归测试。
