本讲解析 `_arraysetops_impl.py` 的 11 个集合运算函数（均基于排序）。去重内核 `unique`/`_unique1d` 有两套引擎：C 实现的哈希快路 `_unique_hash` 与排序慢路；只要请求 `return_index`/`return_inverse`/`return_counts` 任一或输入为 masked 数组，即强制走排序路。关键恒等式 `u[inv]==原数组` 由掩码前缀和配合 `perm` 散播实现无损重建；`axis` 借「结构化视图」把多维去重规约为一维。Array API 形式 `unique_all/counts/inverse/values` 为薄封装，返回 NamedTuple 且固定 `equal_nan=False`，其中 `unique_values` 传 `sorted=False` 走哈希快路、不保证排序。`isin` 提供 `table`（整数/布尔、受 6 倍内存预算约束）、`sort`（通用）、小数组逐元素三套算法，`kind=None` 自动选择。`intersect1d`/`union1d`/`setdiff1d`/`setxor1d` 由前两者积木拼装，`assume_unique` 可声明免去预处理。核心直觉：排序是算法副产物，故集合结果天然有序。
