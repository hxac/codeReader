本讲覆盖 `numpy/lib/_shape_base_impl.py` 与 `_twodim_base_impl.py` 两文件中的矩阵构造与翻转工具。核心结论：`tile` 是「复印机」按份数复制元素、`kron` 是「放大镜」把每个元素撑成一块 `b`，`kron` 输出形状逐轴相乘，靠「奇偶位置交错插轴 + 广播相乘 + reshape」实现。`eye`/`diag`/`diagflat` 共用「跨步写扁平内存」招数 `res[:n-k].flat[i::n+1]=data`，`diag` 为双向函数（1D 构造、2D 抽取），`diagflat` 先 `ravel()`。`vander` 用「反向视图 + 一次 `multiply.accumulate` 累乘」生成范德蒙矩阵。`fliplr`/`flipud` 返回步长取负的视图，为 O(1)，前者要求 2D、后者接受 1D。`tri` 用 `greater_equal.outer` 一次生成三角掩码，`tril` 传 `k`、`triu` 传 `k-1`（上三角为下三角 k-1 的补集）；`mask_indices` 是通用「掩码→下标」接口，与特化的 `tril_indices`/`triu_indices` 结果等价。关键术语：跨步切片写对角线、交错插轴、反向视图累乘、outer 生成掩码、步长取负视图、`@finalize_array_function_like` 支持 NEP-35 `like=` 参数。
