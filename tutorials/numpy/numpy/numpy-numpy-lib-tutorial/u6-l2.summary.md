本讲精读 `_function_base_impl.py` 的四个数值信号函数：`interp`（一维线性插值）、`unwrap`（相位解卷绕）、`angle`（复数辐角）、`sort_complex`（复数排序）。核心结论：`interp` 的二分搜索与线性插值由 C 内核完成，Python 层仅做实/复内核分流并处理 `period`——用 `x%period` 折叠、在采样点两端拼接跨周期镜像点，把角度插值归约为普通插值；`unwrap` 按「差分折回半周期、算修正量、小跳变清零、修正量累加和加回」还原连续相位，`discont<period/2` 时按 `period/2` 处理；`angle` 是 `arctan2(虚部,实部)` 的薄封装，纯实数输入虚部视为 0，结果落在 $(-\pi,\pi]$；`sort_complex` 借复数「先实部后虚部」字典序排序并强制返回复数 dtype。本讲引入术语：相位卷绕/解卷绕、辐角、`discont` 阈值、dispatcher+impl 双函数写法（NEP-18）；`unwrap` 内部复用了上一讲的 `diff`。综合实践以 `angle→unwrap→interp` 串联还原连续相位。
