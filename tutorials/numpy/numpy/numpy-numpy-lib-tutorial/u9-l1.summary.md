本讲源码集中在 `_nanfunctions_impl.py`，确立 NaN 聚合的「替换-聚合-还原」三段式范式与五个私有基础设施：`_nan_mask` 返回非 NaN 掩码、`_replace_nan` 将 NaN 替换为哨兵值并回传掩码、`_copyto` 把全 NaN 切片结果还原为 NaN、`_remove_nan_1d` 不保序就地剔除 NaN（供排序型统计）、`_divide_by_count` 用 errstate 抑制 0/0 警告。关键术语：单位元（sum→0、prod→1、min→+inf、max→-inf）、类型短路（整数/布尔不可能含 NaN，掩码返回 None）、对象数组用 not_equal(a,a) 利用 NaN 不自等兜底。结论：`nansum`/`nanprod`/`nancumsum`/`nancumprod` 各仅两行，全 NaN 切片返回单位元；`nanmin`/`nanmax` 走快慢双路径，普通 ndarray 用天生忽略 NaN 的 fmin/fmax.reduce，子类与对象数组回退到 _replace_nan(±inf)+amin/amax 并还原全 NaN 切片。nanmean/nanvar/nanmedian/nanpercentile 留待 u9-l2，将复用本讲全部基础设施。
