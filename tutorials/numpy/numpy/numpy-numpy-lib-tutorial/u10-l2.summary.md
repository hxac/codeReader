本讲聚焦 `numpy/lib/_polynomial_impl.py` 单文件，讲解 numpy 旧式多项式 API。核心约定是系数按**降幂**排列为一维数组（与新 `numpy.polynomial` 升幂相反）。`polyval` 用 Horner 嵌套求值（`y=y*x+pv`）把复杂度降到 O(n)，当 `x` 为 poly1d 时返回多项式复合；`poly`（卷积连乘一次因子）与 `roots`（构造友矩阵求特征值）互为逆运算，构成「根↔系数」对偶，方阵输入还可得特征多项式；`polyadd/polysub/polymul/polydiv` 与 `polyint/polyder` 共享「truepoly 标志 + atleast_1d 对齐 + 操纵系数」写法，加减靠高位补零、乘法用卷积、除法用长除法、微积分靠系数乘除降序自然数；`polyfit` 以范德蒙矩阵配合 `lstsq` 做最小二乘，外围含 `rcond=len(x)*eps` 截断、列 2-范数缩放、`RankWarning` 与按 χ²/dof 缩放的协方差。`poly1d` 是顶层封装，提供属性别名（`c/r/o`）、运算符重载（委托四则与微积分函数）、`__call__` 委托 `polyval`，并通过 `__array__` 伪装成数组——需注意 `np.square(p)` 是系数逐元素平方而非多项式平方。
