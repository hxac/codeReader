本讲精读 `numpy/lib/_function_base_impl.py` 中的三个数值微积分函数，三者共享同一底层招数：**沿指定轴构造 `slice1=1:` 与 `slice2=:-1` 两个错位切片，再做逐元素运算**。它们均遵循 dispatcher+impl 双函数写法（NEP-18 派发），并由顶层 `numpy/__init__.py` 直接挂到 `np.` 命名空间（不经薄模块）。

`np.diff` 求 n 阶**离散差分**：一阶为 `a[1:]-a[:-1]`，n 阶递归循环 n 次；布尔数组改用 `not_equal`（而非 `subtract`）；`prepend`/`append` 经广播与 `concatenate` 扩展边界；输出沿轴缩短 n，`n=0` 零拷贝返回原对象。

`np.gradient` 估计 N 维**数值梯度**：内点用二阶精度**中心差分** `(f[2:]-f[:-2])/(2h)`，边界由 `edge_order` 选一阶或二阶**单边差分**；均匀与非均匀间距走两套系数公式；输出与输入**同形**，单轴返回数组、多轴返回元组；整数输入被转 `float64` 防止模运算。

`np.trapezoid` 用**复合梯形法则**求定积分：核心为 `(d*(y[1:]+y[:-1])/2).sum(axis)`，宽度 `d` 来自 `diff(x)` 或标量 `dx`；对 `matrix` 等子类有 `add.reduce` 回退。

三者形成离散微积分对偶：微分（gradient/diff，输出**同形或缩短**）与积分（trapezoid，输出**降一维**）。关键术语：错位切片、中心/单边差分、edge_order、复合梯形、均匀/非均匀间距。
