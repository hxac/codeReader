本讲以使用直觉为导向速览 Generator 常用方法，源码聚焦 _generator.pyx 与类型存根 _generator.pyi。核心结论：所有方法共享一套公共参数——size（None 返标量、给定返数组）、dtype、out、endpoint（是否含上界）、axis，认清即可举一反三。三个最小模块：random 产 [0,1) 浮点、integers 产区间整数，均按 dtype 分派、integers 默认用更快的 Lemire 无偏算法；standard_normal 与 random 同构（Ziggurat），uniform 经 x=low+(high-low)·U 推广区间并对宽度施非负约束；choice 先造索引再索引数组，有放回等概率复用 integers、无放回用 Floyd 算法或尾部洗牌，shuffle 为原地 Fisher-Yates（返回 None）、permutation 返副本。引入术语：半开区间、Lemire、Ziggurat、Fisher-Yates、Floyd 算法、double_fill/cont 通用模板、可复现性边界；方法最终委托 _common.pyx 模板与 C 采样函数，三层结构留待第 5 单元。
