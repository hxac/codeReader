本讲讲解 numpy.fft._helper.py 中的两个纯 Python helper：fftshift 与 ifftshift。它们不计算任何傅里叶变换，只沿指定轴做循环平移 np.roll，把零频（DC）从索引 0 搬到频谱中心以便观察。核心公式：fftshift 取 shift = dim // 2，ifftshift 取 shift = -(dim // 2)，二者仅差一个负号；由 roll(roll(x,s),-s)==x，ifftshift 是 fftshift 的精确逆，对任意形状与奇偶长度恒成立。偶长度下二者完全相同（互为对合），奇长度下相差一个样本，根因是 floor(n/2) 与 -floor(n/2) mod n 奇长度时差 1。axes 控制多维平移范围：None 平移全部轴，整数只平移一个轴，元组/列表平移指定轴且各轴平移量由各自长度独立决定。本讲引入的关键术语有：循环平移 roll、零频居中、standard 频率排列、对合（involution）、逆运算。典型用法为 fft 后同时对频谱与 fftfreq 频率轴做 fftshift 以便观察，再 ifftshift 摆回 standard 排列喂给 ifft 还原。
