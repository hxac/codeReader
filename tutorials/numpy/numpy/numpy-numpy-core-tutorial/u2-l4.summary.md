本讲精读 numpy/core/__init__.py，即子模块垫片的「宿主」包入口。`__all__` 声明 17 个子模块名却顶部一个都不 import，以「只声明、不绑定」迫使访问走 `__getattr__`，每次都打弃用警告，并用 `# noqa: F822` 压制 linter 误报。包级 `__getattr__` 用 `getattr(_core, name)` 把名字解析到整个 `_core`，既接子模块名也接顶层类型名，缺失时由 getattr 自抛 AttributeError 而不报警；与子模块级转发差别在解析目标范围与是否给 `_raise_warning` 传 submodule，据此警告文本写 numpy.core 或 numpy.core.numeric。`_ufunc_reconstruct` 是全文件唯一 eager 绑定，因 1.20 前旧 pickle 硬编码该路径，必须 import 后直接命中、不报警、不懒加载。类型存根 `.pyi` 走相反路线，全量 import 子模块、略去该函数。
