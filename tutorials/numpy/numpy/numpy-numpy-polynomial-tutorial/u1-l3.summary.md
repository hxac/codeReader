本讲以标准幂基便捷类 `Polynomial` 为对象，演示六大便捷类共用的面向对象用法，含四个最小模块：创建与求值、算术与幂运算、fit 拟合、deriv/integ/roots。核心结论：`Polynomial(coef)` 把系数（低次到高次）连同 `domain`/`window`/`symbol` 封装成对象，默认 `domain==window==[-1,1]`；`p(x)` 求值前先用 `mapdomain` 把自变量从 `domain` 经 `off+scl·x` 线性映射到 `window`，再委托 `polyval` 用 Horner 法计算。算术运算符经 `_get_coefficients` 校验「类型/域/窗/符号」同类后委托虚函数 `_add/_mul/_div/_pow`，`maxpower=100` 防膨胀，`/` 仅对标量，多项式带余除法走 `divmod`。`fit` 默认自动推断 `domain`、把数据映射到 `window` 后构造范德蒙矩阵做最小二乘求解，数值更稳；返回系数是 window 变量下的，须用 `p(x)` 求值而非 `polyval(x,p.coef)`，要标准基系数用 `convert()`。`deriv`、`integ`、`roots` 均尊重映射：分别乘 `scl`、`1/scl`、把友矩阵特征值从 window 映回 domain。关键术语：`ABCPolyBase`、虚函数委托、`mapparms`、Horner 法、范德蒙矩阵、最小二乘、友矩阵、同类约束、`convert`。`Chebyshev` 等其余便捷类接口完全一致，差异仅在基与默认 domain。
