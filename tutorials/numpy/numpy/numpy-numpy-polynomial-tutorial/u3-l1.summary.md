本讲义拆解 `numpy.polynomial` 子包的「地基」文件 `polyutils.py`，其 `__all__` 暴露七个公开工具，按「规整 → 去零 → 映射 → 格式化」四个最小模块展开。`as_series` 是全包输入总闸，负责转 1-D、校验非空、可选去尾零、用 `np.common_type` 求公共 dtype（失败则退化为 object 以支持 `Decimal`）并返回副本，惯用法 `[c]=pu.as_series([c])` 统一了输入规整。去尾零有两套：`trimseq` 是轻量结构去零（不复制、保 dtype、保底返回单个零），`trimcoef` 是带容差 `tol` 的数值去零（严格 `>`，先 `as_series` 再清洗返回副本），经 `polytrim=pu.trimcoef` 别名进入各正交族。坐标映射三件套支撑 domain↔window 的仿射映射：`mapparms(old,new)` 求参数 `off+scl·x`，`mapdomain` 作用到点集（保留 ndarray 子类），`getdomain` 用 min/max（复数取包围矩形）猜 domain，三者共同支撑 `_polybase.py` 的 `__call__/roots/fit`。`format_float` 是打印子系统底层原语，非浮点走 `str`、NaN/Inf 走全局字符串、浮点走 Dragon4 并尊重 `np.get_printoptions()`，`parens` 仅科学计数法分支加括号。
