本讲是 numpy.polynomial 学习手册首篇，聚焦子包入口 `__init__.py`，分三个最小模块讲清定位、表示约定与便捷类。核心结论：该子包是"高效处理多项式"的独立子包，核心理念为"一切操作即系数操作"，独立于 legacy 的 `numpy.poly1d`——后者系数降幂，新 API 系数升幂。引入的关键约定：多项式用 1-D 系数数组表示，下标即次数，`coef[i]` 为第 i 次项基函数 `P_i(x)` 的系数，即 `p(x)=c0·P0+c1·P1+…`；同一系数数组在不同基（如标准幂基与 Chebyshev 基）下代表不同函数。引入术语：基(basis)、系数、次数(degree)、便捷类(convenience class)。子包为六种基各提供一个便捷类 `Polynomial/Chebyshev/Legendre/Laguerre/Hermite/HermiteE`，接口一致、可直接从 `numpy.polynomial` 命名空间取用，是首选入口；各类自带 `domain/window/basis_name/maxpower` 四个类属性，其中仅 `Laguerre` 默认 domain 为 `[0,1]`，其余为 `[-1,1]`。
