本讲聚焦 `numpy.polynomial.polynomial` 的求值操作，核心是 `polyval`。它用 Horner 法（秦九韶算法）将多项式改写为嵌套形式，从最高次系数起步、每步「乘 x、加低一次系数」，把求值复杂度从朴素逐项求和的 O(n²) 降到 O(n)，且因避免单独计算高次幂而数值更稳；实现中以 `c[-1] + x*0` 一次性完成形状与 dtype 对齐。`tensor` 参数决定多维系数（第 0 轴为次数轴，其余轴枚举多个多项式）与点 x 的组合方式：`True` 为外积（每个多项式作用于每个点，输出 `c.shape[1:]+x.shape`），`False` 为逐元素配对（输出 `c.shape[1:]`），机制是给 c 追加 `x.ndim` 个尾随 1 轴。`polyvalfromroots` 直接用 ∏(x−rₙ) 连乘求值，省去基转换、数值更稳，且多一道 `x.ndim<r.ndim` 护栏。`polyval2d/3d/nd` 与 `polygrid2d/3d` 均委托共享引擎 `_valnd`/`_gridnd`，将 N 维求值拆为逐维 1 维 `polyval` 折叠：配对点仅首维用 `tensor=True`，笛卡尔积全程用 `tensor=True`。最后给出范德蒙等价 `dot(polyvander(x,n),c)≡polyval(x,c)`，作为连接求值与最小二乘拟合的桥梁。
