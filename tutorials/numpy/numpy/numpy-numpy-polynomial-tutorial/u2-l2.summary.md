本讲解析 `ABCPolyBase` 的「双区间 + 线性映射」设计支柱：便捷类对象除 `coef` 外携带 `domain`（对外用户坐标区间）与 `window`（对内基函数参考区间），系数始终按 window 变量表达。底层 `polyutils.mapparms(old, new)` 算出仿射映射 L(x)=off+scl·x 把 old 端点映到 new 端点，`mapdomain` 为其批量作用。各操作坐标换算方向不同：求值 `__call__` 先 domain→window 再委托 `_val`；求根 `roots` 把 `_roots`（友矩阵特征值）结果 window→domain 反向返回；`fit` 用 `getdomain` 取数据范围映到 window 做最小二乘（数值稳定关键）；`deriv` 乘 scl、`integ` 乘 1/scl 体现链式法则。口诀「输入往 window 走，输出往 domain 回」。六族中唯独 `Laguerre` 默认 domain/window 为 [0,1]（源于半无界区间正交性），其余五族为 [-1,1]；默认 domain==window 时映射恒等，同系数配不同区间即代表不同函数。
