本讲讲解 MaskedArray 的持久化机制，覆盖 pickle 序列化、空壳重建与深拷贝三个最小模块。pickle 路径上，`__reduce__` 返回 `(_mareconstruct, args, state)` 三元组；`__getstate__` 复用 ndarray 的五元组状态（version/shape/dtype/字节序/raw 字节），再追加 mask 字节与 fill_value，拼成七元组状态。反序列化遵循「建壳+灌态」分离：`_mareconstruct` 先用 `self.__class__` 与 `_baseclass` 造类型正确的空壳，保住子类身份与 recarray 等基类；`__setstate__` 随后灌入状态。关键桥梁是 `make_mask_descr`——mask 的 dtype 不单独存储，而从 data 的 dtype 推导，保证结构化屏蔽自洽往返。深拷贝走另一条路：`__deepcopy__` 用 copy=True 造独立 data/mask，深拷贝 `__dict__` 全部簿记属性，并对 object dtype 额外逐元素深拷贝，以避免嵌套可变对象被共享污染。mrecords 的 `_mrreconstruct` 是同构但独立的实现，仅 view 方式与手写 mask dtype 不同。
