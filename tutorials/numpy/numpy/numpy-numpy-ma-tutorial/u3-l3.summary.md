本讲义聚焦 `numpy/ma/core.py` 中两件事：`np.ma.masked` 单例的不可变保护，以及掩码数组的打印管线。

`masked` 是 `MaskedConstant` 的全局唯一实例，代表「取一个被屏蔽标量位置」的统一返回值，因此必须全局唯一以保证 `x is np.ma.masked` 判断可靠。唯一与不可变通过一组重写方法实现：`__new__` 用类属性 `__singleton` 缓存、首构造把只读的 `data(np.array(0.))`/`mask(np.array(True))` 视图化存入；`__setattr__` 拦截写、`copy`/就地运算/`__reduce__` 全返回 `self`。运算传播靠「降级」守护唯一性：`__array_finalize__` 把非正牌实例就地改回 `MaskedArray`，`__array_wrap__` 先 `view(MaskedArray)` 再转发——可观察结果是「标量运算保单例、数组运算降级为全屏蔽 `MaskedArray`」。

显示符号由全局 `masked_print_option`（`_MaskedPrintOption` 实例，默认 `'--'`）管理，`set_display` 同时影响数组屏蔽位与单例自身的字符串，`enable(0)` 退回填充值显示。打印链路为 `__str__`/`__repr__` → `_insert_masked_print` → `_recursive_printoption`：先对超过阈值（多维 100、一维 1500）的轴做「四角提取」省去大数组的 object 装箱开销，再用 `_replace_dtype_fields(...,"O")` 转 object、`np.copyto(where=mask)` 填符号，最后交 `np.array2string` 渲染出带 `...` 的字符串。`__repr__` 复用同一管线渲染 `data`，并按单行/多行规则排版关键字。
