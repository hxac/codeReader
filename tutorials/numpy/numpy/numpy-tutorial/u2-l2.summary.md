本讲围绕三个源码文件讲清 NumPy 类型体系。numerictypes.py 定义以 generic 为根的标量类型继承树：bool 不经 number 节点，整数分有符号/无符号，浮点与复数归入 inexact；issubdtype 本质是把参数归一化为标量类型后调用 issubclass，故同族不同宽度的类型互不为子类型（float64 非 float32 的子类型）。dtypes.py 是公开 DType 类的薄容器，DType 类由 C 层动态注册；须区分三层对象——标量类型（np.float64）、dtype 实例（np.dtype("float64")）、DType 类（np.dtypes.Float64DType），后两者经 type(d) 相连，前两者经 d.type 相连。_type_aliases.py 用 allTypes、sctypeDict、sctypes 三张表搭起别名体系：double/int_ 等入前两张表，float/int 等 Python 名只入 sctypeDict 以支持字符串查询，sctypes 按族分组供 isdtype 使用。dtype 的 kind/char/num/itemsize/str 共同精确刻画每个类型，为下一讲类型提升规则奠基。
