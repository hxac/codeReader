本讲义只读 README、pyproject.toml、numpy/__init__.py 三类元信息文件。结论：NumPy 是 Python 科学计算的基础包，定位为底层依赖而非终端框架，提供四大核心能力——N 维数组（ndarray）、广播函数（ufunc）、C/C++/Fortran 集成工具（C-API、f2py）、线性代数/傅里叶变换/随机数子系统。pyproject.toml 是包的「身份证」：构建后端为 meson-python，依赖含 Cython，要求 Python≥3.12，版本 2.6.0.dev0；真实 version.py 由 gitversion.py 构建时生成，源码树仅有存根 version.pyi。numpy/__init__.py 把 _core 的 C 扩展对象与 lib 的纯 Python 工具汇聚到 np. 命名空间，用 __all__ 并集维护公开 API；linalg/fft/random 等子系统由 __getattr__ 懒加载，导入时跑 _sanity_check 防 BLAS ABI 错误，并注册 numpy.test()。
