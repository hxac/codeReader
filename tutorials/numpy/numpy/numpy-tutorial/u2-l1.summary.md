本讲系统梳理 NumPy 数组创建方式，按实现位置分三层。C 原语层含 array、asarray、asanyarray、empty、zeros、arange，由 C 扩展 _multiarray_umath 提供、经 multiarray.py 再导出，Python 层无 def。Python 组合层含 ones、full、identity、*_like，定义在 numeric.py，统一用「empty 申请未初始化内存、copyto 广播填充」拼装。数值区间层 linspace 等在 function_base.py，用 arange 生成下标再缩放。核心结论：arange、zeros、empty 不在 numeric.py，不能凭名猜文件。归一化家族中，array 默认 copy=True、subok=False 总复制且强制基类，asarray 默认 copy=None 按需复制，asanyarray 放行子类，require 逐条检查内存 flag 不满足则复制。引入术语 array_like、视图与拷贝、copy 三态、subok。底层 ctors.c 为总装车间，PyArray_NewFromDescr 是组装原语，PyArray_FromAny 是 array/asarray 总入口，输入已是合适 ndarray 且无约束时走 Py_NewRef 快速路径不复制；empty 与 zeros 仅差 _NPY_ARRAY_ZEROED 清零标志。
