本讲围绕 NumPy 核心对象 ndarray 拆解三个模块。其一，ndarray 并非 Python 写的类，而是 C 扩展 _multiarray_umath 中定义的内置类型，经 multiarray.py、_core/__init__.py、numpy/__init__.py 三跳再导出为 np.ndarray，运行时各处指向同一对象。其二，创建函数有分工：array/asarray/empty/zeros 是 C 层底层原语，ones/full/identity 则是 numeric.py 中用 empty 申请内存、再以 copyto 填充组合出的纯 Python 薄封装。其三，shape、strides、dtype、ndim、T 等属性由 getset.c 中 array_getsetlist[] 登记的 getter 在访问时动态计算返回，而非普通实例变量。核心结论：strides 单位是字节，C 连续数组满足 strides[n-1]=itemsize、strides[i]=strides[i+1]*shape[i+1]，元素字节偏移为各下标乘对应 strides 之和；转置、切片、广播常不复制数据，本质是仅改写 shape 与 strides 两个小数组而底层数据不动。本讲引入的关键术语：ndarray、shape、ndim、dtype、strides、itemsize、flags、C/F 连续、视图、getset、再导出。
