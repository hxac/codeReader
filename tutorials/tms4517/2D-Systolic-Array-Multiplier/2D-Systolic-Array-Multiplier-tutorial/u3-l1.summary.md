本讲精读 Verilator C++ 测试平台 `tb/tb_topSystolicArray.cpp`，拆为三个最小模块。其一，主循环与节拍：用 `i_clk ^= 1` 造时钟，`dut_reset` 在 `sim_time=3,4` 拉高异步复位 `i_arst`，`toggle_i_validInput` 每 `3N+3` 个上升沿触发一轮乘法，输入在 `eval()` 之后驱动、下一沿采样。其二，宽端口打包：Verilator 把超 64 位 packed 信号拆成小端 32 位字数组；8 位输入元素须按 `i_a[i] |= (elem << (8*j))` 四字节塞一字，而 32 位输出元素可按 `o_c[(N*i)+j]` 直读，形成「输入打包、输出直读」的不对称。其三，软件参考模型校验：`calculateResultMatrix` 三重循环算期望 `C=A×B`，`verifyOutputMatrix` 在 `o_validResult` 单拍脉冲时逐元素比对，不一致即 fail-fast 退出。波形由 `VerilatedVcdC` 每拍 `dump`，与 Makefile 的 `--trace`、`+verilator+rand+reset+2` 配套。关键术语：DUT、`VtopSystolicArray`、packed 数组、`numArrays`、`o_validResult`、宏 `N`。
