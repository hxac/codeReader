本讲深入脉动阵列的最小细胞——处理单元 PE（Processing Element），它是一个 8 位乘、32 位累加的 MAC（乘加）单元。核心含三模块：其一，MAC 数据通路，组合乘法器 mult 算 i_a·i_b，累加寄存器对 mac_d/mac_q（约定 _d 为下一拍值、_q 为当前值）存中间和，o_y 持续等于 mac_q；关键式 mac_d = i_doProcess ? mac_q+mult : 0，清零在开关拉低后的下一拍才生效，而非当拍。其二，输入寄存与透传，a_q/b_q 寄存后经 o_a/o_b 送右、下邻居，每跨一个 PE 延迟一拍，正是脉动错峰的来源。其三，门控与复位，i_doProcess「一信号两用」，同时控累加/清零与流动/冻结使二者同步，i_arst 为最高优先级异步复位。全阵列共享顶层生成的 doProcess_q，一次乘法持续约 3N-2 拍后统一拉低，令所有 PE 同时清零、冻结，准备下一轮。
