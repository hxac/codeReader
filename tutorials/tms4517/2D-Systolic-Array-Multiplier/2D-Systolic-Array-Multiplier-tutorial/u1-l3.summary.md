本讲精读顶层模块 topSystolicArray.sv 的端口与数据通路，作为进阶层 u2 各黑盒的承接基础。覆盖三个最小模块：端口与时钟复位约定、validInput→矩阵变换→脉动阵列→validResult 的完整通路、接口时序波形对照。

关键术语：顶层模块封装全部内部逻辑，对外接口刻意极简；所有寄存器在时钟上升沿更新，复位 i_arst 为异步、高有效；valid 采用单向触发握手，无 ready 信号、不可反压或中途取消。

端口共七支：i_clk、i_arst；输入 i_a/i_b（N×N×8 二维打包数组，每元素 8 位）加 i_validInput；输出 o_c（N×N×32，每元素 32 位）与 o_validResult。内部四部分：编译期参数检查（N_VALID 配 $error，要求 2<N<257）、矩阵变换（生成 row_q/col_q）、脉动阵列实例（由 doProcess_q 门控）、控制计数器。一次乘法耗时 3N−2 拍（localparam MULT_CYCLES，N=4 时为 10），o_validResult 为单拍脉冲，使用者应在该拍采样 o_c；该周期数同时驱动计数器计满阈值与完成脉冲的点亮条件。
