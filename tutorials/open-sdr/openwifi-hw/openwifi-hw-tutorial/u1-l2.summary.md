本讲义拆开 openwifi-hw 仓库,讲清三件事。其一目录结构:顶层为说明/授权文件与一次性准备脚本(prepare_*.sh、get_ip_openofdm_rx.sh),核心是 ip/ 与 boards/ 两目录。ip/ 下有且仅有 6 个自定义 WiFi IP:xpu、tx_intf、rx_intf、openofdm_tx、openofdm_rx、side_ch(README 白名单确认),另含跨 IP 共享的 board_def.v 与若干 Tcl 脚本;boards/ 按 BOARD_NAME 每板一目录,入口为 boards/openwifi.tcl。其二两个 git 子模块:adi-hdl(来自 analogdevicesinc/hdl,提供 AD9361 射频底座)与 ip/openofdm_rx(来自 open-sdr/openofdm 的 dot11zynq 分支,提供 OFDM 接收机);子模块只记录地址与 commit,克隆后默认为空,须经 get_ip_openofdm_rx.sh 或子模块初始化填充。其三软硬件边界:引入 PL(FPGA 可编程逻辑)与 PS(ARM 处理系统)两概念,本仓库为纯 PL 侧设计,产物是 .xsa/.ltx,PS 侧驱动在另一仓库 openwifi。
