本讲讲解三个 Python 脚本如何离线生成 OpenOFDM 的查找表（LUT），灌入 ROM 供硬件查表。`gen_atan_lut.py` 以「地址=256·tan(θ)、值=512·θ」把比值映射成定点相位，服务于 phase.v；`gen_rot_lut.py` 把 [0,π/4] 的 (cos θ, sin θ) 打包成 32 位表项，并用 `2^ceil(log2(402))=512` 凑深度，以双口 RAM 同时服务 sync_long 与 equalizer；`gen_deinter_lut.py` 用「32 项目录 + 子表」两级查表，按 `{ht,rate}` 索引，每项 22 位指令（addra/addrb/bita/bitb/erase/out_stb/done），并以 erase 位表达去穿孔空位。核心概念包括定点刻度（左移放大，π=1608）、`.mif`/`.coe` 双格式、脚本与 RTL 共享相位轴（`ROTATE_LUT_LEN_SHIFT=ATAN_LUT_SCALE_SHIFT`）。关键约束：`SCALE=SIZE*2`，改一个常数会沿 common_defs→common_params(PI)→coregen ROM 连锁修改。deinter 表直接复用 decode.deinterleave() 公式，是交叉验证的前提。
