本讲精读 `clk_divider`：其本质是一个**自由运行的二进制计数器**，每个 `clk` 上升沿执行 `out <= out + 1'b1`，配低有效同步复位 `nrst` 与计数使能 `ena`，模块里没有任何专门的分频逻辑；输出位宽由参数 `WIDTH`（默认 32）决定，通过 `#(.WIDTH(...))` 参数化例化。

核心结论：分频是二进制自增的天然副产物——计数器第 N 位 `out[N]` 是主时钟的 1/2^(N+1) 分频（即 f_out[N] = f_clk / 2^(N+1)），且每一位都是 50% 占空比的方波。所有位共享同一个 `clk` 上升沿更新，因此同步、相位一致、无毛刺，等价于一棵一次成型的「时钟树」。

引入的关键术语：自由运行计数器、分频、派生时钟（derived clock）、占空比、时钟使能（clock enable）。

关键局限与用法：只支持 2 的幂次分频；不应把 `out[N]` 当成真正的时钟去驱动 `always_ff`，而应作为慢数据信号（如用高位点 LED 闪烁）或经 `edge_detect` 转成主时钟域里的单拍使能脉冲，以避免凭空新增时钟域。这一「时钟使能优于派生时钟」的范式，为下一讲 `edge_detect` 及后续时钟域跨越（CDC）内容奠基。
