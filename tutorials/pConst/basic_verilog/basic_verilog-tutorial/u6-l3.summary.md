本讲从单信号处理推进到批量数据并行计算与模拟量生成，覆盖四个常用数据处理 IP。加法树 `adder_tree` 用「两两配对、分层相加」的流水线把 N 个输入并行求和，关键路径从 O(N) 降至 O(log N)，输出位宽自动扩 ⌈log₂ N⌉ 位（输入非 2 的幂则用 0 补齐）；把内部 `always_ff` 改成 `always_comb` 即得纯组合版。滑动平均 `moving_average` 用「累加器＋延迟线」实现 O(1) 窗口更新（每拍加新样本、减最老样本，最老样本由 `delay` 提供），窗口为 2 的幂时除法退化为移位、最省资源。PWM 与 PDM 两种调制器均复用 `pulse_gen`：PWM 周期固定、改脉宽（`cntr_low` 随设定值变）；PDM 脉宽固定、改周期，分别用脉冲的「宽窄」和「疏密」表示模拟量。两者都取 `clk_divider` 某一位当作 `pulse_gen` 的派生时钟，属于模块内部生成的时钟域，需 `create_generated_clock` 约束。`moving_average` 可与 PWM/PDM 串联，把含噪序列去噪后映射成随占空比或密度变化的方波，实现数字量到模拟量的低通重建。
