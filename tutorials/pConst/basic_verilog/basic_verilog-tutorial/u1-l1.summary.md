本讲是手册导览篇。basic_verilog（作者 Konstantin Pavlov）是一个"必备可综合 Verilog/SystemVerilog 模块库"，本质是参数化、风格统一的硬件"积木盒"，由大量独立 `.sv`/`.v` 模块及配套 testbench 组成，无传统程序入口。协议为 CC BY-SA 4.0：可商用，但须署名原作者并以相同协议发布衍生作品。顶层目录按职责组织，README 的 DIRECTORY 表是导航地图，含 `example_projects/`（工程模板）、`benchmark_projects/`（跨 IDE 基准）、`scripts/`（编译脚本）等。README 用绿圈标基础、红圈标进阶，建议新手先绿后红；本手册 u1、u2 单元即从绿圈模块（`clk_divider.sv`、`edge_detect.sv`、`gray2bin.sv` 等）切入。需注意文档会滞后于代码：如 README 标注的 `debounce.v` 实际不存在（应为 `debounce_v1.v` 等），`bin2gray`/`gray2bin` 描述疑似写反，应以实际代码为准。
