本讲是「存储器与 FIFO」单元首篇，讲解两个 RAM/ROM 模板：单口 `true_single_port_write_first_ram` 与真双口 `true_dual_port_write_first_2_clock_ram`。核心术语：单口 RAM 同一时刻只能读或写；真双口 RAM 有 A/B 两组独立端口，可同时读写、支持跨时钟域；write-first（写优先）指同址又读又写时输出新值。核心结论：块内存读是同步的，因非阻塞赋值读路径天然延迟一拍；地址位宽写作 `clogb2(RAM_DEPTH-1)`，由 u2-l4 等式等于 `$clog2(RAM_DEPTH)`。`(* ramstyle = RAM_STYLE *)`（无下划线、Quartus/Vivado 通用、默认 `"block"`）引导综合工具把数组推断为块内存而非触发器。初始化用 `generate` 二选一：`INIT_FILE` 非空则 `$readmemh` 装入（仿真与下板均有效），否则全零；将 `wea` 恒接 0 即把单口 RAM 当 ROM 用。真双口模板即单口逻辑复制为 A/B 两套，仓库 FIFO 直接例化它作「一写一读」存储体，为下讲 u4-l2 埋下伏笔。
