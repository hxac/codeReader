本讲承接 UART（u5-l1）的异步通信，转向**同步串行总线 SPI**：由主机驱动 SCLK，四线 SCLK/MOSI/MISO/nCS 分工收发。核心模块 `spi_master` 用**一个 `sequence_cntr` 计数器 + `localparam` 窗口**（WRITE_SEQ/READ_SEQ）替代显式状态机，每位占 2 步（SCLK 上升、下降两个边沿），位宽变化只改算式不动逻辑。关键设计：用 `edge_detect` 把慢速 `spi_clk` 的边沿拆成 `spi_clk_rise`/`spi_clk_fall` 单拍脉冲作为节拍（前提是 `spi_clk` 为系统 `clk` 的整数倍同步分频且 ≥2 周期）；移位硬件**内部恒按 LSB first**，靠入口/出口各一个 `reverse_vector` 组合翻转实现 MSB/LSB first 可配置。模块只支持 mode 0/mode 2（CPHA 固定 0）：MOSI 在下降沿改、MISO 在上升沿采。`ncs_pin` 划定事务边界，`oe_pin` 服务双向缓冲，`spi_busy = (sequence_cntr!=0)` 握手，`FREE_RUNNING_SPI_CLK` 决定空闲时 SCLK 是否持续翻转。读事务实为"**先写后读**"，契合 ADC/Flash 等"发命令→读响应"器件。testbench 用 `always_ff` 搭最小从机移位器形成回环自检，是"仿真里造对手方"的典型手法。注意本讲多处强调以代码实际行为为准（CPOL 注释、`encoder.v` 误列）。
