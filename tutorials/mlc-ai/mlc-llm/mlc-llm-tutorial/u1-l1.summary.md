本讲是 MLC LLM 学习手册的开篇，建立全局心智模型，只读不写代码。核心定位：MLC LLM 是「基于 ML 编译的通用 LLM 部署引擎」，兼具**编译器**（把模型优化成硬件专用代码）与**部署引擎**（驱动推理并对外服务）两重身份，使命是让任意开源 LLM 原生部署到从手机、PC、服务器到浏览器的任意平台，不挑硬件、不绑云。其工作分三阶段：**下载预量化权重 → 编译模型库（用 Apache TVM，仅首次，结果缓存）→ 启动运行时**。跨平台需区分**平台**（OS/运行环境）与**后端**（CUDA、ROCm、Metal、Vulkan、WebGPU/WASM、OpenCL 等计算驱动），切换只需 `--device`，唯一硬约束是显存「装得下、算得动」。对外抽象为 **MLCEngine**——一套跨平台/后端的统一引擎，提供 **OpenAI 兼容 API**，可经 chat CLI、Python（同步 `MLCEngine`/异步 `AsyncMLCEngine`）、REST 服务器三种入口访问，背后是同一套引擎与编译器。验证安装用 `import mlc_llm; print(mlc_llm.__path__)`。后续 u1-l2 将进入仓库目录结构与源码。
