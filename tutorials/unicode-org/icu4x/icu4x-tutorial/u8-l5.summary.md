本讲介绍 ICU4X 实验性组件 `icu::experimental`，它是一个「孵化器 crate」：处于 pre-1.0、每个版本都做不兼容的大版本 bump，能力成熟后「毕业」迁到顶层稳定组件；在元 crate 中由 `unstable` feature 门控启用。该 crate 共 8 个子模块，本讲覆盖最常用的四个：`displaynames`（语言/地区显示名）、`relativetime`（相对时间）、`dimension`（货币/百分比/单位）、`transliterate`（文字转写）。引入的关键术语：孵化器 crate、`unstable` feature、BCP-47-T（用 `-t-` 扩展描述转写「源→目标」文字系统）。重点 API 演进（#8135）：`LanguageIdentifierDisplayNameOwned` 改用专用的 `LanguageIdentifierDisplayNameOptions`（仅含 `language_display` 字段），不再复用通用 `DisplayNamesOptions`。`relativetime` 内部组合 `PluralRules` 与 `DecimalFormatter`；`transliterate` 直接返回 `String`（非惰性 `Writeable`），并依赖 normalizer 与 casemap。
