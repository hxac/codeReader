本讲承接 u4-l1 的 online softmax 主循环，叠加四个横切特性。①GQA 头映射：以 `off_hkv = off_hq // group_size` 让多个 query 头共享同一 KV 头（Q/O 用 query 头、K/V 用 KV 头），零拷贝、零额外显存。②attn_bias：用 stride=0 编码广播维，紧凑掩码（如 `[1,1,1,Nkv]`）永不物化，统一的 4 维寻址式同时覆盖完整与紧凑掩码。③causal：以「query 对齐 KV 尾部」为约定，用 `kv_offset=Nkv-Nq` 做 `end_n` 块级跳过与 `tl.where` 元素级遮蔽，零额外存储。④dropout：复刻 SDPA 的 Philox 逐元素布局（`linear=off_hb*Nq*Nkv+offs_m*Nkv+offs_n`），uint32 位转保证逐位一致以支持反向重放，`l_i/LSE` 保持未 dropout。四特性均由 `tl.constexpr` 编译期消除未启用分支，互不干扰。
