本讲是 RTL-Coder 全局导图：该项目把开源大模型改造成 Verilog 代码生成器，破解 RTL 训练数据稀缺的痛点。两大核心贡献为：自动化数据集生成——GPT 仅用于造数据，产出约 2.7 万条「指令-代码」样本，汇总成 Resyn27k.json（对应 data_generation/ 与 dataset/）；基于代码质量评分的训练方案——在普通微调 MLE 之上引入评分并从算法层降低显存（对应 train/ 下 mle.py、mle_scoring.py、mle_scoring_grad_split.py）。项目发布四个 HuggingFace 模型：Deepseek（精度最高但输出不自动停止，需 endmodulemodule 后处理）、Mistral（默认推荐）、GPTQ-4bit（省 GPU 显存）、GGUF-4bit（可跑 CPU）。评测用 VerilogEval（含 Machine/Human 子集）与 RTLLM 两基准，benchmark_inference/ 脚本只负责生成代码，打分交给基准工具。GPT 仅造数据，最终模型基于开源底座，与 OpenAI 无商业竞争。
