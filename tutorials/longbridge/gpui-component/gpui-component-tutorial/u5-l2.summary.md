本讲讲解 gpui-component 的三种非模态悬浮层（不遮罩、不抢焦点、点外部或移开即消失），与 u5-1 模态 Dialog 形成对照。Popover 是点击/右键触发的弹出层，无状态 RenderOnce，开/关状态外置到由 use_keyed_state 托管的 PopoverState；支持受控（open 须配 on_open_change 成对使用）与非受控（default_open）；定位靠 on_prepaint 量触发器边界、resolved_corner 按锚点角算原点、deferred(anchored()) 延迟绘制；支持 ESC、点外部、DismissEvent 三种关闭并保存/还原焦点。Tooltip 由 Root 内每窗口唯一的 TooltipOverlay 统一托管（同一时刻仅一条），带显示延时、宽限期与滑动动画，经 ManagedTooltipExt 接入，用 epoch 版本号取消陈旧计时器，定位上方优先放不下翻下方。HoverCard 悬停触发富内容卡片，复用 Popover 渲染管线，靠双悬停标志与关闭延时让鼠标能在触发器与正文间移动而不误关。选型口诀：提示用 Tooltip、悬停看详情用 HoverCard、点击展开操作用 Popover。
