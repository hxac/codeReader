本讲义收尾「浮层与导航组件」单元，含五个最小模块。Tab/TabBar 为无状态标签条，选中索引外置到 View；五种 `TabVariant`（Tab/Outline/Pill/Segmented/Underline）配合四态样式表决定外观，Pill/Segmented/Underline 用 `on_prepaint` 测矩形加 `with_animation` 插值实现滑动指示器，`epoch` 保证动画与文字淡入同步。Sidebar 是泛型折叠容器，子项须实现 `SidebarItem`（`Collapsible+Clone`）；三种 `SidebarCollapsible`（Icon/Offcanvas/None）控制折叠形态，宽度动画用 `use_keyed_state` 跨帧保持并以版本号防竞态，内容虚拟化于 `list`。Breadcrumb 最简：无状态、末项高亮、相邻项自动插 `ChevronRight` 分隔符。Select/Combobox 共享 `SearchableListState` 与 `SearchableListDelegate` 引擎，状态为独立 `Entity`、外壳为无状态 `RenderOnce`，下拉层是 `deferred(anchored())` 内的虚拟化 `List`，经 `register_deferred_popover` 与 Root 协作；Select 单选、Combobox 支持多选/自定义触发器与 footer，以 `selection_changes` 区分单选替换与多选切换，`on_will_change` 提供选择否决点。核心范式为「状态外置」与「状态实体＋无状态外壳」，并为 u6 的 Dock、u7 的虚拟化 List 承接埋下伏笔。
