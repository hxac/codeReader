本讲为手册首篇，仅读项目说明与目录结构，覆盖三模块。定位：MuPDF 是 Artifex Software 的轻量级开源文档处理框架，以可移植 C 写成，提供查看器、命令行工具（mutool）与软件库三形态。核心是双层架构——fitz 通用层（source/fitz）定义 document/page/device 等格式无关抽象，格式专用层（source/pdf、source/xps、source/html 等）各自把本格式翻译成 fitz 抽象，使一份应用代码可处理十余种格式。格式清单的权威来源是 source/fitz/document-all.c 的 handler 注册表：每种格式对应一个 fz_document_handler，经 fz_register_document_handlers 登记后由 fz_open_document 按扩展名或内容匹配调用，共 14 个 handler，其中 EPUB/MOBI/FB2/Office/Markdown 等电子书格式共享 source/html 的 HTML 排版引擎；include/mupdf/fitz/config.h 用 FZ_ENABLE_* 宏做编译期裁剪，gz handler 因处理外层压缩而始终注册。授权为双许可：GNU AGPL v3 强 copyleft，要求衍生作品（含网络服务）整体开源、无担保；商业授权付费免除上述限制，个人本地学习可免费适用 AGPL。
