本讲承接 u2-l2，完成 generate_xview3.py 主循环的后半段，覆盖三个最小模块。一是把落入芯片的标注转写为 YOLO 归一化标签（每行 class x_center y_center width height）：场景全局像素坐标先局部化（减去芯片起点 row_start/col_start）再除以 imgsz 归一化；因 SAR 船舶为点目标，框宽高取极小常数 WIDTH/HEIGHT_MEDIAN=0.01。二是类别编码，用 label = is_vessel + is_fishing 得到连续的 0/1/2 三类（非船、船、渔船），nan_true 宽松分支把渔船属性缺失的船补成渔船（训练默认开启）。三是背景负样本采样：空芯片先暂存，场景遍历完后按 int(正样本数×30%) 随机抽样，仅写图像不写 .txt，YOLO 即视为纯负样本，用于让模型见过开阔海面、抑制误检。get_annots() 中变量名 x/y 与行列含义互换，靠 np.stack([…, y, x, …]) 顺序抵消，最终输出仍是标准 YOLO 格式。
