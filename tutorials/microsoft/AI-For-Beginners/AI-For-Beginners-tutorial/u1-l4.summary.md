本讲通过两个纯 Python 脚本（无需 PyTorch/TensorFlow）演示 AI 的核心思想：从数据中学习规律（权重），而非手写规则。围绕 `examples/01-hello-ai-world.py` 中的 `SimpleAILearner` 类，讲解三个互相咬合的概念：训练数据与模式、权重与学习率、训练循环与误差。模型形式为预测 ŷ = w·x，靠更新公式 w ← w + η·error·x 学习，其中权重（weight）是待学参数、误差（error）是调整依据、学习率（learning rate）是步长旋钮；训练采用外层 epoch 循环与内层逐样本更新，并用累加误差监控收敛。关键术语包括模式（pattern）、监督学习（supervised learning）、梯度下降（gradient descent）的简化形式、发散（divergence）。进阶脚本 `02-simple-neural-network.py` 把单权重升级为人工神经元：加权求和 + 偏置 + sigmoid 激活，引入前向传播（forward propagation）与反向传播（backpropagation），完成从回归到二分类的跨越。核心结论：训练数据决定学什么、学习率决定学多快、误差驱动收敛是几乎所有训练流程的骨架；本讲为后续感知机与神经网络课程铺垫了连接主义直觉。综合实践要求修改代码使模型学会 y = 3x。
