本讲是学习手册开篇，旨在建立对 AI-For-Beginners 仓库的全局认知。该仓库属「课程仓库」而非传统软件项目：核心产物是 Markdown 讲义与可执行 Jupyter Notebook，定位为微软出品的「12 周、24 课、对初学者友好」的 AI 课程。

三个最小模块为课程定位与目标读者、12 周 24 课内容表、技术栈与覆盖范围。内容表以「罗马数字大单元 + 阿拉伯数字课程」两级编号，将课程归为七大板块：AI 导论、符号 AI、神经网络、计算机视觉、NLP、其他 AI 技术、AI 伦理，另含多模态附加内容；其中计算机视觉与 NLP 篇幅最大，目录名编号即学习顺序。

技术栈以 Python 3 与 Jupyter 为载体，深度学习采用 TensorFlow + PyTorch/Keras「双框架并行」，理解一版即可；图像处理用 OpenCV，测验应用用 Vue.js。课程聚焦神经网络与深度学习，明确不涵盖经典机器学习（指向姊妹课程 ML-for-Beginners）、云框架、聊天机器人及深层数学。README、AGENTS、etc/Mindmap 三文件共同构成课程总入口。
