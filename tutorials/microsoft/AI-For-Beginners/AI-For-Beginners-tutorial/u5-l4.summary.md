本讲讲解多模态网络 CLIP 与 VQGAN，是 CV 与 NLP 两条线的汇合点，含四个最小模块：图文对齐、对比学习、零样本检索、VQGAN+CLIP 生成。图文对齐：CLIP 用图像编码器 ViT 与文本编码器 Transformer 把图与句压成同维向量、落入同一高维空间，以余弦相似度度量图文匹配。对比学习：训练时在 batch 内最大化正对相似度、最小化负对相似度（InfoNCE 损失），负样本由 batch 内其他对天然充当，batch 越大学得越好。零样本：因已预训练，新任务无需训练；分类（1 图选 N 类）用 logits_per_image，检索（1 句选 N 图）用 logits_per_text，均为相似度矩阵上 softmax 取 argmax，softmax 方向须跟随被选集合。VQGAN+CLIP：把 CLIP 当作图文相似度损失，对随机隐向量 z 做梯度下降、经 VQGAN 生成图，与神经风格迁移「冻结网络、优化输入」同构（VGG 换成 CLIP，像素换成 z）。关键术语：CLIP、对比学习/InfoNCE、余弦相似度、ViT、零样本分类、文本图像检索、VQGAN。
