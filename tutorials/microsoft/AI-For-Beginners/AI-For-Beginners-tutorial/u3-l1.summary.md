本讲是计算机视觉（CV）单元开篇，定位为「喂给神经网络之前」的图像预处理。核心立论：图像在 Python 里就是 NumPy 数组——灰度图为 (H,W)、彩色图为 (H,W,3)，图像处理本质是数组运算；能用传统 CV 替神经网络分担的工作就一定用，以减少对训练数据的依赖。

引入的关键术语：计算机视觉及其任务（图像分类、目标检测、事件检测、3D 重建）；OpenCV（cv2 接口，图像处理事实标准，默认 BGR 顺序，喂 matplotlib 前需转 RGB）；颜色空间（灰度、HSV、BGR/RGB）；经典滤波算子——均值、高斯、中值用于去噪，自适应阈值与 Otsu 用于二值化；几何变换（缩放、仿射、透视）；帧差法（相邻帧相减求范数再加阈值，廉价检测视频运动，但只知「动了多少」）；光流（optical flow，分稠密与稀疏，给出每像素 (dx,dy) 向量以判断运动方向）。

核心结论：盲文读图→灰度→二值化→切字符、帧差运动检测，是「纯 CV 减负」的两个范例；本讲为下一讲 CNN 提供「图像即数组」的直觉与数据预处理基础。
