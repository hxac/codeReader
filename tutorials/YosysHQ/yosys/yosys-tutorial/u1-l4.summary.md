本讲完成 Yosys 第一次完整综合，分三块。①shell 与命令调度：交互 shell 与 .ys 脚本共用同一套命令语言，都经 Pass::call 拆词（处理 # ; !）、用命令名查全局表 pass_register 调用对应 pass 的 execute；命令出错只被 shell 的 try/catch 捕获而不崩溃，main() 用 run_shell 决定结尾进交互还是写后端。②cmos 计数器示例：counter.ys 演示完整流水线——read_verilog 读设计产出含 process 的 RTLIL，read_verilog -lib 把库单元注册为黑盒，synth 宏命令把行为级 process 翻译成通用 $ 单元，dfflibmap/abc -liberty 读 liberty 做工艺映射换成具名库单元，最后 stat 统计、write_verilog/write_spice 多后端输出同一份 RTLIL。③help：help 也是 pass，文本来自各 pass 的 help() 方法。核心结论：-lib 注册端口、-liberty 提供逻辑功能与面积，二者配合才完成工艺映射。
