本讲是 AES 硬件核学习手册的起点，覆盖三个最小模块：README 项目说明、AES 算法背景、ECB 工作模式。核心结论：本仓库用 Verilog 实现面向 FPGA/ASIC 的低功耗 AES 加解密核，采用密钥扩展架构，宣称吞吐 0.06 Gbps。AES 是对称分组密码，数据块固定 128 位；密钥支持 128 位（10 轮）与 256 位（14 轮），跳过 192，由 CTRL_KEYLEN_BIT 运行时切换；每轮含 SubBytes、ShiftRows、MixColumns、AddRoundKey 四变换，最终轮省去 MixColumns。本核采用 ECB 单块模式：一个 128 位块进、一个块出，无初始向量、块间独立，相同明文得相同密文。顶层接口为写 BLOCK/KEY、触发 init/next、读 RESULT 的寄存器访问，并配 ready/valid 状态位。贯穿全手册的原则是"以源码为准"——README 仅写 256 位密钥，源码实际兼容 128/256 两种。引入的关键术语（密钥扩展、轮密钥、S-box、有限域 GF(2^8)）供后续讲义承接。
