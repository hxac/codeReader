本讲精读扩展架构中负责权重缓存与时间歪斜的 `dff8` 与 `weightFifo` 两个模块。`dff8` 是 8 位参数化、带同步复位与使能的 D 触发器，按复位清零、使能采新值、否则保持三级优先级工作；自回环 `q<=q` 综合成带使能 FF 而非锁存器，端口标 `signed` 体现权重补码语义。`weightFifo` 用数组化例化加两个 `generate` 把 `FIFO_INPUTS×FIFO_DEPTH` 个 `dff8` 排成网格：`assignConn` 串联级间数据形成移位链，`colEn` 使能按列共享，构成多列并行纵向移位寄存器。核心结论：`weightOut` 是 `weightIn` 延迟 `FIFO_DEPTH` 拍的副本，模块内不翻转位序；按列使能 `en[i]` 同时驱动第 i 列全部触发器，让控制器逐列错拍推进，制造脉动阵列所需的列间时间歪斜（skew），这是核心 rtl 中 `addr_sel` 地址偏移歪斜在权重侧的对应物，体现权重固定（weight-stationary）思路。`top.v` 中 `FIFO_INPUTS=FIFO_DEPTH=WIDTH_HEIGHT=16`，`en=mem_to_fifo_en|fifo_to_arr_en` 分别承载 `tpu_fill_fifo` 装载与 `tpu_mat_mult` 排出两阶段。
