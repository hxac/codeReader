本讲是 u5 单元收口，视角由 ASIC 流片转向 FPGA SoC 集成。最小模块为板级顶层 `tpu.v` 与 `tpu_system.v`，二者几乎不含逻辑，只做接线：把 `CLOCK_50`、外设（`HEX/KEY/SW/LEDR`）与整组 `HPS_DDR3` PHY 信号对接到 Qsys（Platform Designer）自动生成的系统模块（`tpus_system`/`mysystem`）。核心结论：顶层仅是接线员，TPU 计算核不在此层，而经 `busConn` 以 Avalon-MM slave 身份挂进 SoC，被 HPS（ARM Cortex-A9 硬核）以内存映射方式访问；`reset_reset_n` 常置 1，复位交由 HPS 上电序列管理。七段码因共阳极低电平点亮，需 `~` 把高有效段码翻成物理电平。关键术语：FPGA SoC、HPS、Qsys、Avalon-MM、双向 `inout` 的 `DQ`/`DQS`、`RZQ` 阻抗校准。相比 ASIC 后端，FPGA SoC 自带 CPU、硬核 DDR3 控制器与自动总线互连，集成迭代远快于定制芯片。
