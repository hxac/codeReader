本讲精读 TPU 主控状态机 systolic_controll，它是「单输入、多命令输出」的节拍器，采用「一时序块+三组合块」的四块结构：时序块只更新寄存器，三个 always@(*) 分别管状态转移、addr_serial_num 与系统控制信号（alu_start、sram_write_enable 为纯组合输出）。

核心为四状态机 IDLE→LOAD_DATA→WAIT1→ROLLING，仅两条条件边：tpu_start 启动、matrix_index==15 且 data_set==1 时结束并拉高单拍脉冲 tpu_done；中间两态专为地址流水提前两拍铺路。关键概念：addr_serial_num（交 addr_sel 解码的地址串行号，在 ROLLING 中恒领先 cycle_num 2 拍以补偿 SRAM 读延迟），以及 ROLLING 内三个嵌套计数器 cycle_num（节拍）、matrix_index（输出批索引，一生产者两消费者）、data_set（输出 SRAM 组）。核心结论：以 cycle_num≥ARRAY_SIZE+1=9 作为写回门控，与阵列首个有效输出严格对齐；灌满 9 拍不写、写回 32 拍，共 41 拍 ROLLING、产出 256 个元素。本讲为 u3 单元入口，下接 addr_sel 与 write_out。
