本讲聚焦 `systolic.v` 的结果收集（gather）纯组合逻辑：如何把 8×8 累加矩阵 `matrix_mul_2D` 的 cell 挑出并拼成输出。`mul_outcome` 是一条 168bit（8 段×21bit）打包总线，段号即行号 i；gather 块开头先逐位清零以防推断锁存器。引入两个关键概念：「反对角线」（满足 `i+j==s` 的 cell 集合）与「互补反对角线对」（`s` 与 `s+8`，cell 数之和恒为 `ARRAY_SIZE=8`）。`matrix_index` 经 if/else 折算成 `upper_bound`/`lower_bound`，且以 8 为周期复用边界（如 0 与 8、7 与 15 完全相同）。两段 for 循环分别扫左上三角（`j<=7-i`，挑 `upper_bound`）与右下三角（`j>=8-i`，挑 `lower_bound`），命中行号不交故各段恰好写一次，使每个 `matrix_index` 恒输出 8 个结果、带宽 100% 利用，走完 0..7 即覆盖全部 64 个输出元素各一次。该总线原样（顶层即 `ori_data`）送入 `quantize` 按相同切法逐段量化；`matrix_index` 的推进时序（在 `cycle_num>=FIRST_OUT(=9)` 后开始递增并拉高 `sram_write_enable`）由控制器 u3-l1 负责。
