本讲精读 systolic.v 的乘加（MAC）逻辑，要点有三。一是双份写法：时序寄存器 matrix_mul_2D 与组合下一拍 matrix_mul_2D_nx 分工，时序块只做 _nx→_2D 提交，关键路径仅一次乘加，利于时序收敛；alu_start=0 时组合块令 _nx 等于当前值，从而冻结累加器。二是组合块由 cycle_num 门控做三选一（优先级固定）：分支 1 首入（乘积符号扩展后覆盖累加器，标志新输出元素起点，优先级最高）、分支 2 累加（乘积加到旧值，即元素的第 2~8 个乘积）、分支 3 保持；三条分支乘法同为 weight_queue[i][j]*data_queue[i][j]，差别仅在覆盖或相加。三是时序由 cell 反对角线编号 s=i+j 决定：第 s+1~s+8 拍累加满 8 个乘积，第 s+9 拍首入下一项，同一反对角线上的 cell 同步推进，整阵呈左上→右下的波前。FIRST_OUT=ARRAY_SIZE+1=9 是首个输出就绪节拍（控制器亦在 cycle_num≥9 才写回），PARALLEL_START=2×ARRAY_SIZE+1=17 配合 %16 表达「每 8 拍一次首入」的稳态流水。16bit 有符号乘积经 {5{mul_result[15]}} 符号扩展成 21bit 后入累加器，防止负乘积因高位补 0 而翻转。下一讲 u2-l3 承接 matrix_mul_2D 与 matrix_index 做结果收集与 mul_outcome 输出。
