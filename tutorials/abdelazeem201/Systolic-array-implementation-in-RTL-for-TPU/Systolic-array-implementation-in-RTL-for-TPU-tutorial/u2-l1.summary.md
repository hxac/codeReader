本讲剖析 `systolic.v` 数据通路最底层的两个 `8×8`、元素为 `signed [7:0]` 的二维寄存器阵列：`weight_queue` 与 `data_queue`，合计 1024bit，分别充当每个 cell 的本地 weight/data 寄存器。数据只从阵列边界进入：weight 由 w0/w1 两路 32bit SRAM 切成 8 个字节灌进第 0 行，data 由 d0/d1 灌进第 0 列，切片用变基部分选择 `[31-8*i -: 8]`，最高字节进下标 0。随后移位 for 循环在同一 `always` 块内、靠非阻塞赋值 `<=`「取旧值」的规则，让 weight 每拍整体下移一格、data 整体右移一格，通信仅在相邻 cell 间发生；填入与移位互补地处理边界（weight 跳第 0 行、data 跳第 0 列）。核心结论：移位天然制造时间错位（skew）——到达 cell(i,j) 时 weight 滞后注入 `i` 拍、data 滞后 `j` 拍，对角线 `i==j` 上两者同时到达，偏离对角线则需由地址歪斜补偿到达时间差，这正是脉动阵列算对矩阵乘的命门，为下一讲 MAC 乘加的 `cycle_num` 调度奠定物理基础。
