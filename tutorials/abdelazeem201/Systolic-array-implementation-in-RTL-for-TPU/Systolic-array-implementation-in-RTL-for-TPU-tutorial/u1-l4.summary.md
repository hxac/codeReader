本讲解析 TPU 的参数化设计与定点数位宽推导。核心三块：其一，`tpu_top` 用四个对外旋钮 `parameter`——`ARRAY_SIZE`、`SRAM_DATA_WIDTH`、`DATA_WIDTH`、`OUTPUT_DATA_WIDTH`——以 `.名(名)` 显式层层传给子模块；`localparam` 则只能内部派生、不可外部覆盖。其二，派生位宽 `ORI_WIDTH = DATA_WIDTH+DATA_WIDTH+5 = 21`（与 `systolic` 的 `OUTCOME_WIDTH` 等价）：其中 8+8=16 为乘积位宽，+5 为累加保护位（理论 3 位即够，多 2 位冗余），且它只依赖 `DATA_WIDTH`、与 `ARRAY_SIZE` 无关。其三，引入定点记法 Qm.n，数值旅程为 8 位 Q4.4 输入 → 16 位 Q8.8 乘积 → 21 位 Q13.8 累加 → 16 位 Q8.8 饱和输出，关键机制是有符号乘法位数相加、符号扩展 `{ {5{mul_result[15]}} , mul_result }` 与量化饱和到 ±32767/−32768。核心结论：本项目是「半参数化」——位宽骨架随参数走，但 `i<4`、`%16`、地址位宽等仍按 8×8 写死，直接改 `ARRAY_SIZE` 无法即插即用。
