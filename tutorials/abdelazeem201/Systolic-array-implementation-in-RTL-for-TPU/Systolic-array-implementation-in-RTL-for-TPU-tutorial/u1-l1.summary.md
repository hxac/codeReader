本项目用 Verilog 实现一个 **8×8（参数 ARRAY_SIZE 可放大到 32×32）定点矩阵乘加速器**，矩阵 A 为 weight、矩阵 B 为 data。TPU 是为神经网络大规模矩阵乘设计的专用加速器，核心计算结构是**脉动阵列（systolic array）**：数据在由相同处理单元 cell 组成的网格里按时钟节拍流动，每经一个 cell 做一次乘加（MAC），靠数据复用与本地化通信获得高吞吐。本项目的「宪法」级数据流是**weight 自上而下、data 自左而右**，每个 cell 做 MAC 并把结果就地累加，属于**输出固定（output-stationary）**式。每个 cell 含**三类寄存器**：ALU 累加、weight 寄存器、data 寄存器，共 64 个 cell。定点格式：输入 8 位（4 整 4 小）、单次乘积 16 位、累加中间结果 21 位（含 5 位保护位）、输出 16 位（8 整 8 小）。项目分三块：**核心 RTL**（算得对，含 systolic、systolic_controll、quantize、addr_sel、tpu_top 五模块）、**后端流程 syn/pnr**（造得出）、**扩展架构 RTL_modified**（用得起）。注意：子目录文档称 32×32，但实际代码默认与综合均按 8×8。
