本讲讲解 SDC（单路延迟换向器）架构中的延时单元 `shift_N`（N=16/8/4/2/1）。它接在 `radix2` 蝶形旁，把 first half 算出的「差」延迟 N 拍后回流为 `din_a`，供 second half 运算，在顶层构成 `delay → shift → din_a` 反馈回路。

实现采用「超宽寄存器移位」而非 RAM+读写指针：用一条 N×24bit 寄存器，每拍执行 `(tmp_reg<<24)+din_r` 左移拼入新样本，`dout` 用固定窗口取最高 24 位读出最老样本，等价于无指针 FIFO，延时恰为 N 拍（N=1 时退化为单拍寄存器）。握手方面，`in_valid` 点火后内部 `valid` 借 `next_valid=valid` 自保持，使残留样本在数据流过后继续冲刷；模块内 `counter_N` 虽每拍递增却不驱动任何输出，属需识别的冗余信号。

核心结论：延时序列 16/8/4/2/1 逐级减半，等于 radix-2 DIF 各级蝶形配对距离（组大小/2），关系式为 N=寄存器位宽/24。下一讲 u3-l4 将转向 ROM 与状态控制，由其 `state` 信号驱动蝶形三态机在 first half 与 second half 间切换。
