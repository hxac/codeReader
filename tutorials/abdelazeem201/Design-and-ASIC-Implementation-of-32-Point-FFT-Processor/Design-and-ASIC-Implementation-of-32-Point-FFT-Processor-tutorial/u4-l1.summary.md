本讲把零件装回整台机器，串讲 32 点 FFT 的五级 SDC（单路延迟换向器）流水线数据流。最小模块即每一级的「蝶形 radix2 + 延时线 shift_N + 旋转因子 ROM_N」三件套，三者构成级内闭合的反馈回路 `delay → shift_N → din_a`，延时深度 16/8/4/2/1 逐级减半并等于各级蝶形配对距离；五个 radix2 沿主路径 `din→op1→…→out` 串联。核心结论有四：其一，时序由 **valid 菊花链**驱动——外部 `in_valid`（打拍成 `in_valid_reg`）启动第 1 级，之后每级 shift/ROM 的使能都接上一级 `radix_noX_outvalid`，radix2 无时钟、由 state 决定何时吐 outvalid，outvalid 既是「算完」标志又是下一级发车信号；其二，第 5 级无 ROM，旋转因子写死 `w_r=256,w_i=0`（定点 1+j0），second half 退化为恒等 `op=din_a`，state 改由顶层 `no5_state`（`r4_valid`+`s5_count`）生成，outvalid 空接；其三，稳态吞吐 1 样本/拍（100% 利用率）；其四，填充时延主源于延时线深度之和 31，testbench 用 68 拍看门狗兜底。引入的关键术语 SDC 流水线、反馈回路、valid 菊花链、no5_state、r4_valid、填充时延供后续讲义承接。
