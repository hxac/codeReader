本讲把 32 点 FFT 提升为「可改造、可扩展」的参数化设计。核心结论：点数 N 决定一切——流水线级数 S=log₂N，第 k 级延时深度 d_k=N/2ᵏ，总填充时延 N−1，位反转位宽 log₂N。引入两组可直接套用的公式：shift_N 的寄存器位宽为 24N、dout 取最高 24 位窗口 [24N−1:24(N−1)]，等价深度为 N 的无指针 FIFO；ROM_N 的 state 三段阈值为 [0,N)、[N,2N)、[2N,3N)。旋转因子按级递减，第 k 级存 W_{N/2ᵏ⁻¹} 共 N/2ᵏ 个，点数翻倍时改 twiddle_gen.py 的 range 与角距、保持 24 位补码定点口径即可重生成。关键术语：级（stage）、延时深度（delay depth）、常数化末级（末级旋转因子恒为 1+j0、无 ROM）、valid 菊花链、位反转排序表。最重要结论：这套 SDC（单路延迟换向器）架构高度模块化、可堆叠，32→64 仅需新增 shift_32 与 ROM_32 两个文件，radix2 蝶形单元点数无关、完全复用，主要工作量在顶层接线和 case(y_1) 排序表扩位。二次开发四个方向：扩缩点数（成本最低）、加宽位宽（成本最高）、插流水线寄存器提频、折叠/展开调面积-吞吐权衡。
