本讲讲解 nginx 自备的六种容器，均建于内存池 ngx_pool_t 之上，节点随池整体回收、普遍无单独 free。关键概念「侵入式容器」将链接字段内嵌于用户结构，借 offsetof 反推宿主地址，实现零分配与泛型。ngx_array_t 动态数组满时优先池顶原地生长、否则两倍扩容，均摊 O(1)、下标 O(1)；ngx_list_t 分块链表每块定长、满则挂新块不拷旧数据，适合仅需顺序遍历的集合如 HTTP 头；ngx_queue_t 侵入式双向循环链表全宏实现、O(1) 增删，靠 ngx_queue_data 反取宿主并自带稳定归并排序；ngx_rbtree_t 红黑树 O(log n) 增删查、用哨兵省 NULL 判断，定时器版本以有符号差值比较处理 49 天毫秒回绕，是 nginx 定时器底座；ngx_hash_t 静态哈希表配置阶段一次建表、运行时只读查，建表时搜索最小合适桶数并紧凑排布桶链，通配符查找借值指针低 2 位编码语义；ngx_radix_tree_t 基数树按键二进制位分叉支持最长前缀匹配，用于 geo 模块 CIDR/IP 映射。
