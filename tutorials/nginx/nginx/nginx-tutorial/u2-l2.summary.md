本讲进入 nginx 自研标准库的「文字层」，含四个最小模块。一是长度前缀字符串 `ngx_str_t`：仅含 `len` 与 `data`，长度显式存储、不靠 `\0`、求长 O(1)；初始化宏 `ngx_str_set`/`ngx_str_null` 是无 `do{}while(0)` 的双语句，置于 `if` 须加花括号防悬空 else。二是整数解析族 `ngx_atoi` 等：长度限定、严格数字，以 `cutoff=MAX/10`、`cutlim=MAX%10` 在乘加前判溢出，变体（`ngx_atosz`/`ngx_atoof`/`ngx_atofp`）仅换返回类型与上界。三是带单位解析 `ngx_parse_size`(K/M)、`ngx_parse_offset`(K/M/G)、`ngx_parse_time`：最后者为顺序约束状态机，单位须从大到小，`is_sec` 决定返回秒或毫秒，毫秒模式禁 `y`/`M`、秒模式禁 `ms`。四是 `ngx_parse_url`：按 `unix:`/`[`/其它前缀三分发到 inet 子解析器，结果填入 `ngx_url_t`，需内存池。配置 slot 函数是文本到解析器的桥梁；`ngx_parse_http_time` 解析 HTTP 日期，勿与时长解析混淆。
