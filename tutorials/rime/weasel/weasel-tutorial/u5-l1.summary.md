本讲聚焦候选窗口「外壳」`WeaselPanel`，拆为三个最小模块。其一为分层窗口与双缓冲：它是以 `WS_POPUP` 配 `WS_EX_TOPMOST|WS_EX_NOACTIVATE` 的分层 popup，不抢焦点、置顶；创建时带 `WS_EX_TRANSPARENT`，`DoPaint` 内用 `ModifyStyleEx` 翻成 `WS_EX_LAYERED` 以兼顾鼠标响应与每像素 alpha 合成。绘制走手动链路 `Refresh→RedrawWindow→DoPaint→memDC→UpdateLayeredWindow`，绕开被宿主吞掉的 `WM_PAINT`。其二为鼠标交互：hover/点击/翻页三类事件统一抽象为四指针的 `_UICallback`（sel/hov/next/scroll_next），由 TSF 层 `HandleUICallback` 派发为 IPC 命令，使 UI 与命令派发解耦。引入术语 `HoverType` 三模式（`NONE`/`SEMI_HILITE` 本地 `m_hoverIndex` 不走 IPC / `HILITE` 发 IPC）及「两段式点击」——按下移高亮并起按压反馈 `bar_scale_`，抬起在当前高亮项上才上屏。其三为光标跟随与 DPI：`MoveTo`/`_RepositionWindow` 定位，超出工作区底部时翻到光标上方并置 `m_sticky=true` 防抖；`DPI_SCALE` 宏与 `dpiScaleLayout=dpi/96` 统一缩放，跨屏由 `WM_DPICHANGED` 与显示器变化触发重绘。布局几何与文字绘制留给 u5-l2、u5-l3。
