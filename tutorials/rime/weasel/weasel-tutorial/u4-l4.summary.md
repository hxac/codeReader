本讲聚焦 `RimeWithWeaselHandler` 的三个收尾模块。`_UpdateUI` 是输出侧唯一的 UI 推送漏斗：采集 `Status`/`Context`、设置 `inline_preedit` 能力位，并按 `_ShowMessage` 返回值在「正常候选（`Hide`+`Update`）」与「瞬时提示」间分叉，`UI::Update` 靠内容相等比较去重，空 context 触发隐藏。引擎通知经 `OnNotify` 写静态缓冲 → `_ShowMessage` 锁内裁决 → `ShowWithTimeout` 启动 `AUTOHIDE_TIMER` 自动隐藏；由 `m_show_notifications` 白名单与 `m_show_notifications_time`（默认 1200，0 为常显）双重控制，`deploy` 必显、`ascii_mode` 只刷图标，内置「不打断」去重保证不重复弹窗。维护模式以 `StartMaintenance`/`EndMaintenance` 配合 `Finalize` 与 `m_disabled` 让引擎重建期按键直接放行，`AddSession` 自愈恢复；深色主题由 `OnColorChange` 经 `m_darkMode` 去抖后触发 `UpdateColorTheme`，为每个会话重算样式并置 `__synced=false`，由下次 `_Respond` 重发配色。
