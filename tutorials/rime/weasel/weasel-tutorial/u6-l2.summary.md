本讲讲解 Weasel 安装/卸载器 WeaselSetup.exe，由 Run 派发命令行（/s、/t、/i 安装，/u 卸载，外加若干只写 HKCU 的轻量配置命令），重活以 runas 自我提权。讲义覆盖四块：交互式对话框（选简繁与用户目录，"修改"模式不重装而走重启服务加部署）、TSF 注册、命令行与权限、多架构文件落点。关键术语：TIP（TSF 文本服务，本质 COM 进程内服务器）、LANGID（简 0x0804、繁 0x0404）、唯一 CLSID 与 Profile GUID 在安装器与 DLL 两处重复定义。核心结论是跨进程分工：WeaselSetup 用 regsvr32 加载 weasel.dll 触发其 DllRegisterServer，简繁经 TEXTSERVICE_PROFILE 环境变量传递，前后用 ITfInputProcessorProfiles 做幂等 Profile 启停，并以 input.dll 的 InstallLayoutOrTip 登记键盘布局。文件落点处理 WOW64 重定向与 ARM64/ARM64X，以 .old.N 加重启延迟删除做占用兜底；卸载须先停 WeaselServer 释放 weasel.dll 句柄；安装末尾写 WER 崩溃转储配置。
