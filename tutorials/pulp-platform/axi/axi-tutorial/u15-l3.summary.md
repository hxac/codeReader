本讲在标准 `axi_xbar`（u6-l1）之上介绍两个高级互连变体。`axi_xp`（Crosspoint，编译层级 Level 6，居全库顶层）追求**同构（homomorphous）端口**：slave 与 master 两侧 ID 等宽（共用 `AxiIdWidth`），实现为「1 个 `axi_xbar` + 每 master 端口 1 个 `axi_id_remap`」，由 id_remap 把内部被加宽的 ID 压回原宽，从而可背靠背级联成 mesh/fat-tree 等规则网络；代价是多出 id_remap 面积与 `AxiSlvPortMaxUniqIds` 并发上限，且不对外引出 xbar 的「默认 master 端口」。`axi_interleaved_xbar`（Level 4）则新增运行期可切换的**地址交错路由**：`interleaved_mode_ena_i=1` 时取地址第 16 位起的 bank 位直接选 master 端口，并把这些位从下游地址中抹掉，实现存储 bank 交错；它不复用 `axi_xbar_unmuxed`，而自拼 `addr_decode+axi_demux+axi_mux+axi_err_slv`，cross 为纯组合、不可流水。两模块均无 testbench、被源码与 README 标为「实验性」，生产使用需自补验证。选型口诀：通用优先 `axi_xbar`；要等宽可级联选 `axi_xp`；要 bank 交错才考虑 `axi_interleaved_xbar`。
