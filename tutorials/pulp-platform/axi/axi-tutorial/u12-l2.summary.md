本讲讲解 AXI-Lite 多路复用族三件套：axi_lite_mux（N→1 汇聚）、axi_lite_demux（1→N 拆分）、axi_lite_xbar（全连接交叉开关）。核心结论：Lite 无 id 字段，但强制保序、单拍事务，故用深度为 MaxTrans 的 select FIFO 替代完整 AXI4 的 axi_id_prepend 与 axi_demux_id_counters——请求方向记录路由决策 select 并按序入队，响应方向按队头回放至原端口。mux 用 rr_arb_tree 轮询选源，写通路 w_fifo→b_fifo 两级、读通路 r_fifo 一级，并用 lock_aw_valid 切断 AW→W 组合依赖防死锁；demux 遵循「译码在外、路由在内」，select 由外部喂入。xbar 每 slave 端口配一个 demux、每 master 端口配一个 mux，cross 矩阵转置相连；译码错误经 axi_lite_to_axi 升格后交 axi_err_slv 回 RESP_DECERR（数据 0xBADCAB1E）。配置复用 xbar_cfg_t，LatencyMode 高 5 位给 demux、低 5 位给 mux。
