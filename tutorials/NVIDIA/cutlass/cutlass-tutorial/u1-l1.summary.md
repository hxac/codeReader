CUTLASS 是 NVIDIA 开源的 **header-only CUDA C++ 模板库**，用于实现高性能 GEMM（通用矩阵乘法）及相关计算，定位为「比闭源 cuBLAS 更灵活、可读、可定制的菜谱」。其核心思想是 **GEMM 层次化分解**：把一次大矩阵乘法拆成 device → threadblock → warp → thread/指令 四层，分别映射到 `gemm/device`、`gemm/threadblock`、`gemm/warp`、`thread` 与 `arch` 等源码目录，借助 Tensor Core 逼近峰值算力。仓库三代 API 并存：2.x（`device::Gemm` 经典四层）、3.x（引入 CuTe 的 `Layout`/`Tensor`/`Atom` 抽象，重组为 kernel + collective MMA + epilogue 三段式，配 `CollectiveBuilder` 自动组装）、4.x（Python 原生 CuTe DSL）。库总入口 `include/cutlass/cutlass.h` 定义了贯穿全库的 `cutlass::Status` 状态码与 warp 常量。支持 FP64~FP4、整数、二值等多种数据类型与 Volta→Blackwell 架构（compute capability 7.0~12.1）；通过 `CUTLASS_NVCC_ARCHS` 指定目标，Hopper/Blackwell 的架构加速特性需带 `a` 后缀（如 `90a`）。
