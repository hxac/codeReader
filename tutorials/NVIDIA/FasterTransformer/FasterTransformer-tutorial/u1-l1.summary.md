本讲是全册第一篇，建立对 FasterTransformer（FT）的全局认知。FT 是 NVIDIA 用 CUDA/cuBLAS/cuBLASLt 与 C++ 高度优化的 Transformer 推理加速库，只做推理不做训练，靠融合 kernel、去 padding、低精度矩阵乘与多 GPU 并行换取数倍加速。核心工具是「支持矩阵」，按模型 × 框架 × 精度 × 并行正交判断能力：FP16 + Tensor parallel + Pipeline parallel 是大模型标配，INT8/FP8/Sparsity 仅覆盖个别模型且有 GPU 架构门槛（INT8 需 Turing、Sparsity 需 Ampere、FP8 需 Hopper）。所有模型在 C++ 下都可用，TF/PyTorch/Triton/TensorRT 只是封装外壳。项目自 4.0 起支持多 GPU/多节点 GPT，2023 年加入实验性 FP8 与 MoE；官方验证最低 Compute Capability ≥ 7.0。重要现状：FT 开发已迁移到 TensorRT-LLM，本仓库冻结，学习价值在于掌握推理加速思想。引入的关键术语——Encoder/Decoder、推理、FP16/INT8/FP8、Tensor/Pipeline parallel、cuBLAS/cuBLASLt——供后续讲义承接。
