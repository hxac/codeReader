本讲把 TensorRT Edge-LLM 的三段式流水线在命令行层面跑通，分导出、构建、推理三个最小模块。导出阶段用 `tensorrt-edgellm-export` 按 `model_type` 分发组件，纯 LLM 只导出 `thinker`，由 `export_onnx` 写出一张同时覆盖 prefill 与 decode 的 ONNX 图，连同 `embedding.safetensors`、`config.json`、分词器等 sidecar。构建阶段 `llm_build` 把参数填入 `LLMBuilderConfig` 后委托 `LLMBuilder::build()`，约束参数 `--maxBatchSize`、`--maxInputLen`、`--maxKVCacheCapacity` 决定引擎输入范围与显存预算，且 engine 不可跨 GPU 型号或 TensorRT 版本迁移。推理阶段 `llm_inference` 加载插件库、解析类 OpenAI 格式的输入 JSON、用 non-blocking stream 构造 `LLMInferenceRuntime`（vanilla 四参数 / spec 五参数）、捕获 CUDA graph、逐请求调用 `handleRequest` 并写出输出 JSON；采样参数写在输入 JSON 而非命令行。
