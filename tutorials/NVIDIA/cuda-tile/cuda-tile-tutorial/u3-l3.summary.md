本讲讲解 `cuda_tile` 方言的「视图（View）」类型族，它在 `ptr<T>` 线性地址之上抽象出「形状 + 步长 + 切割方式」的几何关系，用于把连续全局显存切分成 tile。`tensor_view` 是基础视图，描述元素类型、逐维 shape 与 stride（可动态），是其余视图的底层。`TileView` 类型接口用 `getViewIndexRank`、`getViewTileType`、`verifyIndices` 三个方法统一「坐标→tile」转换；三类分块视图的前两者实现一致，差异集中在 `verifyIndices` 与额外参数。`partition_view` 是完美对齐、无重叠的网格分割；`strided_view` 用 `traversal_strides` 表达带间隔或重叠滑窗；`gather_scatter_view` 用 `sparse_dim` 表达按行号稀疏采集，索引在该维为一维 tile。公共参数 `dim_map` 是 tile 维到 tensor_view 维的排列（默认恒等，非恒等等价于 load 后 permute），`padding_value`（zero/neg_zero/nan/pos_inf/neg_inf）决定越界填充，其中特殊值仅限浮点；不设则为未指定值。校验要点：tile 维须为 2 的幂，traversal_strides 可为任意正值。
