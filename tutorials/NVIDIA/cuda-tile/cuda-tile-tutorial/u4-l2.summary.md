本讲覆盖 cuda_tile 方言 Integer 分组的整数算术与比较操作，所有操作派生自 CudaTileIntegerOpDef，逐元素作用于整数 tile，合法元素类型仅 i1/i8/i16/i32/i64（i4 不可用）。核心判据是「位结果是否与符号有关」：addi/subi/muli/shli 不带 signedness；divi/remi/maxi/mini/shri/cmpi 必须带（Signed/Unsigned 二选一）。加减乘左移与取负默认绕回语义，可挂可选 overflow 提示 NSW/NUW/NW，这是编译期假设而非运行时检查，运行时违反即为未定义行为。divi 默认向零取整，可改为 positive_inf 或 negative_inf，后者与 unsigned 是非法组合，会被 verifier 拒绝；remi 采用截断取余，有符号时结果符号跟随被除数。cmpi 用六种谓词逐元素比较，输出同形状的 i1 tile，signedness 为必填项（equal/not_equal 实际与符号无关）。这套「signedness / overflow / 舍入模式」心智模型将向后续浮点算术讲义承接。
