本讲把视角从「库是什么」推进到「代码在哪里」。顶层分五目录：`hdl/`（可综合 RTL，仅 5 个 `.vhd`：主包、私有包加 round/saturate/resize 三个可流水线化组件）、`bittrue/`（位真模型与测试）、`tb/`（VHDL 测试台）、`sim/`（VUnit 仿真装配）、`lib/en_tb/`（不可综合的测试台基础子库）。核心结论是「三语言镜像架构」：VHDL 为金标准语义，Python 同名同参数镜像之作参考模型，MATLAB 仅做数据格式转换后调用 Python，故算法只维护 VHDL 与 Python 两份。验证闭环由 `bittrue/cosim/` 用 Python 生成黄金数据、`tb/` 逐拍比对 UUT 输出、`sim/run.py` 按 VHDL-93 编译 RTL、VHDL-2008 编译测试台并分库装配完成。引入的关键术语：位真（bit-true）、镜像（mirror）、参考模型、UUT、`FixFormat_t`/`FixRound_t`/`FixSaturate_t`、寄存器模式、`en_tb` 库隔离。
