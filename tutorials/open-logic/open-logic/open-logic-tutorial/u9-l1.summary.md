本讲承接 u8-l3，进入 fix 区域的「高级运算」。核心积木 `olo_fix_madd` 实现 `Out = MaccIn op (A×B)`，带可选预加器，`MaccIn`/`Out_Data` 同格式（`AddChainFmt`）可直接级联成 MACC 链；它故意只截断、不做 round/saturate 以整块映射进 FPGA DSP 块，需要收敛格式时在链末端接 `olo_fix_resize`。复数运算在此基础上拼装：`olo_fix_cplx_mult` 用 2~4 个 madd 实现 MULT4/MULT3/TDM 三种结构与 MULT/MIX（混频，靠对调 Add/Sub 切换）两种模式；`olo_fix_cplx_addsub` 按 Parallel/TDM 实例化 add/sub，是最简复数实体。此外 `olo_fix_bin_div` 用「取绝对值→非恢复除法→还原符号」实现除法，分 SERIAL（省资源）与 PIPELINED（满吞吐）两版，输入有反压、输出无反压；`olo_fix_limit` 把信号钳位到 [LimLo,LimHi]，限幅可静态（`real` 泛型）或动态（端口），格式全相同时无需 round/sat。关键认识：高级 fix 实体多为基本积木（madd/add/sub/resize）的拼装，读懂结构先看它复用了哪些下层实体。
