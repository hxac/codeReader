本讲讲解 Open Logic 的 SPI 主机 `olo_intf_spi_master` 与从机 `olo_intf_spi_slave`。SPI 为同步、串行、全双工协议，主机产生 SCLK 与 CS_n，从机被动响应。核心结论：CPOL/CPHA 定义四种时钟模式，主机用 `getClockLevel()` 与状态分工实现，从机用 `SpiCpol_g=SpiCpha_g` 判采样沿，主从必须配同一模式（注意主机默认 Mode 3、从机默认 Mode 0，默认值不一致）。主机提供命令/响应接口，支持多从机（`SlaveCnt_g`）、可变位宽（`Cmd_TransWidth`）与 CS 保持（`Cmd_CsHold`），SCLK 由 `Clk` 整数分频得到，偏差>10% 报错。从机提供 RX/TX/Response 三接口，TX 在 CS_n 下降沿提前锁存，输入经 `olo_intf_sync` 同步致约 4 拍延迟（故 `Clk` 须 ≥10 倍 SCLK），采样与发送同沿以提速，并支持连续事务与三态 MISO。LSB/MSB 位序由一个双向移位寄存器实现，主从位序也须一致。两实体均采用两进程法加 record 与 AXI-S 握手。
