本讲讲解 Open Logic 的两个 CIC 抽取滤波器：`olo_fix_cic_dec_tdm`（TDM 输入/TDM 输出）与 `olo_fix_cic_dec_par_tdm`（并行输入/TDM 输出）。CIC（级联积分–梳状滤波器）由 N 级积分器（高速）→抽取→N 级差分器（低速）串联而成，只用加减法和寄存器、无需乘法器，常作多级抽取链的第一级，后面再接 FIR 补偿其通带滚降。核心性质是积分累加导致位增长：固有增益 G=(R·M)^N，位增长 b=⌈log₂G⌉。Open Logic 用两级增益校准将增益归一到 1.0：右移 b 位粗校到 [0.5,1]，再乘细校系数 k_fine=2^b/G；G 为 2 的幂时设 `GainCorrCoefFmt_g="NONE"` 可省乘法器。tdm 版用共享累加器+延迟线在时分数据上做多通道积分（积分延迟 Channels_g−1 拍，差分延迟 Channels_g·DiffDelay_g 拍）；par_tdm 版用每通道独立累加器（无积分延迟线），并坚持先抽取→并行转 TDM（`olo_base_wconv_xn2n`）→最后移位，使全通道共用一个移位器。`FixedRatio_g` 决定移位实现：固定模式为纯连线、可配模式需 `olo_base_dyn_sft` 动态移位器（更贵，抽取比只能在复位期改）。两实体共享同一 Python 位真模型，由 VUnit 自动协仿真验证采样率与通道顺序。
