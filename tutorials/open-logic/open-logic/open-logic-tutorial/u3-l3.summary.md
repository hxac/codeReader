本讲围绕位宽转换与 TDM（时分复用）两条线索。先确立 TDM 约定：采样率相同的多路信号在同一接口隐式按 0,1,…,N-1 循环，用 Last 标记每个采样的最后一个通道以重建通道映射。在此基础上讲解四个实体：整数倍的 n2xn（窄到宽 / TDM→并行，采样率÷n）与 xn2n（宽到窄 / 并行→TDM，采样率×n）资源最优，用字使能 WordEna 处理包尾不足整字的情况；任意比的 n2m 用移位寄存器配合最大公约数 GCF 分块与 LastChunk 数组实现，并支持 Trailing-Only 字节使能；tdm_mux 用三级流水线，仅在每采样第 0 通道锁存选择信号、末通道输出一次，输出采样率为输入的 1/N 且无反压。全部转换统一采用小端对齐：最先到达的数据放在位向量最低位，由此决定 TDM 通道的输出顺序。最后以 xn2n 串接 tdm_mux 选取指定通道作为综合实践。
