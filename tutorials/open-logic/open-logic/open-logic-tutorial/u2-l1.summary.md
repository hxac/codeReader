本讲介绍 Open Logic base 区域的五个公共包，是全库的工具箱与类型地基。pkg_array 提供数组类型（定宽/不定宽 StlvArray_t、IntegerArray_t/RealArray_t/BoolArray_t）与 flatten/unflatten 转换，是其他包的类型基础。pkg_math 提供 log2/log2ceil、max/min、choose、count、整数与位向量互转、fromString 等编译期纯函数，用于由泛型推导端口宽度。pkg_logic 提供移位、binaryToGray/grayToBinary、ppcOr、to01、位序翻转及 PRBS 多项式常量等位级函数。pkg_string 提供 compareNoCase、hex2StdLogicVector、errorMessage 等字符串工具，用于解析字符串型泛型。pkg_attribute 封装跨厂商综合属性，包体为空，靠"一次声明全部厂商属性加语义常量"让一份 VHDL 跨 Vivado/Quartus/Efinity/Gowin 可综合。核心结论：包间依赖链为 array → math → {logic, string}，attribute 独立无依赖，直接决定 compile_order 前几行顺序；math/logic 故意不调用 errorMessage 以避免循环依赖；log2ceil 对 0 特判返回 0 以支持零长度数组；grayToBinary 因链式数据依赖需循环而 binaryToGray 一行并行。各包均有配套 VUnit 测试台校验。
