本讲把采样→奖励→优势→损失→更新串成完整链路。GRPO 用同一 prompt 的 N 条回答做组内归一化得优势 A=(R−μ)/(σ+ε)，丢弃 Critic 价值网络，代价是退化组风险（监控 Adv Std）。奖励为长度/思考格式/闭合/重复惩罚与 RM 分数的混合。损失共享 ratio=exp(logp−old_logp) 与 k3 估计的 token 级 KL；GRPO 用 min(r·A, clip(r)·A)，CISPO 改写为 clamp(ratio).detach()·A·logπ——被夹的 ratio 仅作权重、梯度从 logπ 流过，避免 clip 截断梯度，作为 loss_type 默认变体。completion_mask 剔除首个 EOS 后与 padding 位置。三模型分工：策略(更新)、参考(冻结供 KL 锚点)、奖励(打分)，学习率 3e-7 远小于 SFT。本实现单次 rollout 单次更新使 ratio≈1、clip 基本不激活，故二者数值接近。
