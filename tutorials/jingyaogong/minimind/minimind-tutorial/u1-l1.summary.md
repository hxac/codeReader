MiniMind 是一个从 0 用 PyTorch 原生实现、不依赖 transformers/trl/peft 高层封装的超小语言模型全链路训练项目，主打「大道至简」与透明可学，门槛低至单卡「3 块钱 / 2 小时」。它同时承担两个角色：可端到端复现的 LLM 训练工程，和面向初学者的入门教程。本讲覆盖三个最小模块：项目介绍、更新日志、已发布模型列表。核心结论是其训练链路即后续各讲目录——Pretrain（预训练）→ SFT（监督微调）→ 可选的 LoRA（低秩微调）、知识蒸馏、RLHF-DPO、RLAIF（PPO/GRPO/CISPO）、Agentic RL（多轮工具调用强化学习）。当前主线为 minimind-3（64M Dense）与 minimind-3-moe（198M-A64M，4 experts/top-1），结构对齐 Qwen3 生态（Decoder-Only + RMSNorm + SwiGLU + RoPE + GQA），v1/v2 已成历史。引入的关键术语与规律：训练阶段、`trainer/train_*.py` 脚本、`xxx_{dim}.pth` 权重名三者一一对应，MoE 权重追加 `_moe` 后缀；2026-04-01 大版本后「思考」不再是独立训练阶段，而是由 chat_template + `<think>` + open_thinking 开关控制，toolcall 也已混入主线 SFT 数据。RLAIF 与 Agentic RL 完成 rollout engine 解耦，支持训推分离。后续讲义将逐层拆解这条链路。
