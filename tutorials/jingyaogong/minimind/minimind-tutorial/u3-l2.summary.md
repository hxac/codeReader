本讲深入 `MiniMindBlock` 的两个核心子模块。**RMSNorm** 是 LayerNorm 的简化版，仅做方差归一化（不去均值、无偏置），用可学习缩放 γ 还原幅度，前向中先升 fp32 计算再回落原精度以保数值稳定。**GQA**（分组查询注意力）让 Q 头（`num_attention_heads=8`）多于 KV 头（`num_key_value_heads=4`），`n_rep=8//4=2`，每个 KV 头被两个 Q 头共享，砍半 `k_proj`/`v_proj` 参数与推理时 KV Cache 显存；**repeat_kv** 用 `expand`+`reshape` 把 KV 头广播对齐到 Q 头数，`n_rep==1` 时短路返回，使同一份代码兼容 MHA/GQA/MQA。`Attention.forward` 数据流为：q/k/v 投影 → q_norm/k_norm（按 `head_dim` 归一化、V 不做）→ RoPE → KV Cache 在序列维 `cat` 累积 → repeat_kv → 缩放点积注意力（除以 √head_dim，加 causal mask）→ o_proj。注意力仅在整段前向、无 padding、无缓存时走 Flash（`scaled_dot_product_attention`），单 token 解码、带缓存或带 mask 时退回手动 `Q@K^T + causal mask` 实现，优先保证语义可控。`MiniMindBlock` 采用 Pre-Norm + 残差结构。
