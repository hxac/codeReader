本讲讲解寄存器地址映射自动生成工具 newad.py，核心是「单点标记、多点生成」：在 Verilog 端口写一行 magic 注释，工具即自动分配地址、生成解码器、地址表与寄存器表，样板代码降到接近零。两类 magic 注释：external 标记寄存器端口（可附 single-cycle/we-strobe/plus-we/strobe 属性），auto 标记需递归下钻的实例（可带时钟域覆盖与 generate 展开）；以 _addr 结尾的 output 会把同名 input 升级为 DPRAM 接口。newad 输入顶层 Verilog 与目录列表，产出 _auto.vh 解码器、addr_map_*.vh 地址命中宏、regmap_*.json 软件用名字-地址表；地址按位宽 2^n 对齐贪心分配，大块内存留空洞。parser.py 的 make_decoder_inner 按 signal_type 把同一端口展开为保持型寄存器、单脉冲、读选通、写使能或 DPRAM。默认正则后端轻量但脆弱，yosys 后端用真解析器规避局限，二者共享端口建模与地址逻辑；软件始终按名字访问寄存器。
