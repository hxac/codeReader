digaree（Digital Arithmetic Execution Engine）是 Bedrock 中介于 FPGA 逻辑与通用 CPU 之间的定点协处理器，定位为运行 RF 失真校正、SRF 腔体 detune/淬灭检测等「几十次乘加」算式。其核心特征是单数据类型、无分支、每条指令定周期，把软件编译流水线内嵌进硬件工程。本讲围绕三个最小模块展开：代码生成链（cgen_srf.py 描述算式产出 ops.h，sched.py 做寄存器分配与列表调度产出 ops.vh，最终被 sf_user.v 的 case(pc) include 进指令 ROM）；定点数学（18 位端口、22 位内部、sf_inv 用 casez 查表给出倒数初值、full_inv 经两次 Newton 迭代精化、pfloat.py 负责 SI 浮点到定点的定标与溢出自检）；执行核与控制层（sf_main.v 的 21 位指令字解码、双端口分布式 RAM 寄存器堆、5 级流水与并行 mul/add/inv 单元、pipe_len=5 与调度器对齐；sf_user.v/sf_user_wrap.v 提供 PC、ROM、meas_mux 参数注入、饱和计数及 newad.py/localbus 接口）。核心结论是「Python 描述算法 → 生成 Verilog → C 与 Verilog 位精确双仿真逐行 cmp 比对」的自测范式，工程与其它子系统解耦、自带独立 Makefile。
