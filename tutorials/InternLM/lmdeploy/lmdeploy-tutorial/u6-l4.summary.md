本讲讲解 TurboMind 的 builders 层——位于 spec（`models/*.py`）与 C++ 句柄之间的「结构描述层」：spec 遍历模型层级、读 HF 权重，builders 负责把 `Linear` 权重组装、融合、TP 切分后灌入 C++ 模块。核心抽象是 `Builder` 基座，统一生命周期为「暂存提交（stage）→ `build()` 落盘 → 冻结」：子模块靠 `__setattr__` 拦截 `BuiltModule` 赋值实现暂存，`SplitSide.OUTPUT/INPUT` 是列/行并行的唯一词汇。`TextModelBuilder` 是根（持 `tok_embeddings` 与 `lm_head`），`ModuleListBuilder` 用 `__setitem__` 表达有序容器，`DecoderLayerBuilder` 为纯容器。`AttentionBuilder` 将分离的 Q/K/V 融合成 `w_qkv`（列并行，经 dequant→repeat_kv→回填头数→TP 交错四步，运行时零开销），`wo` 行并行。`MoeBuilder` 极薄，只管 gate（广播）与非专家参数，专家外挂 `ModuleList`（复用 `FfnBuilder`）；`MLABuilder` 含数学变换，离线 `fold_kv_b` 把 KV 压缩矩阵折进 q/o 投影以省 KV cache。
