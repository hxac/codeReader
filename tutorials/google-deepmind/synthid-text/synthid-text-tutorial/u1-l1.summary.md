本讲是整本手册的开篇，只通过 `README.md` 和 `pyproject.toml` 两个「门面文件」建立全局认知，不涉及 `.py` 源码。核心结论是：SynthID Text 是 DeepMind 发表在《Nature》论文里、用于识别 LLM 生成文本的水印方法的**参考实现（reference implementation）**，非生产实现。它的核心思路不是事后靠风格猜测，而是**生成时嵌入隐藏统计信号、检测时读取**，更可靠也更难绕过。系统分为两个阶段：**水印施加（watermarking）**用 PyTorch，借 logits 处理器和 mixin 挂到 HuggingFace 模型上；**水印检测（detection）**用 JAX/Flax，检测侧又分 **Weighted Mean（免训练）** 与 **Bayesian（需按密钥训练）** 两条路线，输出一个 [0,1] 分数，需自定阈值。官方演示模型为 Gemma 与 GPT-2。引入的关键术语包括：水印、g 值、accumulate_hash、静态水印配置、ngram、keys/层数、阈值与假阳率。重要边界：仅供研究复现、哈希无密码学安全保证、静态配置不适合生产（生产用 HF Transformers 官方实现）；且文档与源码可能不一致（如 `train_detector_bayesian.py` 实为 `detector_bayesian.py`），遇冲突以源码为准。
