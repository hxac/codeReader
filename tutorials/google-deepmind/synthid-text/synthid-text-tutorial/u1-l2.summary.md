本讲只讲"怎么装、跑、测"，围绕 pyproject.toml、README、ci.yaml。核心结论：核心依赖同时含 PyTorch（水印施加）与 JAX/Flax（检测），关键库被锁死版本以保证可复现。安装靠可选依赖组分三档：仅核心、notebook-local（补 Jupyter 与数据工具）、test（补 pytest）。官方主入口是自包含的端到端 Notebook（加载带水印模型→生成文本→重算 g 值→打分），GPT-2 任意配置即可跑。测试由 pytest 自动发现 *_test.py，GitHub CI 在 Python 3.9–3.11 上用可编辑安装 + pytest -v 验证。引入术语：虚拟环境、可选依赖组（extras）、可编辑安装（-e）、g 值、得分。
