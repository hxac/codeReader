本讲引入 SynthID Text 检测侧的可训练路线——贝叶斯检测器（detector_bayesian.py，JAX/Flax），用贝叶斯公式直接建模后验 P(w|g)。相比上讲 mean_score 的等权平均，它以可学习似然实现按位置、按层的信息量加权。核心含三个最小模块。其一，两个不对称似然模型：水印侧 LikelihoodModelWatermarked 携带可学参数 beta（每层偏置）与 delta（层间权重矩阵）；非水印侧 LikelihoodModelUnwatermarked 无参数、对所有 g 值返回常数 0.5，因非水印 g 值由哈希取出、理论服从 Bernoulli(0.5)，该常数充当无信息基线。其二，潜变量 ψ（p_two_unique_tokens）刻画「锦标赛是否真有两个不同 token」，由 logistic 回归 sigmoid(Δx+β) 参数化，经下三角掩码做成自回归（第 l 层只看更浅层），故 L2 正则只罚 delta；水印似然 0.5[(g+0.5)ψ+(1−ψ)] 是锦标赛分布(0.75/0.25)与无偏硬币(0.5/0.5)按 ψ 的混合，ψ=0 处对后验零贡献。其三，_compute_posterior 是贝叶斯公式对数几率形式的直译：相对惊奇度按掩码求和、加可学习先验的对数几率、再 sigmoid，clip 与对数计算保证数值稳定。本讲只讲推理链路，beta/delta/先验当作已知，训练与端到端 API 留待后续讲义。
