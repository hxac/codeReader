本讲拆解 NVDLA 时序地基。NVDLA 分两个时钟域：core 域 nvdla_core_clk（计算与存储，源自顶层 dla_core_clk）与 falcon 域 nvdla_falcon_clk（配置，源自 CSB 时钟 dla_csb_clk）。跨域单比特信号须过三级同步器 sync3d，多比特走异步 FIFO。sync3d 家族按上电默认值分三档：无复位、_s 置位（默认 1）、_c 清零（默认 0），按信号安全默认态选用，中断跨域用 _c。复位遵循「异步复位、同步释放」，由 NV_NVDLA_reset（单源）与 NV_NVDLA_core_reset（双源 AND，两路都释放才解除）以 sync_reset 实现；partition_o 中 falcon 复位以 core 复位为输入，形成先 core 后 falcon 的上电顺序。slcg 二级时钟门控在各子引擎内按 op_en 与工作模式门控相与决定开/关钟以省电；override 信号经 sync3d_s 同步后可强制开钟调试。
