本讲围绕 `components` 包，介绍如何把触发器、选择器、计数器、移位寄存器、比较器等硬件原语封装成一行可调用的函数。核心思想是：VHDL function 无时钟、无状态，触发器被拆成「计算下一态的组合函数」加调用方的 `when rising_edge(Clock)` 自反馈，寄存器因此「长」在调用现场而非函数体内。`ffdre` 是带使能与同步复位的 D 触发器，靠 `SIMULATION` 在仿真模型与可综合布尔式之间切换；`ffdse`、`fftse`（置位版）以及异步 `ffrs`/`ffsr` 均由其复用派生。`mux` 是位宽无关的二选一选择器，`en` 本质即内嵌的 mux；`registered` 把「是否寄存一拍」做成可配置的 `when` 条件，一个 generic 即可开关流水线级数。最关键的易踩坑点：函数的 `INIT` 参数是复位值，不是 FPGA 上电初值；上电初值必须写在信号声明 `signal q := ...` 里，两者名字撞车但语义不同，工程上应同时设置并保持一致。此外该包不在 `context Common` 套餐内，需单独 `use PoC.components.all;`。
