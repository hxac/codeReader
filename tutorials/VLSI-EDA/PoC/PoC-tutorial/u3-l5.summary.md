`src/arith/` 是 PoC 的算术工具箱，由一组互不相干的小算术核组成，统一在根包 `arith.pkg.vhdl` 中声明并先于具体核编译（命名空间包模式）。本讲覆盖三组最小模块：计数器族、优先编码器、随机与迭代运算核。计数器族按输出形态分家：`arith_counter_free` 仅输出单比特选通 `stb`（直接取自寄存器 MSB，位宽用 `log2ceil(DIVIDER)` 推算，刻意不暴露计数值以放权综合器）；`arith_counter_gray` 输出 Gray 码值，每步只翻转 1 位，用于跨时钟域安全读取（如 FIFO 指针）；`arith_counter_ring` 为独热/Johnson 环形计数；`arith_counter_bcd` 输出十进制。核心结论是：跨域传计数值须用 Gray 码计数器，传单比特选通用 free 计数器加同步器即可。`arith_firstone` 把优先编码包装成「令牌传递」接口，通用实现用 `~rqst + tin` 进位链扫描定位最低位请求，Xilinx 专用实现用 `MUXCY` 进位链原语，由 `VENDOR` 守卫二选一。`arith_prng` 为 XNOR 反馈 LFSR（抽头来自 XAPP052，3~168 位，使全 0 合法）；`arith_sqrt` 为 `(N+1)/2` 步迭代开方；`arith_addw` 为 carry-select 宽位加法，架构由 `tArch` 枚举切换。关键术语：LFSR、Gray 码、进位链、令牌传递仲裁、carry-select、握手启动多周期迭代时序，为后续跨时钟域同步器与跨钟 FIFO 讲义承接。
