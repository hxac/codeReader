本讲建立对 PoC 的最外层认知。PoC 全称「Pile of Cores」，是以 VHDL/Verilog 源码形式交付的可复用硬件 IP 核库，覆盖算术、缓存、时钟域穿越（CDC）、FIFO、RAM 包装器、I/O 控制器等常见功能，每类在 `src/` 下都有对应子命名空间。完整的 PoC 由「核 + 公共 VHDL 包 + 分组层级 + Python 基础设施（pyIPCMI，用于统一驱动多家厂商工具链）」四部分组成，本讲只覆盖最外层。项目由德累斯顿工业大学 VLSI 教席维护，采用 Apache License 2.0（允许商业使用与修改，含专利授权），2014-12-16 初次提交，2016-05-13 发布 1.0（Python 基础设施完全重写）。关键现状：本仓库 `VLSI-EDA/PoC` 是历史快照，已 fork 到 Open-Source VHDL Group 维护的 `VHDL/PoC`（用 GHDL、NVC 与 OSVVM 测试台经 GitHub Actions 检查）。此外 `README.md` 由 `README.tpl` 自动生成，不可直接手编。
