本讲是 LightLLM 学习手册首篇，只覆盖两个最小模块——**项目说明**与**版本与依赖**。核心结论：LightLLM 是一个纯 Python 的 LLM 推理与服务框架，主打轻量、易扩展、高性能，区别于 vLLM/TGI/FasterTransformer 的三大技术特色为：① 纯 Python 主体（性能关键路径才下沉为 Triton/CUDA kernel）；② token 级 KV Cache 管理（分配回收精确到单 token，为 chunked prefill、RadixCache 复用奠基）；③ 多进程协作架构（HttpServer/Router/ModelBackend/Detokenization 分离，经 zmq、rpyc、共享内存通信）。引入的关键术语：推理与服务框架、KV Cache、张量并行、Router 调度、token 级管理。环境基线：当前版本 1.1.0，要求 Python≥3.9.16，setup.py 的 install_requires 给出 pyzmq/rpyc/uvloop/transformers/safetensors/triton 等 10 个最小依赖，torch 等重依赖锁于 requirements.txt。项目借鉴 FasterTransformer/TGI/vLLM/flashinfer 等，并有 Pre³ 约束解码、Past-Future Scheduler 调度器等论文支撑。
