本讲义精读 `scipy.special/_basic.py`,核心论题是：为何其中一族函数不是 NumPy ufunc。给出第二组理由（承接 u4-l1 的「整数位数无界」）：**输出结构由标量整数参数决定**，而 ufunc 的输出形状只能由输入广播决定。涵盖三类函数：①零点函数 `jn_zeros`/`yn_zeros`/`jnyn_zeros`/`kelvin_zeros`，输出长度由标量 `nt` 决定，真正求根下放给 `_specfun.jyzo`/`klvnzo`；②导数族 `jvp/yvp/ivp/kvp/h1vp/h2vp`，共享 `_bessel_diff_formula`，用 DLMF 10.6.7 解析递推把第 n 阶导表达为相邻阶贝塞尔的线性组合，复用底层 `jv/yv/...` ufunc 批量求值（非数值微分），破坏 ufunc 资格的是标量整数 `n`——它选择算法而非逐元素运算；③「一次返回所有阶」序列函数 `riccati_jn/yn`（走 gufunc `_rctj/_rcty`）与 `lmbda`（走 `_specfun` 的 `lamv/lamn`），输出多一条长度 `n+1` 的阶轴。引入术语：相位 `phase`（J/Y/H 为 -1，I/K 为 +1，`kvp` 额外乘 `(-1)**n`）、gufunc、`_specfun`（Zhang & Jin）、通用模式「Python 入口校验分派 + C/C++ 内核填满超长数组 + Python 切片」。
