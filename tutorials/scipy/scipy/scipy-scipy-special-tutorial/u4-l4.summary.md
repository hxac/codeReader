本讲聚焦 scipy.special 中「薄包装」模式，覆盖 `_lambertw.py` 与 `_spherical_bessel.py` 两个 Python 包装层文件。薄包装指本身并非 ufunc、只在底层 ufunc 之外做轻量收尾的纯 Python 函数，典型工作有三：承载 ufunc 装不下的带默认值配置参数（如 `lambertw` 的 `k`/`tol`）、把整数阶/分支号强制规整为 `long` 以稳定命中 C 内核、以及按布尔开关分发。以 `derivative` 参数为例，布尔值在 Python 层选择 `_xxx` 与 `_xxx_d` 两个底层 ufunc，因为 ufunc 类型分发只看 dtype、看不到布尔开关。核心概念 `use_reflection` 装饰器依据 DLMF 10.47(v) 奇偶反射公式，将实数负半轴输入翻到正半轴再补符号：符号由 `sign_n_even` 决定（jn/in 给 +1、yn 给 -1），导数模式依链式法则再乘 -1；复数输入跳过反射，实数输入用 `xpx.apply_where` 按掩码分路求值以避免在负实数上误调内核。`spherical_kn` 因反射关系非简单符号翻转，通过 `reflection_fun` 注入口走「提升复数 + 取实部」的解析延拓。
