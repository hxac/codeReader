本讲解读 `scipy/special/_logsumexp.py`，覆盖 `logsumexp`、`softmax`、`log_softmax` 三个纯 Python 函数。核心动机是数值稳定性：朴素 `log(sum(exp(a)))` 会因 `exp` 溢出失效，解法是 **max-shift 技巧**——先减去最大值再取指数，使所有指数恒非正，最后把最大值加回。`_logsumexp` 进一步采用 Blanchard–Higham 精化：把最大项单独抽出用 `log1p` 高精度计算余项。三者均非 ufunc，因其跨元素求和破坏了逐元素前提，构成「为何不用 ufunc」的一类理由。进阶特性包括：权重 `b` 可负、`return_sign` 做符号/幅度分离（复数下返回单位复数辐角）、复数结果虚部经 `_wrap_radians` 折回 (-π, π]。跨后端能力由 Array API 三件套提供：`array_namespace` 推断后端、`xp_promote(force_floating=True)` 统一 dtype、`@xp_capabilities()` 标注能力矩阵；默认 `SCIPY_ARRAY_API` 关闭即纯 NumPy，开启后才分发到 PyTorch/JAX 等。关键术语：max-shift、log1p 精化、符号分离、复数相位包裹、Array API，供后续讲义承接。
