本讲精读 scipy.special 最底层的 C++ 内核 xsf_wrappers.h/.cpp。xsf 是 SciPy 自研的现代 C++ 特殊函数库,函数居于 xsf 命名空间(含折叠进来的 xsf::cephes::);xsf_wrappers 是套在外的 extern "C" 薄壳,为只懂 C 方言(npy_cdouble、函数指针)的 Cython 与生成式 ufunc 提供稳定的 C 链接符号。关键术语:两种复数桥接——返回值与按值复数参数用 to_complex/to_ccomplex 值拷贝,多输出指针参数用 reinterpret_cast 零拷贝,二者均依赖 npy_cdouble 与 std::complex<double> 内存布局相同;三套命名约定 special_*/xsf_*/*_wrap(复数版带 c 前缀)与三种转发骨架(纯实数直传、复数单返回、多输出指针);双轨注册——纯 xsf 单来源函数(如 agm)走 _special_ufuncs.cpp 直接注册,需混编 Boost+xsf 的函数(如 hyp1f1)留在 functions.json 多源路径。wrapper 另有职责:为 cython_special 的 typed 标量 API 提供 cimport 符号,故即便 ufunc 已迁走仍需保留。
