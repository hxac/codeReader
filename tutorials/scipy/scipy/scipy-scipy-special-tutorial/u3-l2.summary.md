本讲精读 `_generate_pyx.py`——把 `functions.json` 声明表翻译成可编译 ufunc 注册代码的「编译器前端」，由 `meson.build` 的 `custom_target('cython_special')` 驱动，落盘五个文件。主流程为：读 JSON → `special_ufuncs` 闸门过滤（排除 189 个已迁移到 C++ 直注册路径的函数，如 `airy`、`erf`）→ 构造 `Ufunc` → `generate_ufuncs`。关键术语与概念：`Func`/`Ufunc` 用两条正则把签名解析成 `(输入,输出,返回值)`，`*` 前缀的返回值（如 sf_error 状态码）被丢弃；`Ufunc` 强制要求 docstring，缺则编译前报错；`iter_variants` 自动派生 `int→long` 及（无整数参数时）`d→f`/`D→F` 的 float32 双胞胎，实现「声明一条、生成多条」；`generate_loop` 产出 NumPy 标准逐元素 C 循环，对 `DANGEROUS_DOWNCAST` 做运行时守卫并调 `sf_error.check_fpe`；`Ufunc.generate` 拼装 `PyUFunc_FromFuncAndData` 完成注册。C++ 内核（如 `betainc`）则外包给 `_ufuncs_cxx`，经导出函数指针 + name override 隔离，避免 C/C++ 混编难题。
