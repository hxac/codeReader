本讲讲解 scipy.special 的 `_basic.py` 中 comb、perm、factorial、factorial2、factorialk、stirling2 等组合与阶乘类函数。它们均为纯 Python 而非 ufunc:`exact=True` 要返回任意精度 Python 大整数且常只接受标量,违反 ufunc 的「固定类型 + 必然逐元素 + 形状一致」契约。核心是 exact 双轨:`exact=False` 委托 binom/poch/gamma 等底层 ufunc 做浮点近似、可批量;`exact=True` 用 `math.comb` 与连乘给出无损整数、更慢。factorial 三函数共享派发器 `_factorialx_wrapper`,差异仅为连乘步长 k;`extend` 参数控制负数处理(`'zero'` 返回 0、`'complex'` 走 Gamma 延拓,与 exact 互斥)。整数溢出保护由阈值表 `_FACTORIALK_LIMITS_64BITS/32BITS` 驱动 `_factorialx_array_exact` 做 dtype 三级升级(long→int64→object),辅以分治连乘 `_range_prod`。便利函数策略各异:diric 用三层掩码补可去奇点、sinc 纯转发 `numpy.sinc`、bernoulli/euler 为外包 `_specfun` 的序列函数(输出长度依赖输入值,故非 ufunc)、softplus 转发 `np.logaddexp` 借力数值稳定实现。
