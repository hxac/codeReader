本讲剖析 `scipy.io._mmio.py`——Matrix Market（`.mtx`）纯文本矩阵交换格式的纯 Python 实现。格式文件分四段：首行 `%%MatrixMarket matrix <format> <field> <symmetry>`、注释行、尺寸行、数据行。核心类 `MMFile` 用三组类常量 `FORMAT_*`（coordinate 稀疏 COO / array 稠密列主序）、`FIELD_*`（integer/real/complex/pattern 等，经 `DTYPES_BY_FIELD` 映射到 numpy dtype）、`SYMMETRY_*`（general/symmetric/skew/hermitian）将规范代码化并配 `_validate_*` 校验。模块级 `mminfo`（只解析头部六元组）、`mmread`、`mmwrite` 三个函数均转发到 `MMFile.info`/`read`/`write`。读取时 `array` 按列主序填稠密数组，`coordinate` 读 `I/J/V` 三数组并做 1 基转 0 基（`I-=1`），对称矩阵只存下三角含对角、读回时按对称关系镜像补全上三角（取负或共轭）。`mmwrite` 可自动推断 field 与 symmetry，`_get_symmetry` 实地探测并以 symmetric>skew>hermitian>general 的优先级返回。关键术语：COO、对称/反对称/厄米特、pattern 字段、round-trip 一致性。本实现与 1.12 后的 C++ 后端 `_fast_matrix_market` 接口一致，是后者行为参考（详见 u4-l1）。
