本讲深入 scipy.io._netcdf（对外暴露 netcdf_file / netcdf_variable），这是 NetCDF3（Classic 与 64-bit offset）的纯 Python 读写器，不支持 NetCDF4。核心结论：NetCDF3 是自描述二进制格式，由 CDF 魔数 + 版本字节 + numrecs + 维度/属性/变量三段 header + data 区构成，全程大端序，空段用 ABSENT 表示。关键术语包括：dimensions/variables/attributes 三段式数据模型、记录维度（record dimension，唯一可无限增长，长度记 None）、TYPEMAP/REVERSE/FILLMAP 三张类型映射表（打通 NetCDF 六种类型码与 numpy dtype，不在 REVERSE 的 int64 会被拒绝）。netcdf_file 采用「两遍写」（先写 begin 占位 0 再 seek 回填真实偏移），记录变量排在最后；netcdf_variable 通过 isrec 判定记录变量，maskandscale=True 时做 _FillValue（优先于 missing_value）掩码与 scale_factor/add_offset 线性缩放，记录变量赋值时自动扩容。mmap 在只读真实文件时默认开启，变量数据是零拷贝 view，但数组存活时无法干净关闭文件，close 借 weakref 检测并发 RuntimeWarning 提示先 .copy()。
