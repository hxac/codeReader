本讲覆盖两个最小模块：`HalfspaceIntersection`（`_qhull.pyx` 中的高层壳类）与 `_plotutils.py` 的三个 2D 绘图函数。

核心结论：`HalfspaceIntersection` 反转了前几讲的输入范式——输入不再是点，而是半空间约束（行形式 `Ax+b≤0`，数组形状 `(nineq, ndim+1)`），输出是这些半空间交集（可行多面体）的顶点 `intersections`，对应 Qhull 的 `qhalf` / 模式 `H`。Qhull 不直接求交，而靠**对偶变换**把问题转为它擅长的凸包：给定严格内部点 `interior_point`（强制参数，须满足 `A·p+b<0`），每个半空间映成对偶点 `d=-A/(A·p+b)`，可行多面体的顶点与对偶点凸包的面一一对应，最后用 `intersections = dual_equations[:,:-1] / -dual_equations[:,-1:] + p` 换算回原空间。

引入的关键术语：半空间、可行区域、内部可行点 `interior_point`、对偶变换与中心对偶、`dual_points`（所有半空间的对偶点）、`dual_vertices`（真正起作用的半空间下标，冗余约束不在内）、`dual_equations`、`dual_volume/dual_area`（属对偶凸包而非原空间，是常见陷阱）。换算中枢是 `_update`，可行点校验用 `dists >= 0`。

绘图模块：`delaunay_plot_2d`/`convex_hull_plot_2d`/`voronoi_plot_2d` 三函数仅支持 2D，统一约定为可选 `ax`、返回 `figure`、收尾调 `_adjust_bounds`（±10% 极差留白）；`voronoi_plot_2d` 额外处理无限脊（虚线 + 法向延伸，含修 GH#19653 的 `aspect_factor`）。`_get_axes` 用惰性导入 matplotlib 使 `scipy.spatial` 不强依赖它。半空间求交无官方绘图助手，需手写。`_Qhull` 底层封装细节留待 u10-l1/u10-l2。
