本讲专讲 Delaunay 三角剖分（凸包、Voronoi 留给后续讲义）。核心结论：Delaunay 满足「空外接圆」准则，几何上等价于把点抬到抛物面 z=‖x‖² 后求下凸包再投影回去，Qhull 即据此把 Delaunay 转为它擅长的凸包问题。三个最小模块：Delaunay 类构造、结果数组编码、点定位与重心坐标。源码上 Delaunay 是高层壳，真正计算由底层 `_Qhull`（持有 C 库 qhT）完成，链路为 `__init__` → `_Qhull(b"d")` → `_update` → `get_simplex_facet_array`。关键术语：simplices（每行单纯形的点索引）、neighbors（第 k 个邻居正对第 k 个顶点，边界记 -1）、coplanar（三元组「点/单纯形/顶点」，记录未参与剖分的点）、qhull_options（默认 `Qbb Qc Qz Q12`，强制 `Qt` 保证单纯形输出，`Qc` 填 coplanar，`QJ` 强制每点出现）。`find_simplex` 用「抛物面粗游走 + 重心坐标定向搜索」定位，外部返回 -1；`transform` 存 T⁻¹ 与 r_n 可手算重心坐标（和为 1）；`plane_distance` 给提升点到各面有符号距离；`tsearch` 已弃用，等价 `find_simplex`。
