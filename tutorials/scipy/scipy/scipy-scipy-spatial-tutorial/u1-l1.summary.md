`scipy.spatial` 是 SciPy 中负责空间几何与空间数据结构的子包，底层借助 Qhull 等 C/C++ 库实现高性能。本讲义只盯住 `__init__.py` 与官方教程两个文件，建立整本手册的「导航地图」，不涉及算法细节。

核心结论有三点。其一，定位：它解决两类典型问题——「离我最近的是谁」（最近邻、KDTree）与「这些点的几何形状是什么」（三角剖分、凸包、Voronoi），并辅以距离度量函数与空间变换。

其二，能力归纳为四大域：最近邻查询（`KDTree`/`cKDTree`）、距离度量（`distance` 子模块）、Qhull 几何剖分（`Delaunay`/`ConvexHull`/`Voronoi`/`SphericalVoronoi` 等）、空间变换（`transform` 子模块、`geometric_slerp`、`procrustes`）。其中需区分两种挂载方式：直接挂载对象由 `from ._xxx import *` 搬到顶层；`distance`/`transform` 则作为子模块命名空间保留，对象藏在下一层。

其三，`__all__` 采用动态推导：`[s for s in dir() if not s.startswith('_')]`，主体按字母序排列，再追加 `distance`/`transform`，预期共 22 项。

引入的关键术语供后续承接：子包、`import *`、`__all__`、四大能力域、直接挂载与子模块命名空间、弃用命名空间（`ckdtree`/`kdtree`/`qhull`，访问触发 `DeprecationWarning`，v2.0 移除）、`PytestTester` 测试钩子。后续讲义将逐个内部模块深入。
