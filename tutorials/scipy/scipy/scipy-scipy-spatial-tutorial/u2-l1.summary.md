本讲围绕 `scipy.spatial` 的 KDTree 入门，覆盖三个模块。其一，kd-tree 基础：它是一种沿坐标轴递归切分空间的空间划分数据结构，把最近邻查询从朴素 O(n) 降到约 O(log n)，靠「区域到查询点最小距离」做整子树剪枝；引入术语最近邻/k 近邻查询、leafsize（停止切分改用暴力比较的阈值）、滑动中点切分规则。其二，继承关系：`KDTree` 是 `cKDTree` 的纯 Python 子类，真正建树与查询都落在 Cython 内核 `cKDTree`，`KDTree` 仅做校验与 `super()` 转发，两者行为一致但默认 leafsize 不同（10 vs 16）；数据被强制为只读 C 连续 float64 并拒绝 nan/inf。其三，构造参数：leafsize、compact_nodes、balanced_tree、copy_data 只影响树形或数据隔离，不影响正确性；boxsize 开启周期边界（环面拓扑），数据须落在 [0,L)，距离按最小镜像计算，0 维视为非周期。后续讲义将承接 Rectangle 几何基元与各类查询语义。
