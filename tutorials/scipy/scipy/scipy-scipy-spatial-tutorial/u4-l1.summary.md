本讲专讲 `scipy.spatial.distance` 中接收两个一维向量、返回标量的距离函数，全部位于 `distance.py` 的纯 Python 实现，核心围绕四条主线。其一，输入校验基座 `_validate_vector`（强制一维、转 C 连续）与 `_validate_weights`（权重非负），它决定了批量能力：仅 `minkowski`/`euclidean`/`sqeuclidean`/`jensenshannon` 用 `_asarray` 支持 `(...,N)`，其余只接受一维。其二，范数族为 Lp 范数特例——`euclidean` 复用 `minkowski(p=2)`，`sqeuclidean` 用 `vecdot` 省开方且权重只应用一次，`chebyshev` 权重仅为 0/1 开关。其三，角度类 `correlation` 与 `cosine` 同源，后者即不中心化的前者，二者均 clip 到 [0,2] 并拒绝复数。其四，参数化度量：`seuclidean` 以 `1/V` 当权重复用 `euclidean`，`mahalanobis` 走二次型 `delta·VI·delta` 且 `VI` 是协方差之逆，`canberra` 用 `nansum` 处理 0/0，`jensenshannon` 用 `rel_entr` 实现并支持 `axis`。关键术语：度量公理、Lp 范数、中心化、二次型、标准化。成对距离 `pdist`/`cdist` 留待后续讲义。
