本讲围绕 `scipy.signal.max_len_seq`，讲解最大长度序列（MLS）的原理与实现。MLS 是线性反馈移位寄存器（LFSR）产生的伪随机二进制序列，排除全 0 状态后周期恰为 2^n−1。LFSR 采用 Fibonacci 型，反馈抽头（taps）取自本原多项式并预存于 `_mls_taps` 表；加速内核 `_max_len_seq_inner` 用环形缓冲（头指针 idx 模运算）替代物理移位，单拍近似 O(1)，并以 `np.roll` 收尾使状态可续算、支持分段拼接。该内核有 Cython 与 Pythran 两份逐行一致的等价实现，由 meson.build 的 use_pythran 开关二选一，产出同名模块对上层透明。将 0/1 映射为 ±1 后，MLS 循环自相关在零位移处为峰值 N、其余为 −1，近似冲激，因而是系统辨识（测冲激响应）与白谱激励的理想探针。本讲引入的关键术语：MLS、PRBS、LFSR、taps、本原多项式、环形缓冲、循环/线性自相关、系统辨识，以及 Cython/Pythran 双路径编译模式。
