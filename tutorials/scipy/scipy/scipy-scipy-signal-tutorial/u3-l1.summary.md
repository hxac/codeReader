本讲是「卷积与相关」单元首篇，围绕 `_signaltools.py` 与两个 C/C++ 头文件，讲解直接法卷积/相关的接口语义，分四个最小模块。核心结论：卷积与相关只差一个「翻转（含复数取共轭）」，scipy 借 `_reverse_and_conj` 让 `correlate` 与 `convolve` 互递归复用。`mode`（`full`/`same`/`valid`）决定输出尺寸，经 `_valfrommode`/`_modedict` 映射为整数 0/1/2；`boundary`（`fill`/`wrap`/`symm`）决定边界延拓，经 `_bvalfromboundary`/`_boundarydict` 映射并左移两位为 0/8/4。C 内核 `_sigtools._convolve2d` 用「位域」把模式、边界、是否翻转（FLIP_MASK）、数据类型打包进同一整数，靠 `flip` 标志（1 翻转做卷积 / 0 不翻转做相关）配合 Python 侧 `.conj()` 一核两用。`_inputs_swap_needed` 仅在 `valid` 模式判定是否交换两输入，相关在交换后需翻转撤销、卷积因可交换律免撤销；`_np_conv_ok` 为 1-D 输入走 NumPy 快速通道。关键术语：翻转、共轭、mode、boundary、位打包、flip 标志、输入交换。本讲不涉频域，`fftconvolve`/`choose_conv_method` 留待后续讲义。
