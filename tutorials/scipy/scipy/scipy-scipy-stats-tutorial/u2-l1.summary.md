本讲是 `scipy.stats` 描述性统计的入口，覆盖 `gmean`、`hmean`、`pmean` 与 `describe` 四个函数。核心洞察是：几何、调和、幂三种平均本质上都是「加权算术平均」套了一层变换——`gmean = exp(mean(log a))`、`hmean = 1/mean(1/a)`、`pmean(a,p) = mean(a**p)**(1/p)`，三者共用同一个内部加权平均内核 `_xp_mean`（负责 axis/weights/nan_policy/keepdims），并经 `@_axis_nan_policy_factory` 装饰器统一处理多维逐轴与 NaN。`pmean` 是「广义平均」，参数 p 取 -1/0/1/2 分别给出调和/几何/算术/平方平均，p==0 时转调 gmean，极小 p 用线性近似保证数值稳定。幂平均关于 p 单调递增，故满足 调和 ≤ 几何 ≤ 算术 ≤ 平方。`describe` 一次返回命名元组 `DescribeResult`（nobs/minmax/mean/variance/skewness/kurtosis），方差默认无偏（ddof=1，由 `_var` 经 `_moment` 二阶中心矩校正得到），峰度为 Fisher 定义。关键术语 `_xp_mean`、`DescribeResult`、axis、weights、ddof、bias 将由下一讲（偏度/峰度与中心矩）承接。
