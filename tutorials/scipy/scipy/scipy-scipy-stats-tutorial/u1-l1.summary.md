本讲是 scipy.stats 手册的「地图课」，仅以子包入口 `__init__.py` 为材料建立全局认知。核心结论有四：其一，scipy.stats 是 SciPy 中负责统计计算的子包，体量最大、子领域最多；其二，模块文档字符串本身即功能地图，按 `autosummary` 主题分组列出概率分布（每个分布是 `rv_continuous`/`rv_discrete` 的子类实例）、汇总与频率统计、假设检验、重采样与蒙特卡洛（`bootstrap`、`permutation_test`、`monte_carlo_test`）、KDE、QMC、灵敏度分析等大类；其三，它主动声明边界——回归/线性模型/时间序列归 statsmodels，表格归 pandas，贝叶斯建模归 PyMC，机器学习归 scikit-learn，可视化归 seaborn，自身只提供「统计原语积木」；其四，`__init__.py` 通过 `from ._xxx import` 把分散实现文件聚合进统一命名空间，再用 `__all__ = [s for s in dir() if not s.startswith("_")]` 自动生成公开 API，并由 PytestTester 提供 `test()` 入口。
