本讲讲解 `scipy.linalg` 中三个 LU 分解相关函数的分工与实现。LU 分解将矩阵拆为 \(A=PLU\)：\(P\) 为部分主元引入的置换矩阵，\(L\) 为单位下三角（保证 \(|L_{ij}|\le 1\)、数值稳定），\(U\) 为上三角。`lu_factor` 返回紧凑的 `(lu, piv)`（上三角存 U、严格下三角存 L 的非对角元，主元已从 LAPACK 的 1 基转为 0 基），直连 f2py 包装的 `getrf`；奇异时（info>0）发 `LinAlgWarning` 而非抛错。`lu_solve` 是 `getrs` 的薄包装，复用该结果做两次三角求解，`trans` 参数支持 \(A/A^T/A^H\) 系统，核心价值是「一次分解、多次求解」。`lu` 面向查看因子，返回独立的 P、L、U，但走的是 C++ 扩展 `_linalg_lu`（而非 f2py），原生支持批处理维度，并提供 `permute_l`、`p_indices` 两种省内存表示。关键术语：部分主元、紧凑 LU 存储格式、主元序列 piv、`getrf`/`getrs`、`_linalg_lu_det.hh`（lu 与 det 共用 getrf，行列式为其副产品，单精度提升 double 累乘防溢出）、`ipiv_to_perm`（逆序应用交换序列）。选型：反复求解用 `lu_factor`+`lu_solve`，查看因子或批量分解用 `lu`。本讲承接 u2 的 `solve`/`inv`/`det`，并引出 Cholesky（u3-l2）、QR（u3-l3）等结构化分解。
