本讲定位 scipy.linalg 为 SciPy 的线性代数子包，含两个最小模块：顶层导出、与 numpy.linalg 对比。入口 __init__.py 本身不含算法，仅以一段文档字符串把全部公开函数按基础运算、特征值、分解、矩阵函数、矩阵方程、随机投影、特殊矩阵七大类列出，再用一组星号导入（`from ._xxx import *`）把 `_basic`、`_misc`、`_decomp_lu`、`_matfuncs` 等实现文件里的函数汇聚到统一命名空间，`__all__` 由扫描当前命名空间自动生成。与 numpy.linalg 为互补而非重复：SciPy 函数更多更专业，同名函数底层多走 LAPACK/BLAS，并多出 `check_finite`、`overwrite_a`、`lapack_driver` 等参数；两包共享同一个 `LinAlgError`（SciPy 直接复用 NumPy 的），SciPy 另定义 `LinAlgWarning`。引入的关键术语：ndarray 容器、汇聚命名空间、星号导入、自动 `__all__`、`PytestTester`、底层例程 `get_blas_funcs`/`get_lapack_funcs`，供后续讲义承接。
