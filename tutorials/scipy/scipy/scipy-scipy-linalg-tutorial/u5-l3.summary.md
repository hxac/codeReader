本讲义讲解 `scipy.linalg.sqrtm`（矩阵平方根）的完整实现链路。核心要点：矩阵平方根 \(R\) 满足 \(RR=A\)，是矩阵函数而非逐元素开方；当矩阵无负实轴特征值时存在唯一主平方根，实矩阵含负特征值时结果自动升级为复数。算法骨架为：先做 Schur 分解 \(A=ZTZ^H\)，在上三角因子上求平方根，再还原。关键概念包括三角平方根的逐元素递推公式 \(R_{ij}=(T_{ij}-\sum R_{ik}R_{kj})/(R_{ii}+R_{jj})\)，及其与 Sylvester 方程 \(R_{11}X+XR_{22}=T_{12}\)（LAPACK `trsyl` 求解，返回需乘回 `scale`）的等价性。`sqrtm` 入口是 Python 薄壳，仅做校验与告警翻译，数值工作委派 C 后端。存在两套分块 Schur 实现：Cython 版 `_sqrtm_triu`+`within_block_loop`（迭代分块，现被 `logm`/`fractional_matrix_power` 复用）与纯 C 版 `sqrtm_recursion_*`（递归分块，是当前 `sqrtm` 的实际路径，额外承担 Schur 前端与实→复「斑马展开」转换）。病态/奇异情形以 `LinAlgWarning` 报告，内部错误抛 `LinAlgError`。
