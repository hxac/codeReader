本讲围绕 `_decomp_svd.py` 讲解 SVD 族六个函数。核心 `svd` 将任意矩阵分解为 A=UΣV^H（U、V 为酉矩阵，Σ 对角线上为降序非负奇异值），其本身是「校验→归一化→覆写门控→ILP64 溢出检查→委派 C++ 后端 `_batched_linalg._svd`→错误翻译」的 Python 薄壳，靠 `full_matrices`、`compute_uv`、`lapack_driver`(gesdd/gesvd) 三个参数控制返回形状与性能，错误统一由 `_format_emit_errors_warnings` 翻译。`svdvals` 是只取奇异值的语法糖，`diagsvd` 把奇异值向量撑回二维 Σ。`orth`（列空间）取大奇异值对应的左奇异向量列，`null_space`（零空间）取小奇异值对应的右奇异向量行，二者均以 `rcond`(默认 eps·max(M,N)) 做数值秩截断；前者用 `full_matrices=False`，后者必须 `True`。`subspace_angles` 用「先正交化再两次 SVD」求两子空间主夹角。引入的关键术语：奇异值、左/右奇异向量、酉矩阵、列空间/零空间、数值秩、rcond、批量维度、`_apply_over_batch`。
