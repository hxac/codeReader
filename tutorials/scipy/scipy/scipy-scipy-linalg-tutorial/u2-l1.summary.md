本讲是 scipy.linalg「特殊结构矩阵求解」的预备知识，覆盖三个最小模块：`norm`、`bandwidth`、`issymmetric`/`ishermitian`。核心结论有三：其一，`norm` 相比 NumPy 同名函数的差异在于默认 `check_finite` 拦截 NaN/Inf，且对一维欧氏范数走 BLAS `nrm2`、对二维 1/无穷范数走 LAPACK `lange`，其余（Frobenius、奇异值范数等）回退 `np.linalg.norm`。其二，`bandwidth(a)` 返回元组 `(lower, upper)`，下带宽 0 即上三角、上带宽 0 即下三角、N-1 即满矩阵；Python 层只做 dtype 与维度校验，真正的「只扫未测对角线」扫描由 C++ 后端 `_bandwidth` 完成，并支持批量输入。其三，`issymmetric`/`ishermitian` 由 Cython `nogil` 内核做精确逐元素比较，按内存布局（C 序/F 序/非连续）分派内核；给定 `atol`/`rtol` 时回退 `np.allclose`。关键术语包括：C 序与 F 序内存布局、带状矩阵的上下带宽、对称（\(A=A^T\)）与 Hermitian（\(A=A^H\)）的数学区别（实矩阵二者等价，复矩阵不同）、容差比较路径。需注意 Hermitian 的复数内核会检查对角元是否为实数，而 `issymmetric` 跳过对角线。这些结构信息是后续 `solve_banded`、Cholesky、`eigh` 等高效求解器的前置输入。
