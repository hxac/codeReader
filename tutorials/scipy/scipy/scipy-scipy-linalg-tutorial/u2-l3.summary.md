本讲覆盖 scipy.linalg 五个最小模块。inv、det、lstsq 的 Python 层都极薄，统一走「校验→dtype/内存对齐→overwrite 门控→委派 C++ 后端 _batched_linalg→用 _format_emit_errors_warnings 聚合错误」的流程。inv 仅方阵非奇异时可逆，奇异时抛 LinAlgError；det 基于 LU 分解，奇异时返回 0，单精度结果提升为 double 防溢出；解方程应优先 solve 而非 inv。lstsq 求最小二乘解，默认驱动 gelsd，返回四元组（解、残差、秩、奇异值），残差仅在超定且满列秩时有效。pinv 与 pinvh 给出 Moore-Penrose 伪逆，对任意形状、任意秩矩阵都存在：pinv 基于 SVD、向量化批处理，pinvh 基于 eigh、逐片处理 Hermitian 矩阵。get_lapack_funcs 按 dtype 选 s/d/c/z 前缀直连 LAPACK；inv/det/lstsq 已迁移至 C++ 后端不再直接调用它，但 solve_triangular、solve_banded 等结构化路径仍在使用。
