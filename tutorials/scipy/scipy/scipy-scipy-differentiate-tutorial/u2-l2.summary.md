本讲打开 `derivative` 的数学内核：`_derivative_weights` 如何计算有限差分权重。核心结论是权重的本质为解一个 Vandermonde 线性方程组——矩条件 Σ wᵢ hᵢᵏ = δ_{k,1}（右端 b[1]=1、余为 0）抽出一阶导系数，故权重只依赖各求值点的相对位置 hᵢ，而与实际步长 H 无关。引入的关键术语与概念：stencil（差分模板）、中心/单侧 stencil、terms=n=order//2、fac=step_factor。中心 stencil 关于 x 对称、点按公比 1/fac 几何排布，精确权重具反对称性，源码强加 weights[n]=0、weights[-i-1]=-weights[i] 抹平浮点残差以提精度；单侧 stencil 按 1/√fac 排布（使每轮「丢 2 点、加 2 点」与中心同步），左权=−右权。权重只依赖 fac 与 n，经挂在 work 上的 diff_state（_RichResult）缓存，fac 或 order 变化时失效重算；包进嵌套对象是为防止 eim._loop 的元素压缩误伤这块全局共享数据。最终权重在 post_func_eval 里以 fc@wc/H 形式作用于函数值，权重与函数值顺序的对齐留待后续讲义。
