本讲建立 SciPy 的全局认知地图，精读 README、包入口 scipy/__init__.py、文档首页三文件。核心结论：SciPy 是面向数学/科学/工程的开源 Python 库，把统计、优化、积分、线性代数、傅里叶变换、信号与图像处理、ODE 求解等高性能数值算法（底层用 C/Fortran/Cython 实现）封装成易用接口。它硬依赖 NumPy 的 ndarray（地基数组、楼上算法），import 时校验 NumPy 版本须落在 [2.0.0, 9.9.99) 区间；并按领域拆成 17 个公开子包（cluster 至 stats），旧的第 18 个 odr 已于 1.19 移除。

供后续讲义承接的关键术语：子包（按领域分目录）、延迟导入（lazy import，借 PEP 562 模块级 __getattr__ 实现「用到才加载」）、公共 API（__version__/test/show_config/LowLevelCallable）、子包清单的双重权威来源（docstring 与 submodules 列表须一致）、version.py 由 meson-python 构建时动态生成。
