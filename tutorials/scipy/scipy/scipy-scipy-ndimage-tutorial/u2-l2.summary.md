本讲把一维相关/卷积推广到任意维邻域，全讲只涉 `_filters.py` 中 6 个函数。`correlate` 与 `convolve` 是极薄壳，差别仅在布尔标志 `convolution`，所有逻辑都在统一引擎 `_correlate_or_convolve`：复数分派（复数 `weights` 仅在相关时取共轭，再拆为实部分量绕过 C 内核只认实数的限制）→ `_check_axes` 规范 axes → `_expand_footprint`/`_expand_origin` 展开 → 卷积翻转 → 逐轴 origin 校验 → 内存别名保护 → 调用同一 C 内核 `_nd_image.correlate`。核心结论：N-D 卷积=沿每个轴翻转核(`weights[::-1,::-1,...]`)+逐轴 origin 取反+偶数长度轴再 `-=1`，之后与相关共用代码。关键概念：`axes` 为子集时，「展开三件套」给未滤波轴补中性值（footprint 插长度 1 新轴、origin 补 0、mode 补 `'constant'`）；其中 `_expand_mode` 相关/卷积**不调用**（显式拒绝逐轴 mode 序列），仅供秩/极值滤波使用。本讲还确立：在相关/卷积中 `weights` 即「加权 footprint」，权为 0 等价于不参与求和。
