本讲承接 u2-l3 的高斯/平滑滤波，把「导数核」与「平滑核」组合成图像处理中最经典的**边缘检测**算子族，全部代码集中在 `_filters.py`。先立两条定位路线：一阶导在边缘处取**极值**，二阶导在边缘处**过零（零交叉）**；其「原子」是一阶差分核 `[-1,0,1]` 与二阶差分核 `[1,-2,1]`。

模块一是 `prewitt`/`sobel`：它们并非纯差分，而是「沿指定轴做一阶中心差分 + 沿其余轴就地平滑」的**可分离**组合，二者唯一区别是平滑核（`prewitt` 用 `[1,1,1]`、`sobel` 用 `[1,2,1]`，均未归一化），借助 `correlate1d` 的 `input=output` 就地链式只分配一个临时数组。

模块二是 `generic_laplace` 引擎：实现「各轴二阶导之和」，第一轴写入输出、后续轴算到临时 `tmp` 再 `output += tmp`，且每次输入都是原图；`laplace` 只是把它的回调填成 `[1,-2,1]` 相关，在阶跃边缘给出「正峰+负峰」并零交叉。

模块三是高斯导数组合：`gaussian_laplace`（Laplacian-of-Gaussian，LoG，等价于先平滑再求拉普拉斯）= `generic_laplace` + 该轴 `gaussian_filter(order=2)`；`generic_gradient_magnitude` 是「各轴一阶导平方和开根号」的引擎，与 `generic_laplace` 的根本差别是「平方后求和 vs 直接求和」，结果恒非负；`gaussian_gradient_magnitude` = 该引擎 + `gaussian_filter(order=1)`。`sigma` 越大响应越宽越低越抗噪，但零交叉定位不变。
