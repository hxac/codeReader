`scipy.ndimage` 是 SciPy 的多维图像处理子包，处理对象为任意维度的 NumPy 数组（不限于二维照片）。它的公开 API 在 `__init__.py` 顶部按五大功能域分组：**Filters**（邻域滤波，如 `gaussian_filter`、`convolve`）、**Fourier filters**（频域乘法核，假定输入已 FFT，共 4 个）、**Interpolation**（几何重采样，如 `map_coordinates`、`affine_transform`）、**Measurements**（连通区域标记与统计，如 `label`、`find_objects`）、**Morphology**（二值/灰度形态学，如 `binary_dilation`、`distance_transform_edt`）。

这些函数经一条**装配链**产生：五个私有实现模块 → `_ndimage_api.py` 聚合裸 API → `_support_alternative_backends.py` 加 CuPy/JAX 后端委托 → `__init__.py` 公开命名空间。本讲引入的关键术语包括：核/邻域、滤波、频域乘法核、样条阶数 order、连通区域、结构元、装配链、数组 API 后端委托。两大常见坑：`label` 返回二元组 `(labeled_array, num_features)`；旧子模块命名空间（如 `scipy.ndimage.filters`）已弃用，应统一从 `scipy.ndimage` 导入。
