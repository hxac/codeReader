本讲义覆盖「顺序统计量 / 回调 / 向量化」四类滤波，对应前置「加权求和」型之外的另一路线。核心结论：`rank_filter`/`median_filter`/`percentile_filter` 是同一引擎 `_rank_filter` 的三件外套，仅把人的输入换算成整数 `rank`（中位数取 `n//2`、百分位取 `int(n*p/100)`、秩直接用）；引擎靠端点短路（`rank` 取端点→`minimum_filter`/`maximum_filter`）与一维快速路径（C++ 双堆 `Mediator`，O(n log w)）优化，多维则走通用 `_nd_image.rank_filter`。`minimum_filter`/`maximum_filter` 共用 `_min_or_max_filter`，矩形邻域可分离（逐轴调 O(n) MINLIST/MAXLIST 的 1-D 滤波），任意 footprint 或带 `structure` 才走 N-D 内核；中位数不可分离故无逐轴路径。`generic_filter`（每元素回调 `function(buffer)`）与 `generic_filter1d`（每线回调须就地改 `output_line`）用 C 遍历邻域+用户函数，支持 `LowLevelCallable`。`vectorized_filter` 走 `np.pad`+`sliding_window_view`+一次向量化调用，从而支持 float16/复数/NaN 控制/偶数窗口常规中位数/多输出，但属暴力法。引入关键术语：顺序统计量滤波、rank、filter_size、端点短路、可分离性、Mediator 双堆、回调契约、三档取舍。
