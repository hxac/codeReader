本讲是 scipy.ndimage 滤波单元（u2）的地基，覆盖 `_filters.py` 中的四个最小模块。`correlate1d` 是滤波子包的原子操作：在指定轴上让一维核沿每条线滑动加权求和，锚点位于 ⌊L/2⌋+origin，最终落到 C 内核 `_nd_image.correlate1d`。`convolve1d` 没有独立内核，仅靠「翻转核 + origin 取反 + 偶数长度再减 1」把卷积改写成相关，复用 `correlate1d`；对称核下两者结果相同。`origin` 控制锚点偏移，合法范围由 `_invalid_origin` 判定为 −⌊L/2⌋ ≤ o ≤ ⌊(L−1)/2⌋（偶数核不对称），正 origin 使结果整体右移。复数输入经 `_complex_via_real_components` 拆成至多 4 路实数相关，按 Re=RR−II、Im=RI+IR 线性组合，绕过 C 内核只认实数的限制。参数校验固定顺序为 dtype→weights 维度→axis→origin→mode 编码，后续高阶滤波的预处理皆是其变体。
