本讲是「插值与几何变换」单元收尾篇，承接 affine_transform，覆盖四个几何变换函数：geometric_transform、shift、zoom、rotate，并指出它们全部复用 zoom_shift（可分离）与 geometric_transform（通用）两条已有 C 内核，无新增内核。核心一：geometric_transform 是最通用的变换，其 mapping 回调按 pull 模型把输出坐标映射为输入坐标，C 内核逐像素调用并样条插值采样；与 map_coordinates 共用 NI_GeometricTransform，靠 mapping 是否为 None 区分。关键术语：回调契约（Python 版 mapping(out_coords, *extra_args, **extra_kwargs) 与 C/LowLevelCallable 版签名，返回值约定 1=成功、0=出错）、extra_arguments/extra_keywords 参数注入、Py_Map 桥接函数。核心二：shift≡affine_transform(matrix=单位阵, offset=-s)，内部取负以凑出 output[o]=input[o-s]，走 zoom_shift 内核。核心三：zoom 是「对角缩放＋重采样到新分辨率网格」，按 grid_mode 把实际缩放因子重算为 (N-1)/(M-1)（像素中心对齐）或 N/M（像素全程对齐），zoom=1 有无损早退。核心四：rotate 直接复用 affine_transform，构造旋转矩阵 rot_matrix 与居中 offset=in_center-R@out_center，ndim>2 时逐 2D 切片调用。三条等价关系：平移=单位阵仿射、缩放=对角阵仿射＋重采样、旋转=旋转阵仿射。
