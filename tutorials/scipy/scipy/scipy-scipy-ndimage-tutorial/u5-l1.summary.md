本讲是形态学单元（u5）首篇，集中讲解 `_morphology.py` 中三个函数：`generate_binary_structure`、`iterate_structure` 与内部辅助 `_center_is_true`。核心结论是：结构元（structuring element）是一小块布尔数组，定义「当前像素的哪些邻居参与运算」；它本质是连通性规则的具象化。`generate_binary_structure(rank, connectivity)` 用一行公式 `np.add.reduce(np.fabs(np.indices([3]*rank) - 1)) <= connectivity` 生成边长为 3 的最小布尔结构元，即「到中心的 L1 距离 ≤ connectivity」的掩码；`connectivity` 取值 1..rank，1 仅含轴向邻居（4-连通），rank 含全部对角邻居。由于偏移限定在 {-1,0,1}，L1 距离恰等于平方欧氏距离。`iterate_structure` 把原结构元贴到中央再自我膨胀 `iterations-1` 次，输出每维长度满足 `shape_i = s_i + (iterations-1)·(s_i-1)`，体现「连续膨胀 n 次等价于用扩张结构元膨胀一次」的恒等式。`_center_is_true` 判断锚点（origin+shape//2）是否为真；中心为真时侵蚀结果必为输入子集，使 C 内核可走只扫描变化像素的快速路径。这些结构元与 `cit` 判定将被后续二值/灰度形态学讲义直接复用。
