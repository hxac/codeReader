本讲是「插值与几何变换」单元（u3）的地基，只读 `_interpolation.py` 并下探 C 内核 `ni_splines.c` / `nd_image.c`。核心概念：B 样条插值用样本值 s[k] 构造连续曲线 S(x)=Σ c[k]βⁿ(x-k)，其中 c[k] 是**样条系数**、n 是 `order`（0–5）。关键结论：只有 `order=0/1` 时系数等于样本；`order≥2` 时必须先做**预滤波**把样本反解为系数，否则插值曲线不经过样本点。预滤波在时域用「增益 + 因果/反因果两次递归扫描」完成，极点由 `get_filter_poles` 给出（如三阶 z=√3−2，增益为 6）。三个核心函数：`spline_filter1d` 是一维预滤波原子（0/1 阶短路拷贝）；`spline_filter` 靠可分离性逐轴串联（只接受 2–5 阶）；`_prepad_for_spline_filter` 只对 `nearest`/`grid-constant` 补 12 点（因 C 端无精确边界初值，靠 |z|¹²≈4×10⁻⁵ 衰减近似）。`map_coordinates` 等函数的 `prefilter=True`（默认）内部即「预填充→spline_filter→C 重采样」，可用手动 `spline_filter`+`prefilter=False` 精确复现；正确预滤波会让 sharp 边缘出现振铃过冲。
