本讲覆盖 scipy.ndimage 的「按标签分组统计」函数组：`sum`/`sum_labels`/`mean`/`variance`/`standard_deviation` 与 `minimum`/`maximum`/`median`。核心结论是：这八个函数共享统一的三参数契约 `input+labels+index`，背后只有两个私有内核——`_stats` 基于 `np.bincount` 在常数次全数组遍历内算出 count/sum/中心化平方和，求和类统计均复用同一引擎（mean=sum/count、variance=sums_c/count）；`_select` 基于一次 `argsort` 加「后写胜」fancy indexing（正向赋值取 max、反向赋值取 min）一次性取出每标签的最小/最大值、位置与中位数。引入的关键术语：`index` 三种形态（标量/序列/None）决定返回形态，且输出顺序严格跟随 `index`；缺失标签在求和类置 0（mean 因 0/0 得 nan）、顺序统计量指向初始化为 0 的哨兵槽。该组函数是纯 NumPy 向量化的 Python 实现，不调用任何 C 扩展，与滤波单元架构不同。`_select` 的「五开关」模型为下一讲通用聚合（`labeled_comprehension`、`extrema`、`center_of_mass`）做铺垫。
