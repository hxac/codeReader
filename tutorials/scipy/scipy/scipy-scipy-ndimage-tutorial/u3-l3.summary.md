本讲讲解 `scipy.ndimage.affine_transform` 的实现原理。其核心是 pull（反向）重采样模型：对每个输出像素 o，先用 `matrix·o + offset` 算出输入坐标，再在该处做样条插值读取；因此用户手上的 push（正向）矩阵需先 `np.linalg.inv` 求逆后传入。关键参数 `matrix` 接受四种形状：对角 (ndim,)、方阵 (ndim,ndim)、齐次 (ndim+1,ndim+1)（须底行 [0,…,0,1]，且忽略传入的 offset）、增广 (ndim,ndim+1)；后两种齐次形状在源码中被归约成方阵加平移。分派逻辑为：对角矩阵走可分离的快速路径 `_nd_image.zoom_shift`（该内核公式 cc=zoom·(o+shift)，故 Python 端需令 shift=offset/matrix 才能还原 matrix·o+offset），其余矩阵走 `_nd_image.geometric_transform` 的仿射分支做完整矩阵乘。`output_shape` 使输出画布可与输入不同，是实现旋转不裁切的关键；`shift`/`zoom`/`rotate` 均为该函数的便捷封装。承接术语：pull/push 重采样、matrix、offset、output_shape、齐次坐标、zoom_shift/geometric_transform 两条 C 内核。
