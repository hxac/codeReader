本讲是 scipy.interpolate 学习手册的第一篇，面向零基础读者建立「全局地图」。核心结论有三：其一，区分插值（要求曲线精确穿过每个已知点 p(xᵢ)=yᵢ）与拟合（允许不穿过点而追求总体误差最小），该子包主打插值，也含平滑样条。其二，该子包解决三大类问题——一维插值（univariate）、规则网格多维插值、散点插值（unstructured/scattered），这构成全手册主线，且与官方 API 分类一一对应。其三，通过精读子包入口 __init__.py 揭示两套机制：它用一连串 `from ._xxx import *` 把十几个实现模块（如 _bsplines.py）的公开名字汇聚成扁平的 scipy.interpolate 命名空间；__all__ 则用 `[s for s in dir() if not s.startswith('_')]` 自动收集而非手写维护。顶部长 docstring 既是官方文档目录也是按三大类组织的功能地图。引入的关键术语包括：插值、拟合、三大类问题分类、BSpline 对象、make_interp_spline（返回 BSpline 的一维插值首选现代接口）、RegularGridInterpolator、griddata、垫片（shim）模块、自动收集 __all__ 机制。后续讲义 u1-l2、u1-l3 将展开目录与构建细节，u2 单元进入一维插值实战。
