本讲承接 u3-l1 的 DCT，讲解离散正弦变换 DST 的四个函数 `dst`/`idst`/`dstn`/`idstn`。核心结论有三：其一，DST 源于信号的**奇对称（反镜像）延拓**，频域只剩正弦项，天然适合「两端为 0」的信号（固定端振动弦、泊松方程内部点），与 DCT 的偶对称（自由端）形成分界。其二，四种 type 的差异仅在于折叠点位置（整数 vs 半整数）：type II/III 互逆、type I/IV 自逆，结构与 DCT 高度同构。其三，DST 与 DCT **共用同一套 `_r2r`/`_r2rn` 内核**，靠 `functools.partial` 的第二个参数传 `pfft.dst` 还是 `pfft.dct` 来区分——这是本讲反复强调的关键。引入的关键术语包括：奇对称延拓、四种 type、type 翻转（idst 通过 2↔3 翻转复用 dst 内核）、`norm` 三模式与 `2-inorm` 方向翻转、`orthogonalize`（仅对 type II/III 修正首尾元素以达正交）。四个函数体仅 `return (Dispatchable(x, np.ndarray),)`，经「公共 API → uarray 分派 → `_execute` 桥接 → `_duccfft` → C 扩展 `pyduccfft`」四层链路完成计算；`norm='ortho'` 配合默认 `orthogonalize=True` 保证能量守恒，是系数截断压缩的数学前提。
