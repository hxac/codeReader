本讲介绍 `scipy.integrate` 中最基础的「固定样本积分」工具：`trapezoid` 与 `cumulative_trapezoid`，源码位于 `_quadrature.py`。固定样本积分指手中只有离散采样点 `(x_i, y_i)`、不知被积函数也无法再采样，用几何公式估面积且无误差估计。核心是复合梯形公式：相邻点连直线求小梯形面积之和，源码用错位切片 `y[1:]` 与 `y[:-1]` 配对、乘宽度 `d`、再求和向量化实现。两函数本质差异仅一处：`trapezoid` 用 `xp.sum` 返回整段定积分，`cumulative_trapezoid` 用 `xp.cumulative_sum` 返回逐点累加的积分曲线，并可经 `initial=0` 补零对齐长度。宽度 `d` 有三种来源（标量 `dx`、一维 `x` 差分、多维 `x` 逐位置差分），故支持非等距样本与任意 `axis` 批量处理。关键术语：复合梯形公式、错位切片、累计求和、`@xp_capabilities` 装饰器与 `array_namespace` 实现的数组 API 多后端（NumPy/CuPy/JAX/Dask）可移植性。
