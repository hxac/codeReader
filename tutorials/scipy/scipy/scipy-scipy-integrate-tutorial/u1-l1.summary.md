`scipy.integrate` 是 SciPy 中专做数值积分与常微分方程（ODE）求解的子包，核心入口是 `__init__.py`，它身兼两职：上半部分以 Sphinx `autosummary` 指令把公开 API 按主题分组（兼作文档目录与人类可读清单），下半部分用多条 `from ._xxx import ...` 语句真正把名字搬入命名空间，最后用 `__all__ = [s for s in dir() if not s.startswith('_')]` 自动收集、过滤掉下划线开头的私有名字来生成公开 API 列表，体现「约定优于配置」。

公开 API 划分为五大类：函数积分（传可调用函数，算法自适应采样，如 `quad`、`quad_vec`、`dblquad`/`tplquad`/`nquad`、`cubature`、`tanhsinh`、`nsum`、`fixed_quad`、`newton_cotes`、`lebedev_rule`、`qmc_quad`）；固定样本积分（只给离散点，按梯形/辛普森等几何公式估算，如 `trapezoid`、`cumulative_trapezoid`、`simpson`、`cumulative_simpson`、`romb`）；ODE 初值问题（IVP，知起点求未来，新 API 为 `solve_ivp` 及 `RK23/RK45/DOP853`、`Radau/BDF`、`LSODA` 等求解器类与基类 `OdeSolver`）；ODE 边值问题（BVP，知两端求整段，仅 `solve_bvp`）；以及封装老 Fortran 求解器的旧 API（`odeint`、`ode`、`complex_ode`）。

本讲引入的关键术语：积分、固定样本积分、函数积分、ODE、初值问题（IVP）、边值问题（BVP）、`__init__.py` 命名空间机制、`from ._xxx import *`、`__all__` 自动汇总。要点是各公开名字分别来自 `_quadrature.py`、`_quadpack_py.py`、`_quad_vec.py`、`_cubature.py`、`_tanhsinh.py`、`_lebedev.py`、`_bvp.py`、`_ode.py`、`_odepack_py.py` 以及 `_ivp/` 子包（注意 `_ivp` 是子目录而非单文件），下划线开头的均为内部模块，应通过公开名字使用。
