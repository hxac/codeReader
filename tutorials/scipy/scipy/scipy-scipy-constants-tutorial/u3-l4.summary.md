本讲义是 scipy.constants 专家篇收尾，讲解该子包的测试体系与回归保护，覆盖四个最小模块。第一，CODATA 查找与基础解析测试（test_codata.py）：用经典 numpy.testing 断言验证 find() 的大小写不敏感子串匹配、无命中返回空列表、全量键数大于 300，并将文本解析值与顶层常量对齐。第二，精确值回归与历史 bug 锁定：test_exact_values 端到端复跑 exact2018 与 replace_exact 回填机制，断言精确常数 value 一致且 precision（相对不确定度 uncertainty/value）为零；test_gh11341 锁定别名兼容性，test_gh14467 用严格相等锁定精确值截断修复，二者为永久回归契约。第三，数组 API 测试与 make_xp_test_case：装饰器读取被测函数 @xp_capabilities 能力元数据，自动生成 skip_xp_backends/xfail_xp_backends 标记，配合 xp 夹具把同一份测试参数化到多个数组库，实现能力声明与测试覆盖同源。第四，跨后端断言工具 xp_assert_equal/xp_assert_close 及 _array_api_no_0d 变体，用 _check_scalar 强制标量输入返回标量。核心结论：标量函数无需参数化，数组函数才需 make_xp_test_case；tests/meson.build 用 install_tag:'tests' 支持按标签安装。
