本讲聚焦 `scipy.cluster.vq.whiten` 这一个预处理函数及其依赖的 array API 工具。核心结论：`whiten` 按列（`axis=0`，每列是一个特征）将观测矩阵除以该列标准差，使每列达到单位方差，从而消除特征尺度差异对欧氏距离的主导，这是 k-means 前要白化的原因。零方差列的处理是重点：用 `xpx.at(std_dev, mask).set(1.0)` 将除数兜底为 1，使该列原值不变，并在开启 finite 检查时发 `RuntimeWarning`。引入的关键术语包括：观测矩阵 `obs`（M 行观测×N 列特征）、**xp**（array namespace，数组后端命名空间）、**`array_namespace(obs)`**（默认未开 `SCIPY_ARRAY_API` 时返回 `numpy`，借此用 `xp.std`/`xp.any` 跨 NumPy/JAX/Dask 通用）、**惰性数组**（Dask 等，`check_finite` 默认对其关闭以免强制求值）、**`_asarray`**（SciPy 版 `np.asarray`，合并规整与 finite 检查）、**`xpx.at(...).set(...)`**（array_api_extra 的跨后端、不可变版函数式索引赋值，兼容 JAX 不可变数组与 Dask 惰性数组）。本讲不涉及 `vq`/`kmeans` 的迭代逻辑，留待后续讲义承接。
