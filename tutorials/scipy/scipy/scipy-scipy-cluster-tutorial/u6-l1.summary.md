本讲承接 u3-l1 的 linkage matrix，把矩阵 Z 翻译成树并完成各类基于树的操作；关键概念有满二叉树、前序遍历与回放。ClusterNode 是纯 Python 节点类，含 id/left/right/dist/count 五属性，count 递归定义为叶子取 1、内部节点取两孩子之和，并按 dist 重载比较运算符供排序。to_tree(Z) 一遍顺序扫描将 Z 还原为树，rd=False 返回根、rd=True 返回 (根, 长度 2n−1 的全节点列表)，并校验簇引用合法与 Z 第四列计数。leaves_list(Z) 是 root.pre_order() 的 Cython 快路径等价物，用显式栈在 Z 上先左后右前序遍历，返回「从左到右」的 int32 叶子序，供 dendrogram 排序。私有 _order_cluster_tree(Z) 用 BFS 与 bisect.insort_left 返回按 dist 升序的内部节点列表。cut_tree(Z, n_clusters/height) 把切点翻译为「重放到第几步」，从各成一簇逐节点合并并重编号，返回 (nobs, n_cols) 的 int64 归属数组，仅支持 NumPy。
