本讲覆盖 `scipy.cluster.hierarchy` 的三个最小模块：`DisjointSet`、`leaders`、`is_isomorphic`，三者围绕「聚类等价性」概念同源、实现独立。`DisjointSet` 是从 `scipy._lib` 重新导出的通用并查集，核心为 `find`（路径折半）与 `merge`（按大小合并、等大取先插入者为父），单次操作近似 $O(\alpha(n))$，成员枚举靠 `_nbrs` 循环链表。`leaders(Z, T)` 用 `cluster_ids` 数组做一遍后序 DFS：叶子继承扁平标签 `T`、同簇向上传播、异簇处分叉记 leader，返回节点号数组 `L` 与对应簇号 `M`；`T` 与 `Z` 不一致时抛 `ValueError`。`is_isomorphic(T1, T2)` 把等价类问题转为「按 `T1` 排序后比较值变化边界」的 $O(n\log n)$ 数组问题，判断两组标签是否代表同一划分。关键结论：后两者均未内部调用 `DisjointSet`，而是各选最贴合数据形态（树/一维数组）的专用算法，体现「数据表示决定算法效率」。引入的关键术语：leader（单色子树的根）、扁平簇标签 `T`、路径折半、按大小合并、聚类等价性。三者工程上都接入 array API，`leaders` 沿用 `lazy_cython`/`lazy_apply` 桥接。
