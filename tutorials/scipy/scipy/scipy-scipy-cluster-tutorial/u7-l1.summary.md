本讲义覆盖 `scipy.cluster.hierarchy` 的合法性校验设施，源码集中在 `_hierarchy_impl.py`，为纯 Python/array API 实现。核心函数 `is_valid_linkage`/`_is_valid_linkage` 充当 linkage matrix 守门员：先做结构检查（dtype、维数、列数、非空），再做内容检查（负索引、负距离、负计数、计数过大、簇未生成就引用、同一簇被合并两次）。`is_valid_im`/`_is_valid_im` 对称校验不一致矩阵 R 的结构及「均值/标准差/计数非负」，不查可正可负的 z 分数。共享引擎 `_lazy_valid_checks` 采用 throw（抛 ValueError）> warning（发 ClusterWarning）> 静默返回布尔的三态语义，并以 `materialize` 开关控制是否为给结论而强制求值 Dask/JAX 惰性数组，避免内部调用意外触发大规模计算。关键术语：linkage matrix Z（(n−1)×4）、不一致矩阵 R（均值/标准差/计数/不一致系数四列）、ClusterWarning、materialize 惰性求值开关。辅助函数 `is_monotonic`、`num_obs_linkage`、`correspond` 均复用 `_is_valid_linkage(throw=True)` 做前置校验，再各加距离列单调不减、行数+1、Z 与压缩距离矩阵 Y 观测数相等的逻辑；公开入口强制 `materialize=True`、私有引擎默认 `False`。
