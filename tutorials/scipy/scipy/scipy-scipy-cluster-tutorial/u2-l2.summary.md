本讲讲解 `scipy.cluster.vq.vq` 的双层结构。公开入口 `vq(obs, code_book)` 是「宽容前门」：规整输入后按「公共 dtype 是否实数浮点」分发——浮点走 Cython 快路径，整数等走纯 Python `_py_vq`（即 `cdist + argmin + min`，慢约 20 倍，返回欧氏距离）。Cython 包装层 `_vq.vq` 是「严格引擎」：连续化、强制 `float32/float64` 且 dtype 一致、初始化 `outdists=inf` 与 `outcodes=int32`，按 dtype 调核心函数。核心 `_vq._vq` 用平方距离展开恒等式把计算拆成「各自平方和 + 一次 BLAS `gemm`（`cal_M` 置 `alpha=-2.0`）」，`nfeat<5` 时回退朴素三重循环 `_vq_small_nf`；最终对微小负数兜底后 `sqrt` 返回欧氏距离。输出 `(code, dist)`：`code[i]` 为最近码字索引，`dist[i]` 为该距离。本讲为 k-means 迭代的热点函数奠基，并引出后续 `update_cluster_means`。
