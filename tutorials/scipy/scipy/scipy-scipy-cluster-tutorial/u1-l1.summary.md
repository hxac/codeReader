`scipy.cluster` 是一个容器包，顶层 `__init__.py` 本身不含算法函数，只通过 `__all__` 和 `from . import` 暴露两个相对独立的子模块：`vq`（向量量化 / k-means）与 `hierarchy`（层次 / 凝聚式聚类），并附带一个全包通用的 `test()` 入口。本讲只做定位与认路，不涉及实现，重点建立术语与对外接口面的认知。

`vq` 子模块对外提供 `whiten`/`vq`/`kmeans`/`kmeans2` 与异常类 `ClusterError`，典型流程是「whiten 归一化 → kmeans 学码本 → vq 编码」。其核心概念：码字（簇心）、码本（簇心集合）、code（簇编号）、畸变（各观测到最近簇心的平方距离之和）。k-means 与向量量化的数学目标等价（都是最小化畸变），这正是子模块命名的由来。输入约定为 M×N 观测矩阵。

`hierarchy` 子模块对外约 30 个名字，按建树 / 切平 / 统计 / 树表示 / 可视化 / 校验分组，最小工作链是 `linkage → fcluster → dendrogram`：从距离矩阵（常由 `pdist` 生成）自底向上合并成 (n-1)×4 的 linkage matrix Z，再在任意阈值处切平得到 flat 簇。两子模块真正实现藏于带下划线的私有模块 `_vq_impl`、`_hierarchy_impl`，`__init__.py` 仅做重新导出。
