本讲讲解 scipy.fftpack 中专为实数序列设计的 FFT：`rfft`、`irfft` 与 `rfftfreq`，以及配套的实数打包格式。核心动机来自**共轭对称性（Hermitian symmetry）**——实数序列的 DFT 满足 `y[k]=conj(y[n-k])`，正负频互为共轭，因此只需保留非冗余的半边频谱，独立信息恰好是 n 个实数，计算与存储均减半。`rfft` 输出长度为 n 的**实数交错打包（interleaved packing）**数组：`[y(0), Re(y(1)), Im(y(1)), …]`，偶数 n 末位单独存放奈奎斯特实部。`irfft` 是逆向「拆包」，按同一约定补回共轭负频再逆变换取实部，遵循 `backward` 归一化保证往返一致。`rfftfreq` 为该打包提供逐位对齐的频率轴，非直流频率成对出现。关键区分点：fftpack 的实数交错打包与 `scipy.fft.rfft` 返回 n/2+1 个复数的复数打包不同；Python 层只是薄壳，实际计算委托给 DUCC 后端的 `rfft_fftpack`/`irfft_fftpack`，`_fftpack` 后缀即用于区分这两套打包约定。
