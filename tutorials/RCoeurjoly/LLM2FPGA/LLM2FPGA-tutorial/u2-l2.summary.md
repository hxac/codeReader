本讲逐行拆解流水线最上游入口脚本 compile-pytorch.py，它完成「PyTorch 模型 → 静态计算图 → torch 方言 MLIR 文本」三步转换，本身不做任何降级。三个最小模块：①用 `importlib` 按文件路径动态加载适配器，临时插入 `sys.path` 解析兄弟 import，再以 `try/finally` 按值回滚，保持作用域卫生；②用 `torch.export.export` 以 `example_inputs()` 为「模具」把模型冻结成 `ExportedProgram` 对象（非字符串、非文件），`strict` 由适配器 `EXPORT_STRICT` 决定（matmul 默认 True、TinyStories 用 False）；③用 `torch_mlir.fx.export_and_import(output_type="torch")` 翻译成 MLIR module，经 `str()` 序列化后同时写文件与打印。关键概念：ExportedProgram、example_inputs 模具、strict 与 EXPORT_STRICT、output_type 选择与「每站只降一段」的设计哲学、torch 方言算子 `torch.aten.matmul`。该脚本同时是适配器契约的强制执行方，产出为下一讲 u2-l3 降到 Linalg 提供起点。
