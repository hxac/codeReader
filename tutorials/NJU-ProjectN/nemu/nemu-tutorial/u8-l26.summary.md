本讲讲解 NEMU 的「宏基础设施」include/macro.h，它支撑一份源码适配多 ISA/模式/目标。核心模块：concat/str/ARRLEN 须分两层（外层先展开参数），否则 ## 与 # 会阻止宏参数展开；MUXDEF 家族（含 MUXONE 等）与封装 IFDEF/IFONE 用「占位符+逗号+CHOOSE2nd」绕开「#ifdef 不能用于宏展开内部」实现编译期二选一，MUXDEF 视定义为 0/1 均为已定义、MUXONE 只认 1；BITS/SEXT 用 Verilog 风格位切片与有符号位域做符号扩展，支撑指令译码；MAP 以 X-macro 模式让一份列表驱动多处代码生成；CONFIG 全链路为：Kconfig 描述选项 → kconfig 生成 autoconf.h（bool 开为 #define X 1、关为不定义）→ common.h 引入 → 源码用 IFDEF/MUXDEF 在 C 侧语句级裁剪、filelist.mk 用 SRCS-$(CONFIG_*) 在 Make 侧整文件级裁剪，两层互补。
