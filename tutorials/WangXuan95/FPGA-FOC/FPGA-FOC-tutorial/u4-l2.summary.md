本讲为专家层「调参与移植」专题，含三个最小模块。其一，foc_top 用 5 个 parameter 抽离可变量：INIT_CYCLES（初始化时长，与 clk 成反比，上限 2³²−2）、ANGLE_INV（传感器装反则取反）、POLE_PAIR（极对数，用于电角度换算）、MAX_AMP（SVPWM 最大振幅）、SAMPLE_DELAY（采样延时）；核心硬联动约束为 SAMPLE_DELAY+T_sn_adc→en_adc < 采样窗口最小值 1024−2·MAX_AMP，故默认 MAX_AMP=384 已近安全上限。其二，关键区分 parameter（编译期常量）与 input port（运行期信号）：Kp/Ki 为 31bit 输入端口，可运行时动态调整而不必重综合，默认 Kp 比 Ki 大约一个数量级；id_aim 示例恒为 0，表示仅做电流环、不做弱磁控制。其三，跨厂商移植：全库除 altpll 外均为纯 RTL，只需替换 PLL IP（如 Xilinx Clocking Wizard）并把 locked 接 rstn 维持锁相复位语义；主时钟须 <40MHz（adc_ad7928 用二分频产生 SPI sck，AD7928 要求 ≤20MHz），最好钉死 36.864MHz 以凑 18kHz 的 SVPWM/控制节拍并免去重算 I2C、UART 的各 CLK_DIV。
