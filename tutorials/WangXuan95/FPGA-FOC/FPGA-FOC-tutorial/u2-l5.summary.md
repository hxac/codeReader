本讲精读 `pi_controller.v`，它是 FOC 电流环中唯一带状态（积分项）的环节，把 d/q 轴目标电流与实测电流的偏差转换为电压指令 vd/vq。核心公式为离散 PI：误差 e = i_aim − i_real，积分项 idelta = Σe，输出 value = Kp·e + Ki·Σe，最终取 `value[31:16]`（除以 2¹⁶）作为 16 位有符号输出，使 Kp/Ki 成为带定点小数分辨率、可运行时调整的大整数增益。

模块用 `i_en → en1 → en2 → en3 → en4 → o_en` 的使能脉冲逐级下传，构成 5 级流水线：拍0算误差并锁存增益；拍1并行算比例项 Kp·e 与积分累加 Σe（二者均只依赖误差，故合并省一拍）；拍2算 Ki·Σe；拍3汇合 P+I 求和；拍4锁存结果并与 o_en 同时有效。积分项 `idelta` 是唯一跨周期保留的状态，由 `protect_add` 兼任 anti-windup 限幅。

关键概念包括：饱和保护（`protect_add` 用 33 位中间值防加法回绕，`protect_mul` 用 57 位中间值，源自 32+25 的历史签名设计）、积分饱和 windup、握手脉冲约定、定点取高位缩放。在 `foc_top.v` 中被双例化为 `u_id_pi`/`u_iq_pi`，共用同一对 Kp/Ki，i_en 接 Park 输出的 `en_idq`，输出 vd/vq 送入下一讲 cartesian2polar。
