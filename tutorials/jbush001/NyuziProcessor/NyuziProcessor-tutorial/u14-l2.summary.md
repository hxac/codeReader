本讲承接 u14-l1 的 AXI 互连，站在 IO 总线另一端逐个拆解挂在总线上的真实外设控制器：UART、SPI、SDRAM、VGA、PS/2。所有外设侧控制器共用极简的 `io_bus_interface`（5 根信号、一次一拍），顶层用「广播写、选择读」让多个外设共用一根总线：写时把信号广播给所有外设、由各外设自带 `BASE_ADDRESS` 自译码，读时用地址高位多路选择对应一路读数据。每个外设把事件（收字节、按键、帧结束）拉成一根中断线，在顶层拼成 `interrupt_req` 位向量（timer=1、uart=2、ps2=3、vga frame=4），再交核心 `control_registers` 做屏蔽/挂起/精确中断；SDRAM 是内存后端、不产生中断。各控制器要点：UART 带 8 项 RX FIFO、电平触发中断；SPI 为 mode 0 主机、8 位全双工、靠 `RX_STATUS` 轮询完成，专驱动 SD 卡；SDRAM 控制器用状态机驱动 SDR SDRAM 命令，做自动刷新与懒开放页策略，区分 page hit/miss 并读优先于写；VGA 一身两职——IO 从口配置 + AXI 主口 DMA 帧缓冲，时序由软件可编程微序列器 `vga_sequencer`（INITCNT/LOOP 微指令）产生，`start_frame` 兼作帧中断与 DMA 同步触发；PS/2 仅接收，下降沿采样、两级同步防亚稳态、近满丢旧。软件侧寄存器编址见 `registers.h`，与硬件 BASE 一一对齐。
