本讲是入门单元收口，讲清 RISC-V 向量核仿真的端到端数据流。因本仓库不公开标量核，仿真由 Python 脚本离线冒充译码器：`sim_generator.py` 依序读三个参数（csv / AVL / VECTOR_LANES），按助记符查表得 microop 与 fu，并由目的寄存器数推导 maxvl=⌊32/N_vregs⌋×LANES、REPEATS=⌈AVL/maxvl⌉，每轮末注入 4 个气泡模拟标量核控制开销；产出 13 个按行对齐的二进制文件与 `autogenerated_params.sv`（写 `SIM_VECTOR_INSTRS`）。运行阶段 `vector_driver` 用 `$readmemb` 回放产物，按 ready/valid 握手喂给 `vector_top`（DUT），二者以 `decoder_results/` 为唯一接口。`compile_vector_simulator.do` 一键完成 vlib/vlog/vsim/log/run；仿真有正常完成、空闲超时（100 拍）、死锁超时（300 拍发射数不变）三种结束条件，`print_perf_stats()` 写出含 total_cycles、stall_pending 等 7 项指标的 `results.log`。换自己程序时输入数据需自备（`init_main_memory.txt` 不由生成器产出）。
