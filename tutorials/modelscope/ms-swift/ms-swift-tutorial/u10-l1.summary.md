本讲阐明 ms-swift 中「损失函数」与「损失权重」是两件正交的事：`swift/loss/` 经 `--loss_type` 管「损失怎么算」，自定义损失需继承 `BaseLoss`、实现 `__call__` 返回标量并登记进 `loss_map`（仅 sft/pretrain/reranker/embedding 可用，RLHF 不走此机制）；`swift/loss_scale/` 经 `--loss_scale` 管「每个 token 算多重」。核心概念：`base_strategy`（default/last_round/all）决定哪些段参与训练，子类 `get_loss_scale` 决定段内子片段权重；`ConfigLossScale` 用 JSON 按精确匹配（value 长度 2）或正则（长度 1）切分赋权，`+` 语法经 `ConcatLossScale` 链式组合、权重相乘。关键优化 `is_binary_loss_scale`：权重仅 0/1 时把 mask 折叠进 labels（-100）以兼容 liger_kernel，出现非二值才保留浮点权重并在 compute_loss 左移对齐后乘到逐 token 损失。注意自定义损失需自行应用 loss_scale，官方范例故意省略。
