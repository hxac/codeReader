本讲是 ms-swift 学习手册开篇，仅基于 README 建立全局认知，不涉及源码。ms-swift（全称 Scalable lightWeight Infrastructure for Fine-Tuning）是魔搭社区的大模型与多模态大模型微调与部署框架，覆盖训练、推理、评测、量化、部署全链路，统一封装为 `swift <子命令>`。讲义含三个最小模块：一、能力矩阵——支持 600+ 纯文本与 400+ 多模态模型，内置 150+ 数据集，轻量方法含 LoRA/QLoRA/DoRA，并行含 DeepSpeed/FSDP/Megatron；二、模型与方法概览，提出训练沿三个正交维度自由组合：训练任务（pt/sft/rlhf/grpo）、微调方式（`--tuner_type`：full/lora/qlora）、推理或并行后端（`--infer_backend`、`--use_vllm`）；三、技术栈，整合 transformers/peft/trl/deepspeed/vllm/sglang/lmdeploy/evalscope，依赖按功能可选安装。核心结论：相比原生 transformers/peft，ms-swift 把数据格式转换、对话模板、LoRA 挂载、多卡启动、checkpoint 管理等胶水工作封装为统一参数体系，并借 template 系统与 args.json 实现「训练即所见，推理即所得」。关键术语 LoRA、template、tuner_type、rlhf_type、能力矩阵供后续讲义承接。
