本讲义为 Mayoiuta 学习手册的开篇，目标是建立项目全局地图。Mayoiuta 是基于 Apache License 2.0 的开源 NPU（神经网络处理器）项目，定位为面向研究者和爱好者的灵活、可扩展 AI 加速探索平台。

核心概念上，NPU 是为神经网络乘加运算（MAC）量身定制的专用芯片，以「固定功能硬件」（如脉动阵列）换取比 CPU、GPU 更高的能效——CPU 通用、GPU 并行、NPU 专为 AI 算得快且省电。仓库源码由两部分构成：hardware/rtl/（9 个 Verilog 文件，按 core/memory/power/sparse/control/top 分域，描述硬件电路）与 driver/win32/（3 个文件，Windows WDF 内核驱动，负责通信与调度），外加 README 与 LICENSE。

README 的架构概览把 NPU 划分为 CU（计算）、MU（存储）、IN（互连）、CU（控制）四类模块，并宣称四大愿景（开放、灵活、可扩展、教育）与五条特性（模块化、可配置参数、仿真环境、开源工具链、详细文档）。但对照仓库现状，仿真环境、工具链（编译器/汇编器/调试器）、详细文档三项均无对应文件，须标注为「待确认」。引入的关键术语包括：NPU、MAC、RTL、固定功能硬件、WDF 驱动、CU/MU/IN/CU 模块划分，以及「待确认」标注法。
