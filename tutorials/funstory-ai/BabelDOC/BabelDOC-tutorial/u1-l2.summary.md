本讲承接 u1-l1 的心智模型，目标是把 BabelDOC 跑起来并看懂 CLI 入口。安装用官方推荐的 uv tool install --python 3.12 BabelDOC，`babeldoc` 命令由 `pyproject.toml` 入口点 `babeldoc.main:cli` 注册；uv（Rust 实现）可隔离环境、自动管理 Python 解释器、安装快。

最小翻译命令需 `--openai`（服务开关）、`--openai-api-key`、`--files`，语言默认 en→zh；CLI 层目前只内置 OpenAI 兼容服务，兼容官方 OpenAI、DeepSeek、GLM 及本地 Ollama。

入口分两层：同步壳 `cli()` 做日志与全局初始化，用 `asyncio.run` 桥接至异步核心 `main()`；后者解析校验参数、构造 `OpenAITranslator` 与 QPS 限流、为每个文件建中心配置对象 `TranslationConfig`，最终交给 `async_translate(config)` 消费事件流产出 mono/dual PDF。`create_parser` 用 configargparse 将参数分为 Translation 与 OpenAI Options 两组并支持 TOML 配置。`--warmup` 等快捷模式无需 key 即可验证安装。`TranslationConfig`、`async_translate`、QPS 等术语将在后续讲义承接。
