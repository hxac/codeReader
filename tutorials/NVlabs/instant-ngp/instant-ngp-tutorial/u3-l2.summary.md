本讲精读 instant-ngp 的核心创新——多分辨率哈希编码（HashGrid）。本仓库属应用层，只负责解析 JSON 编码参数并自动推导 per_level_scale，真正的哈希内核（哈希函数、三线性插值、碰撞梯度回传）位于依赖库 tiny-cuda-nn 中，不在本仓库。三个最小模块：多层级网格——L 层分辨率按几何级数递增，每层三线性插值出 F 维特征，拼接成 L·F 维送入 MLP，使模型同时具备全局结构与高频细节；空间哈希——每层顶点存入固定大小的哈希表，超出容量即发生哈希碰撞，靠梯度下降自组织竞争化解，这是 instant-ngp「参数少、速度快、高细节」的根本；per_level_scale——把最细层与最粗层的分辨率比在 L−1 个台阶上等比分摊，NeRF 模式下最细层还乘以场景的 aabb_scale，若 JSON 显式给定正值则跳过自动推导。核心编码参数为 n_levels、n_features_per_level、log2_hashmap_size、base_resolution、per_level_scale。
